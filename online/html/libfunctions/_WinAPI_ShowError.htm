_WinAPI_ShowError

Beschreibung anzeigen in

Zeigt eine "Error" MessageBox an mit optionalem Programmende

#include <WinAPIError.au3>
_WinAPI_ShowError ( $sText [, $bExit = True] )

Parameter

$sText Text, der angezeigt werden soll
$bExit [optional] Gibt an, ob das Programm danach beendet werden soll:
True - beendet das Programm
False - kehrt zum Programm normal zurück

Rückgabewert

Keine.

Verwandte Funktionen

_WinAPI_ShowMsg, _WinAPI_MsgBox

Beispiel

#include <WinAPI.au3>

_WinAPI_ShowError("Fehler 1", False)
_WinAPI_ShowError("Fehler 2", True)
_WinAPI_ShowError("Fehler 3", False) ;MsgBox erscheint nicht, da die vorherige das Skript beendet
