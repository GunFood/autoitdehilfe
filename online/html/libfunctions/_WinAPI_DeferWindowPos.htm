_WinAPI_DeferWindowPos



Beschreibung anzeigen in

Aktualisiert die angegebene mehrere-Fenster-Position Struktur für ein angegebenes Fenster.

#include <WinAPISysWin.au3>
_WinAPI_DeferWindowPos ( $hInfo, $hWnd, $hAfter, $iX, $iY, $iWidth, $iHeight, $iFlags )

Parameter

$hInfo Das Handle einer mehrere-Fenster-Position Struktur welches die Größe- und Positionsinformationen für ein oder mehrere Fenster beinhaltet. Diese Struktur wird von _WinAPI_BeginDeferWindowPos() zurückgegeben oder von dem letzten Aufruf der _WinAPI_DeferWindowPos() Funktion.
$hWnd Das Handle eines Fenster wessen Aktualisierungsinformationen in der Struktur gespeichert sind.
Alle Fenster innerhalb einer mehrere-Fenster-Positionsstruktur muss das selbe Parent haben.
$hAfter Das Handle zu dem Fenster welches dem positionierten Fenster in der Z-Reihenfolge vorangeht.
Dieser Parameter muss ein Fensterhandel sein oder einer der folgenden Werte. Dieser Parameter wird ignoriert wenn das $SWP_NOZORDER Flag im $iFlags Parameter gesetzt wurde.
    $HWND_BOTTOM
    $HWND_NOTOPMOST
    $HWND_TOP
    $HWND_TOPMOST
$iX Die X-Koordinate der oberen linken Ecke des Fensters.
$iY Die Y-Koordinate der oberen linken Ecke des Fensters.
$iWidth Die neue Breite des Fensters in Pixeln.
$iHeight Die neue Höhe des Fensters in Pixeln.
$iFlags Eine Kombination der folgenden werde welche die Größe und Position des Fensters beeinflussen.
    $SWP_DRAWFRAME
    $SWP_FRAMECHANGED
    $SWP_HIDEWINDOW
    $SWP_NOACTIVATE
    $SWP_NOCOPYBITS
    $SWP_NOMOVE
    $SWP_NOOWNERZORDER
    $SWP_NOREDRAW
    $SWP_NOREPOSITION
    $SWP_NOSENDCHANGING
    $SWP_NOSIZE
    $SWP_NOZORDER
    $SWP_SHOWWINDOW

Rückgabewert

Erfolg: Der Rückgabewert identifiziert die aktualisierte mehrere-Fenster-Position Struktur.
Fehler: 0. Es ist _WinAPI_GetLastError() aufzurufen um erweiterte Fehlerinformationen zu erhalten.
