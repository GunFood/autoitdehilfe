$tagNMTVKEYDOWN

Show description in

Contains information about a key press in a tree-view control. It is used with the $TVN_KEYDOWN notification code

Global Const $tagNMTVKEYDOWN = $tagNMHDR & ";word VKey;uint Flags"

Parameters

$tagNMHDR Contains information about a notification message
VKey Virtual key code
Flags Always zero
