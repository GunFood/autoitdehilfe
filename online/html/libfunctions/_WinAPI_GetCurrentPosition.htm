_WinAPI_GetCurrentPosition



Beschreibung anzeigen in

Ermittelt die aktuelle Position für den festgelegten Gerätekontext

#include <WinAPIGdi.au3>
_WinAPI_GetCurrentPosition ( $hDC )

Parameter

$hDC Handle zum dem Gerätekontext.

Rückgabewert

Erfolg: $tagPOINT Struktur, welche die logischen Koordinaten der aktuellen Position enthält.
Fehler: Setzt @error auf ungleich 0.
