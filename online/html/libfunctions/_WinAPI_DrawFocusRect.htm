_WinAPI_DrawFocusRect



Beschreibung anzeigen in

Zeichnet ein Rechteck in den benutzten Stil um anzudeuten, dass das Rechteck den Fokus besitzt

#include <WinAPIGdi.au3>
_WinAPI_DrawFocusRect ( $hDC, $tRECT )

Parameter

$hDC Das Handle zum Gerätekontext.
$tRECT Die $tagRECT Struktur gibt die logischen Koordinaten des zu zeichnenden Rechtecks an.

Rückgabewert

Erfolg: True
Fehler: False

Bemerkungen

Die _WinAPI_DrawFocusRect() ist eine XOR Funktion, das bedeutet, dass ein zweiter Aufruf der Funktion mit dem selben Rechteck das Rechteck vom Bildschirm entfernt.
