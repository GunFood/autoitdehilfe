Erzeugt eine logische Schriftart mit spezifischen Eigenschaften

#include <WinAPIGdi.au3>
_WinAPI_CreateFontIndirect ( $tLogFont )

Parameter

$tLogFont $tagLOGFONT Struktur welche die Eigenschaften der logischen Schriftart definiert

Rückgabewert

Erfolg: Handle zu einer logischen Schriftart
Fehler: 0

Bemerkungen

Diese Funktion erzeugt eine "logische" Schrift mit den in der tagLOGFONT Struktur festgelegten Eigenschaften.
Wenn diese Schrift in der SelctObject Funktion gewählt wird, versucht der GDI Font Mapper die logische Schrift mit einer existierenden physikalischen Schrift abzugleichen.
Falls keine genaue Übereinstimmung gefunden wird, stellt der Font Mapper eine alternative Schriftart bereit, welche den gewünschten Eigenschaften bestmöglichst entspricht.

Verwandte Funktionen

$tagLOGFONT
