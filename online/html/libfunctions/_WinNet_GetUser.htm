_WinNet_GetUser



Beschreibung anzeigen in

Ermittelt den Standard Benutzernamen, oder den Benutzernamen, mit dem eine Verbindung hergestellt wurde

#include <WinNet.au3>
_WinNet_GetUser ( $sName )

Parameter

$sName Entweder den Namen eines lokalen Gerätes, das umgeleitet wird zu einer Netzwerkressource, oder der Name einer Netzwerkressource,
zu der eine Verbindung ohne Umleitung eines lokalen Gerätes hergestellt wurde. Wenn dieser Parameter leer ist, liefert das System
den Namen des aktuellen Benutzers für den Prozess zurück.

Rückgabewert

Erfolg: der Benutzername
Fehler: Setzt das @error Flag auf ungleich null
