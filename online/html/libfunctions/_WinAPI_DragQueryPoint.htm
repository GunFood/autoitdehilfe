_WinAPI_DragQueryPoint



Beschreibung anzeigen in

Gibt die Position des Mauszeigers zum Zeitpunkt des Loslassens der Datei während einer Drag-und-Drop Operation zurück

#include <WinAPISysWin.au3>
_WinAPI_DragQueryPoint ( $hDrop )

Parameter

$hDrop Das Handle zur Dropstruktur welche die Dropdatei beschreibt. Dieser Parameter wird zur WM_DROPFILES Nachricht im WPARAM Parameter übergeben.

Rückgabewert

Erfolg: Die $tagPOINT Struktur welche die Koordinaten des Mauszeigers enthält während die Datei losgelassen wurde.
Fehler: Setzt das @error Flag auf ungleich null.
