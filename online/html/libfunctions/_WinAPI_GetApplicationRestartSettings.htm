_WinAPI_GetApplicationRestartSettings



Beschreibung anzeigen in

Ruft die für den angegebenen Prozess registrierten Neustart-Informationen ab

#include <WinAPIDiag.au3>
_WinAPI_GetApplicationRestartSettings ( [$iPID = 0] )

Parameter

$iPID [optional] Die PID des Prozesses. Standard (0) ist der aktuelle Prozess.

Rückgabewert

Erfolg: Das Array, das die folgenden Informationen enthält:
    [0] - Die Neustart-Befehlszeile.
    [1] - Die Flags.
Fehler: Setzt das @error Flag ungleich Null, @extended Flag kann den HRESULT Fehlercode enthalten.

Bemerkungen

Diese Informationen sind nur für den aktuellen Prozess verfügbar; die Funktion _WinAPI_GetApplicationRestartSettings() kann nicht aufgerufen werden, nachdem das Programm neu gestartet wurde, um die Befehlszeile für den Neustart zu erhalten.

Diese Funktion erfordert Windows Vista oder höher.
