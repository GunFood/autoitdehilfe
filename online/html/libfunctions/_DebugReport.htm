Funktionreferenz

_DebugReport

Beschreibung anzeigen in

Schreibt zu einer Debug-Sitzung

#include <Debug.au3>
_DebugReport ( $sData [, $bLastError = False [, $bExit = False]] )

Parameter

$sData Daten, die gemeldet werden sollen
$bLastError [optional] True, wenn _WinAPI_GetLastErrorMessage() an das Ende der Meldung angehängt werden soll
$bExit [optional] True, wenn das Script beendet werden soll

Rückgabewert

Erfolg: 1 (siehe Bemerkungen).
Fehler: 0

Bemerkungen

Wenn _DebugSetup() nicht aufgerufen wurde, wird die Funktion nicht ausgeführt.
@error wird nicht geändert.
@extended wird zum Windows API _WinNet_GetLastError() gesetzt, wenn $bLastError = True.
Ansonsten wird es nicht geändert.

Bei jedem Aufruf von _DebugReport() wird die Ausgabe in eine eigene Zeile gestellt, wenn $bLastError = True ist.

Verwandte Funktionen

_DebugSetup

Beispiel

#include <Debug.au3>

_DebugSetup()

_DebugReport("Nachricht1")
_DebugReport("Nachricht2", True) ; Mit Anhang der letzten Fehlermeldung.
