$tagNMITEMACTIVATE

Beschreibung anzeigen in


Gesendet von einem ListView-Control wenn der Benutzer ein Element aktiviert

Global Const $tagNMITEMACTIVATE = $tagNMHDR & ";int Index;int SubItem;int NewState;int OldState;" & _
"int Changed;int X;int Y;int lParam;int KeyFlags"

Parameter

$tagNMHDR Enthält Informationen über den Benachrichtigungscode
Index Index des ListView Items. Wird der Elementindex nicht für die Benachrichtigung benutzt, enthält dieses Mitglied -1
SubItem 1-basierter Index des Subelements. Wird der Subelementindex nicht für die Benachrichtigung benutzt oder die Benachrichtigung auf ein Subelement angewandt, enthält dieses Mitglied Null.
NewState Neuer Elementstatus. Dieses Mitglied ist Null für Benachrichtigungen die dieses nicht nutzen
OldState Alter Elementstatus. Dieses Mitglied ist Null für Benachrichtigungen die dieses nicht nutzen
Changed Satz von Flags welche die Element-Attribute angeben die sich geändert haben.
Dieses Mitglied ist Null für Benachrichtigungen die dieses nicht nutzen.
Dieses Mitglied kann einen oder mehrere der folgenden Flags gesetzt haben:
    $LVIF_COLUMNS - Das Columns Mitglied ist gültig
    $LVIF_DI_SETITEM - Das Betriebssystem soll die angeforderten Elementinformationen speichern
    $LVIF_GROUPID - Das GroupID Mitglied ist gültig
    $LVIF_IMAGE - Das Image Mitglied ist gültig
    $LVIF_INDENT - Das Indent Mitglied ist gültig
    $LVIF_NORECOMPUTE - Das Control wird keine LVN_GETDISPINFO Nachricht erzeugen um Textinformationen zu ermitteln
    $LVIF_PARAM - Das Param Mitglied ist gültig
    $LVIF_STATE - Das State Mitglied ist gültig
    $LVIF_TEXT - Das Text Mitglied ist gültig
X Legt die x-Koordinate des Punktes fest
Y Legt die y-Koordinate des Punktes fest
lParam Anwendungsspezifischer Wert des Elements. Dieses Mitglied ist undefiniert für Benachrichtigungen die dieses nicht nutzen
KeyFlags Modifizierende Tasten die zur Zeit der Aktivierung gedrückt wurden.
Dieses Mitglied enthält Null oder eine Kombination der folgenden Flags:
    $LVKF_ALT (1) - Die ALT-Taste ist gedrückt.
    $LVKF_CONTROL (2) - Die CONTROL-Taste ist gedrückt.
    $LVKF_SHIFT (4) - Die SHIFT-Taste ist gedrückt.
