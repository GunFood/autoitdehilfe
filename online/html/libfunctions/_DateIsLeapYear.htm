_DateIsLeapYear



Beschreibung anzeigen in

Prüft ein Jahr um festzustellen ob es ein Schaltjahr ist

#include <Date.au3>
_DateIsLeapYear ( $iYear )

Parameter

$iYear Das 4-stellige Jahr, das geprüft werden soll.

Rückgabewert

Erfolg: 1
Fehler: 0 falls es kein Schaltjahr ist und setzt @error auf ungleich 0
@error: 1 - ungültiges Jahr.

Beispiel

#include <Date.au3>
#include <MsgBoxConstants.au3>

If _DateIsLeapYear(@YEAR) Then
    MsgBox($MB_SYSTEMMODAL, "Schaltjahr", "Das Jahr ist ein Schaltjahr.")
Else
    MsgBox($MB_SYSTEMMODAL, "Schaltjahr", "Das Jahr ist kein Schaltjahr.")
EndIf
