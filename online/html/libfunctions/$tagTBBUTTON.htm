$tagTBBUTTON

Beschreibung anzeigen in

Enthält Informationen über einen Button in einer Toolbar

Global Const $tagTBBUTTON = "int Bitmap;int Command;byte State;byte Style;short Reserved;dword_ptr Param;int_ptr String"

Parameter

Bitmap 0-basierender Index des Buttonbildes. Setzt man dies auf $I_IMAGECALLBACK, so wird die Toolbar eine $TBN_GETDISPINFO Nachricht senden, um den Bilderindex zu ermitteln, falls dieser benötigt wird.
Setzt man dies auf $I_IMAGENONE, so wird angegeben, dass der Button kein Bild enthält. Das Aussehen des Buttons enthält keinen Freiraum für eine Bitmap, sondern einfach nur Text.
Falls ein Button eine Trennung ist, so bestimmt das Bitmap die Breite der Trennung in Pixel.
Command Befehls-ID die mit dem Button verknüpft ist. Diese ID wird in einer $WM_COMMAND Nachricht verwendet wenn der Button ausgewählt wurde.
State Button-Zustandsflag
Style Button-Stilflag
Reserved Reserviert 2 Bytes im x86 Modus, 6 im x64 Modus
Param Anwendungsspezifischer Wert
String 0-basierender Index des button strings oder ein Verweis zu einem String der den Text für den Button enthält
