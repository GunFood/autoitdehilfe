Function Reference

_IE_VersionInfo



Show description in

Returns an array of information about the IE.au3 version

#include <IE.au3>
_IE_VersionInfo ( )

Return Value

Returns an array $__gaIEAU3VersionInfo:
    $__gaIEAU3VersionInfo[0] = Release Type (T=Test or V=Production)
    $__gaIEAU3VersionInfo[1] = Major Version
    $__gaIEAU3VersionInfo[2] = Minor Version
    $__gaIEAU3VersionInfo[3] = Sub Version
    $__gaIEAU3VersionInfo[4] = Release Date (YYYYMMDD)
    $__gaIEAU3VersionInfo[5] = Display Version (e.g. V3.1-0)

Remarks

The array contents make it easy to determine of the UDF version is higher or lower than a required version.

Example

; Retrieve and display IE.au3 version information

#include <IE.au3>
#include <MsgBoxConstants.au3>

Local $aVersion = _IE_VersionInfo()
MsgBox($MB_SYSTEMMODAL, "IE.au3 Version", $aVersion[5] & " released " & $aVersion[4])
