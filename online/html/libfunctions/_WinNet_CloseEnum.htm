_WinNet_CloseEnum



Beschreibung anzeigen in

Beendet die Auflistung der Netzwerk-Ressourcen, die mit durch Aufrufen der Funktion _WinNet_OpenEnum() begonnen wurde

#include <WinNet.au3>
_WinNet_CloseEnum ( $hEnum )

Parameter

$hEnum Handle, das die Auflistung der Netzwerk-Ressourcen identifiziert. Das Handle wird von der Funktion _WinNet_OpenEnum() zurückgegeben.

Rückgabewert

Erfolg: True
Fehler: False

Verwandte Funktionen

_WinNet_OpenEnum
