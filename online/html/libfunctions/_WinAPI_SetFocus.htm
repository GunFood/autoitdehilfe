_WinAPI_SetFocus

Beschreibung anzeigen in

Setzt den Fokus der Tastatur auf das angegebenen Fenster

#include <WinAPISysWin.au3>
_WinAPI_SetFocus ( $hWnd )

Parameter

$hWnd Identifiziert das Fenster, das den Fokus erhalten soll. Wenn der Wert 0 ist, werden Tastatureingaben ignoriert.

Rückgabewert

Erfolg: Das Handle des Fensters, das den Fokus der Tastatur erhalten hat
Fehler: 0. Es ist _WinAPI_GetLastError() aufzurufen um erweiterte Fehlerinformationen zu erhalten.
