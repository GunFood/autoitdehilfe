Include Konstanten für AutoIt Konstanten

Es folgt eine komplette Liste der in AutoIt verfügbaren include Konstanten für AutoIt.

    Bei der Verwendung von #include <Constants.au3> wird alles einbezogen außer der Schrift, GUI, Inet und Windows Konstanten.

Datei Verwandte Control(s)
#include <AutoItConstants.au3> AutoIt verwandte
#include <ColorConstants.au3> Color
#include <Constants.au3> AutoIt verwandte Funktionen (Include AutoIt, Color, Dir, File, MsgBox, Process, String, ...)
#include <DirConstants.au3> Dir
#include <FileConstants.au3> File
#include <FontConstants.au3> Font
#include <GUIConstantsEx.au3> GUI
#include <InetConstants.au3> Inet
#include <MsgBoxConstants.au3> MsgBox
#include <ProcessConstants.au3> Prozess verwandte
#include <StringConstants.au3> String
#include <WindowsConstants.au3> Windows
