_WinAPI_DefRawInputProc



Beschreibung anzeigen in

Ruft die standard Raw-Input-Prozedur um die Raw-Input-Nachrichten zu verarbeiten welche eine Anwendung nicht verarbeitet.

#include <WinAPISys.au3>
_WinAPI_DefRawInputProc ( $paRawInput, $iInput )

Parameter

$paRawInput Ein Zeiger zu einem Array von $tagRAWINPUT Strukturen welche von _WinAPI_GetRawInputBuffer() zurückgegeben wird.
$iInput Die Anzahl der Strukturen im Array.

Rückgabewert

Erfolg: 1.
Fehler: 0 und setzt das @error Flag auf ungleich null. @extended kann den Fehlercode enthalten.

Verwandte Funktionen

_WinAPI_GetRawInputBuffer
