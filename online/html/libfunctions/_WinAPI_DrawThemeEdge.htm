_WinAPI_DrawThemeEdge

Beschreibung anzeigen in


Zeichnet eine oder mehrere Kanten welche durch den visuellen Stil eines Rechtecks definiert sind

#include <WinAPITheme.au3>
_WinAPI_DrawThemeEdge ( $hTheme, $iPartID, $iStateID, $hDC, $tRECT, $iEdge, $iFlags [, $tAREA = 0] )

Parameter

$hTheme Das Handle, eines für ein Fenster, angegebener Themedaten.
$iPartID Der Bereich welcher das Rechteck beinhaltet.
$iStateID Der Zustand des Bereiches.
$hDC Das Handle eines Gerätekontextes.
$tRECT Die $tagRECT Struktur beinhaltet das Rechteck.
$iEdge Der Typ der inneren und äußeren zu zeichnenden Kanten.
Dieser Parameter muss eine Kombination eines innerkanten Flags und eines außerkanten Flags ($BDR_*) oder eins der Kombinationflags ($EDGE_*).
$iFlags Der Typ der zu zeichnenden Kante. Es kann eine Kombination von $BF_* Konstanten sein.
$tAREA [optional] Die $tagRECT Struktur welche das Rechteck beinhaltet welche ein inneres Rechteck erhält sofern $BF_ADJUST verwendet wird.

Rückgabewert

Erfolg: 1.
Fehler: 0 und setzt das @error Flag auf ungleich null, das @extended Flag kann den HRESULT-Fehlercode enthalten.

Bemerkungen

Die $BDR_*, $EDGE_*, $BF_* Konstanten benötigen ein #include <BorderConstants.au3>.
