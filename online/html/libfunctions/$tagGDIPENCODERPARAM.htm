$tagGDIPENCODERPARAM Struktur

Global Const $tagGDIPENCODERPARAM = "struct;byte GUID[16];ulong NumberOfValues;ulong Type;ptr Values;endstruct"

Parameter

GUID Identifiziert die Parameter Kategorie (GDI_EPG Konstanten)
NumberOfValues Anzahl der Werte in dem Array auf welches das Strukturmitglied Values verweist
Type Identifiziert den Datentyp des Parameters (GDI_EPT Konstanten)
Values Zeiger auf ein Array mit Werten. Jeder Wert hat den Typ der durch den das Strukturmitglied Type festgelegt wurde.
