_WinAPI_GetMapMode

Retrieves the current mapping mode

#include <WinAPIGdi.au3>
_WinAPI_GetMapMode ( $hDC )

Parameter

$hDC Handle to the device context.

Rückgabewert

Success: The value specifies the mapping mode (MM_*).
Failure: 0.
