$tagNMCBEENDEDIT



Beschreibung anzeigen in

Enthält Informationen über das Ergebnis einer Bearbeitung innerhalb eines ComboBoxEx Controls

Global Const $tagNMCBEENDEDIT = $tagNMHDR & ";bool fChanged;int NewSelection;wchar szText[260];int Why"

Parameter

$tagNMHDR Enthält Informationen über den Benachrichtigungscode
fChanged Weist darauf hin ob der Inhalt der Edit Box des Controls sich geändert hat
NewSelection Der 0-basierte Index des Elements das nach der Beendigung der Bearbeitung ausgewählt ist.
Dieser Wert kann $CB_ERR sein wenn kein Element ausgewählt werden soll
szText einen null-terminierten String die den Text enthält innerhalb der Edit Box des Controls
Why Der Vorgang der die $CBEN_ENDEDIT Benachrichtigung erzeugt hat
Dieser Wert kann einer der folgenden sein:
    $CBENF_DROPDOWN - Der Benutzer hat die Ausklapp-Liste aktiviert
    $CBENF_ESCAPE - Der Benutzer hat die ESC Taste gedrückt
    $CBENF_KILLFOCUS - Die Edit-Box hat den Tastaturfokus verloren
    $CBENF_RETURN - Der Benutzer hat die Bearbeitung durch drücken der ENTER Taste beendet
