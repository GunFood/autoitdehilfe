_WinAPI_SetLastError

Beschreibung anzeigen in

Setzt den letzten Fehlercode für den aufgerufenen Thread

#include <WinAPIError.au3>
_WinAPI_SetLastError ( $iErrorCode )

Parameter

$iErrorCode Der letzte Fehlercode für den Thread

Rückgabewert

Keine.

Bemerkungen

Der letzte Fehlercode wird im lokalen Gültigkeitsbereich eines Threads abgelegt, sodass bei mehreren Threads die Werte nicht überschrieben werden.
