_SQLite_SafeMode

Show description in

Disable or Enable Safe mode

#include <SQLite.au3>
_SQLite_SafeMode ( $bSafeModeState )

Parameters

$bSafeModeState True or False to enable or disable SafeMode

Return Value

Returns $SQLITE_OK.

Remarks

Safe mode prevents SQLite from crashing which can be caused by passing invalid query or database handles.
It's enabled by default.

Example

#include <SQLite.au3>
#include <SQLite.dll.au3>

_SQLite_Startup()
ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)
_SQLite_SafeMode(False)
_SQLite_Exec(-1, "CREATE tblTest (a,b,c);"); No database open, this will crash due to SafeMode = false
_SQLite_Shutdown()
