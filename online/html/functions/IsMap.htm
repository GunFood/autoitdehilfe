Funktionreferenz

IsMap

Beschreibung anzeigen in


Prüft, ob eine Variable vom Typ Map ist.

IsMap ( variable )

Parameter

variable Zu prüfende(r) Variable/Ausdruck.

Rückgabewert

Erfolg:
Fehler: 0 wenn der Parameter nicht vom Typ Map ist.

Bemerkungen

Kann nützlich sein, um Map/Non-Map-Parameter für benutzerdefinierte Funktionen zu validieren.
Siehe language datatypes für eine detaillierte Beschreibung.

Verwandte Funktionen

IsArray, IsBinary, IsBool, IsFloat, IsHWnd, IsInt, IsNumber, IsPtr, IsString, MapExists, VarGetType

Beispiel

#include <MsgBoxConstants.au3>

Local $mMap[] ; Eine Map deklarieren.
If IsMap($mMap) Then
    MsgBox($MB_SYSTEMMODAL, "", "Die Variable ist eine Map")
Else
    MsgBox($MB_SYSTEMMODAL, "", "Die Variable ist keine Map")
EndIf
