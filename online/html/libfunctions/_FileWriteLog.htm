_FileWriteLog

Beschreibung anzeigen in


Schreibt das aktuelle Datum, die aktuelle Zeit und den angegebenen Text in eine Log Datei

#include <File.au3>
_FileWriteLog ( $sLogPath, $sLogMsg [, $iFlag = -1] )

Parameter

$sLogPath Pfad der Datei, in die geschrieben werden soll, oder ein Handle einer Datei, das von FileOpen() zurückgegeben wurde.
$sLogMsg Text, der in die Log-Datei geschrieben werden soll.
$iFlag [optional] Flag welches festlegt, ob $sLogMsg an den Anfang oder das Ende der Datei geschrieben werden soll.
Falls $iFlag = -1 (Standard) wird $sLogMsg an das Ende der Datei geschrieben.
Sonst wird $sLogMsg an den Anfang der Datei geschrieben.

Rückgabewert

Erfolg: 1
Fehler: 0 und setzt das @error Flag auf ungleich null.
@Error: 1 - Die angegebene Datei konnte nicht geöffnet werden
2 - Die Datei konnte nicht geschrieben werden
@extended: 1 wenn versucht wird, am Anfang mit dem Dateihandle zu schreiben

Beispiel

Beispiel 1

#include <File.au3>

; Öffnet die Logdatei im Schreibmodus

Local $hFile = FileOpen(@TempDir & "\Example.log", 1) ; Öffnet die Logdatei im Schreibmodus

_FileWriteLog($hFile, "Text 1") ; Schreibt in die Logdatei. Das Dateihandle wurde von FileOpen zurückgegeben.
FileClose($hFile) ; Schließt das Dateihandle um die Datei wieder freizugeben.

Beispiel 2

#include <File.au3>

; Schreibt in die Logdatei durch angegeben des Dateipfades.

_FileWriteLog(@TempDir & "\Example.log", "Text 2")
