_Array1DToHistogram

Beschreibung anzeigen in

Wandelt ein 1D Array in ein Histogramm um

#include <Array.au3>
_Array1DToHistogram ( $aArray [, $iSizing = 100] )

Parameter

$aArray umzuwandelndes Array
$iSizing [optional] ermöglicht das Histogramm zu zoomen. Standard = 100 Prozent

Rückgabewert

Erfolg: das Array wurde umgewandelt.
Fehler: setzt das @error Flag auf ungleich null.
@error: 1 - $aArray ist kein 1D Array

Bemerkungen

Die ideale Ausrichtung der negativen Werte ist aufgrund der Verwendung nicht fixierter Schriftarten nicht perfekt.

Beispiel

Beispiel 1

#include <Array.au3>

Local $aNEW[] = [0, -80, -49, -44, 80, 100, 8, 7, 6, 5, 4, 3, 2, 1]
Local $aArray = _Array1DToHistogram($aNEW, 125)
_ArrayDisplay($aArray, "_Array1DToHistogram")

Beispiel 2

#include <Array.au3>

Local $iLimit = 500
Local $aValues[51]
Local $t, $n

For $i = 1 To $iLimit
    $t = TimerInit()
    Sleep(10)
    $n = Round(TimerDiff($t))
    $aValues[$n] += 1
Next
Local $aArray = _Array1DToHistogram($aValues)
_ArrayDisplay($aArray, "_Array1DToHistogram", "10:20")
