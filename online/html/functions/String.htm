Funktionreferenz

String



Beschreibung anzeigen in

Gibt einen beliebigen Ausdruck als String zurück.

String ( expression )

Parameter

expression Der Ausdruck, der in einen String konvertiert werden soll.

Rückgabewert

Gibt den String zurück.

Bemerkungen

Die maximale Stringlänge ist 2147483647 Zeichen (denke aber daran, dass keine Zeile in einem AutoIt Skript 4095 Zeichen überschreiten kann.)

Verwandte Funktionen

BinaryToString, Chr, ChrW, HWnd, Int, IsString, Number, Ptr, StringToBinary

Beispiel

#include <MsgBoxConstants.au3>

Local $sString = String(10) ; Der an String übergebene Ausdruck wird in Stringdarstellung zurückgegeben, aktuell ist 10 eine Zahl.
MsgBox($MB_SYSTEMMODAL, "", "$sString enthält den Wert: " & $sString & " und IsString gibt folgendes zurück: " & IsString($sString))
