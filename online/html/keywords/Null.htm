Schlüsselwortreferenz

Null



Beschreibung anzeigen in

Schlüsselwert welcher im Funktionsaufruf genutzt werden kann.

$var = Null

Bemerkungen

Verwendet dieses Schlüsselwort um bestimmte Funktionsweisen bei Funktionen, welche sie als Parameter akzeptieren, auszulösen.
Null wird nur als Null ausgewertet, in Mathematischen Operationen hat das Schlüsselwort immernoch den Wert 0 und als String keinen Wert (leerer String).
COM Methoden wird Null als VT_NULL VARIANT übergeben. DllCall's wstr und str Typen werden zu NULL gesetzt (pointer).

Verwandte Funktionen

Default, IsKeyword

Beispiel

; Nachrichtenfenster mit einem "Fehler"-Titel.

DllCall("user32.dll", "int", "MessageBoxW", "hwnd", 0, "wstr", "Some text", "wstr", Null, "dword", 0)
