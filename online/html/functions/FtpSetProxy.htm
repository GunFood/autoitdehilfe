FtpSetProxy

Beschreibung anzeigen in

Setzt den Internet-Proxyserver für den FTP-Zugang.

FtpSetProxy ( mode = 0 [, "proxy:port" [, "username" [, "password"]]] )

Parameter

mode Der Proxy-Modus, der verwendet werden soll:
    $PROXY_IE (0) = (Standard) Nutzt die Internet Explorer-Einstellungen für den Proxyserver.
    $PROXY_NONE (1) = Verwendet keinen Proxyserver (direkter Zugang)
    $PROXY_SPECIFIED (2) = Nutzt den angegebenen Proxyserver

Die Konstanten sind in "AutoItConstants.au3" definiert.
proxy:port [optional] Die Adresse und der Port des zu nutzenden Proxyservers. Siehe Bemerkungen weiter unten.
username [optional] Wenn benötigt, der Benutzername für den Proxyserver.
password [optional] Wenn benötigt, das Passwort für den Proxyserver.

Rückgabewert

Keine.

Bemerkungen

Es werden nur CERN-Proxy-Server unterstützt.

Proxy-Server bestehen normalerweise aus der Webadresse und dem Port. Wenn der Proxy "www.proxy.com" heißt und der Port "8080" genutzt wird, dann muss dieser Proxy-Server so gesetzt werden:
FtpSetProxy(2, "www.proxy.com:8080")

Die meisten Proxy-Server benötigen keinen Benutzernamen und Passwort.

Wenn Modus 0 verwendet wird, so kann ein Benutzername und Passwort festgelegt werden. Diese werden mit dem im Internet Explorer eingestellten Proxy verwendet.

Verwandte Funktionen

HttpSetProxy, InetGet, InetGetSize, InetRead

Beispiel

; Keinen Proxy nutzen
FtpSetProxy(1)

; IE Standards für Proxy nutzen
FtpSetProxy(0)

; Den Proxy "www-cache.myisp.net" auf Port 8080 nutzen
FtpSetProxy(2, "www-cache.myisp.net:8080")
