_WinNet_GetConnectionPerformance

Beschreibung anzeigen in


Liefert Informationen über den Durchsatz einer Netzwerkverbindung

#include <WinNet.au3>
_WinNet_GetConnectionPerformance ( $sLocalName, $sRemoteName )

Parameter

$sLocalName Name des lokalen Geräts (z.B. "F:" oder "LPT1"), das als Netzwerkressource freigegeben ist und abgefragt werden soll.
Wenn nicht angegeben, wird die Netzwerkressource in $sRemoteName verwendet.
Wenn ein lokales Gerät angegeben ist, wird $sRemoteName ignoriert.
$sRemoteName Name der Netzwerkressource, die abgefragt werden soll.
Die Verbindung zur Ressource muss bestehen.
Zum Beispiel, wenn die Ressource eine Datei auf einem Fileserver ist, dann stellt das Geöffnethalten der Datei die Verbindung sicher.

Rückgabewert

Erfolg: Ein Array mit dem folgenden Format:
    $aInfo[0] - Beschreibung der Verbindung. Kann eine Kombination der folgenden Werte sein:
        1 - Die gelieferte Information gilt für den Durchsatz der Netzwerkkarte.
        2 - Die Verbindung ist nicht geroutet.
        4 - Die Verbindung läuft über ein normalerweise langsames Medium.
        8 - Einige der gelieferten Daten werden dynamisch ermittelt.
    $aInfo[1] - Durchsatz des Mediums zur Netzwerkressource in 100 bps (Bits pro Sekunde).
    $aInfo[2] - Zeitverzögerung vor dem Senden von Informationen (in einer Richtung).
    $aInfo[3] - Größe der Daten, die ein Programm für eine Einzelanfrage an die Ressource verwenden sollte.
Fehler: Setzt das @error Flag auf ungleich null
