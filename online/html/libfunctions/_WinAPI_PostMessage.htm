_WinAPI_PostMessage



Beschreibung anzeigen in

Platziert eine Nachricht in die Nachrichten-Warteschlange und kehrt umgehend zurück

#include <WinAPISysWin.au3>
_WinAPI_PostMessage ( $hWnd, $iMsg, $wParam, $lParam )

Parameter

$hWnd Identifiziert das Fenster dessen Fensterprozedur die Nachricht erhält. Ist dieser Parameter 0xFFFF ($HWND_BROADCAST), wird die Nachricht an alle Top-Level Fenster gesendet, einschließlich deaktivierten, unsichtbaren, herrenlosen Fenstern, überlappenden Fenstern, und Pop-up Fenstern; jedoch wird die Nachricht nicht an Child-Fenster gesendet.
$iMsg Legt die Nachricht fest, die gesendet werden soll
$wParam Erster Nachrichtenparameter (LPARAM)
$lParam Zweiter Nachrichtenparameter (WPARAM)

Rückgabewert

Erfolg: True
Fehler: False. Ruft _WinAPI_GetLastError() auf, um erweiterte Fehlerinformationen zu erhalten.
