Schlüsselwortreferenz

#NoTrayIcon

Beschreibung anzeigen in

Bewirkt, dass das AutoIt-Icon nicht im Systeminfo-Bereich der Taskleiste erscheint, wenn das Skript gestartet wird.

#NoTrayIcon

Bemerkungen

Es ist möglich Opt("TrayIconHide", 1) zu verwenden um das AutoIt Tray-Icon zu entfernen, allerdings wird es für eine Sekunde sichtbar sein, wenn das Skript startet. Wird #NoTrayIcon direkt an den Anfang des Skripts geschrieben, wird das Icon gar nicht erst dargestellt.

Man kann das Icon zu einem späteren Zeitpunkt wieder anzeigen lassen Opt("TrayIconHide", 0)

Verwandte Funktionen

TrayIconHide (Option)

Beispiel

#NoTrayIcon

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Zeigt ein Nachrichtenfenster mit dem Hinweis, dass das Trayicon erschint wenn OK angeklickt wurde.
    MsgBox($MB_SYSTEMMODAL, "", "Das Tray-Icon erscheint sobal OK angeklickt wurde.")

    ; Zeigt das Trayicon.
    Opt("TrayIconHide", 0)

    ; Wartet 5 Sekunden bevor das Skript beendet wurde.
    Sleep(5000)
EndFunc   ;==>Example
