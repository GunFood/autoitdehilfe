_SQLite_ErrMsg



Beschreibung anzeigen in

Gibt einen String zurück, der in englisch die Fehlerbedingung des letzten sqlite3_* API Aufrufes beschreibt

#include <SQLite.au3>
_SQLite_ErrMsg ( [$hDB = -1] )

Parameter

$hDB [optional] Eine offene Datenbank, Standard ist die zuletzt geöffnete Datenbank.

Rückgabewert

Erfolg: die Fehlermeldung
Fehler: Setzt das @error Flag auf ungleich null
@error: 1 - Fehler beim Aufruf von SQLite API 'sqlite3_errmsg16'
2 - Aufruf wegen Sicherheitsmodus verhindert

Bemerkungen

Die SQLite-Dateien können von https://www.autoitscript.com/autoit3/pkgmgr/sqlite heruntergeladen werden. Abgespeichert können diese in @ScriptDir, @SystemDir, @WindowsDir oder @WorkingDir werden.

Verwandte Funktionen

_SQLite_ErrCode

Beispiel

#include <MsgBoxConstants.au3>
#include <SQLite.au3>
#include <SQLite.dll.au3>

_SQLite_Startup()
ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)
_SQLite_Open()
If $SQLITE_OK <> _SQLite_Exec(-1, "CREATE TABLE test ('a', 'b);") Then _ ; Ein Anführungszeichen nach b fehlt
        MsgBox($MB_SYSTEMMODAL, "SQLite Fehler", "Fehlerkode: " & _SQLite_ErrCode() & @CRLF & "Fehlermeldung: " & _SQLite_ErrMsg())
_SQLite_Close()
_SQLite_Shutdown()
