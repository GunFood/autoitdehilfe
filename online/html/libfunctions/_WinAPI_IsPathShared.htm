_WinAPI_IsPathShared

Beschreibung anzeigen in

Bestimmt, ob der Pfad gemeinsam genutzt wird

#include <WinAPIFiles.au3>
_WinAPI_IsPathShared ( $sFilePath )

Parameter

$sFilePath Der vollständig qualifizierte lokale Pfad, der getestet werden soll.

Rückgabewert

Erfolg: True - Der Pfad wird geteilt.
False - Sonstiges.
Fehler: Setzt das @error-Flag auf ungleich Null.
