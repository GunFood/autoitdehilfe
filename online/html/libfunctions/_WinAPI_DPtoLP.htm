_WinAPI_DPtoLP

Beschreibung anzeigen in

Wandelt Gerätekoordinaten in logische Koordinaten um

#include <WinAPIGdi.au3>
_WinAPI_DPtoLP ( $hDC, ByRef $tPOINT [, $iCount = 1] )

Parameter

$hDC Das Handle zum Gerätekontext.
$tPOINT Die $tagPOINT Struktur oder eine Struktur von Punkten ("long x1;long y1;...long xN;long yN") welche die X- und Y-Koordinaten beinhaltet welche umgewandelt werden sollen.
$iCount [optional] Die Anzahl an Punkten welche standardmäßig 1 beträgt.

Rückgabewert

Erfolg: True
Fehler: False

Bemerkungen

Die _WinAPI_DPtoLP() Funktion schlägt fehl, wenn Gerätekoordinaten 27 Bits überschreiten oder wenn die umgewandelten logischen Koordinaten 32 Bits überschreiten.
Im Falle eines Überlaufs sind die Ergebnisse aller Punkte undefiniert.
