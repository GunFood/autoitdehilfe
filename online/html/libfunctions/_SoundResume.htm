_SoundResume



Beschreibung anzeigen in

Setzt einen pausierenden Sound fort

#include <Sound.au3>
_SoundResume ( $aSndID )

Parameter

$aSndID Sound ID Array welches durch _SoundOpen() oder einem Dateinamen zurückgegeben wurde

Rückgabewert

Erfolg: 1
Fehler: 0 und setzt @error auf ungleich 0
@Error: 1 = Pausefehler
3 = Ungültige Sound ID. Es ist das durch _SoundOpen() zuürckgegebene Array oder ein Dateinamen zu verwenden.

Bemerkungen

_SoundPlay() wird in manchen Fällen ebenfalls einen pausierenden Sound wieder weiter abspielen.

Verwandte Funktionen

_SoundOpen, _SoundPause, _SoundPlay

Beispiel

#include <MsgBoxConstants.au3>
#include <Sound.au3>

Local $sound = _SoundOpen(@WindowsDir & "\media\Windows XP Startup.wav")
If @error = 2 Then
    MsgBox($MB_SYSTEMMODAL, "Fehler", "Die Datei existiert nicht.")
    Exit
ElseIf @extended <> 0 Then
    Local $iExtended = @extended ; Zuweisung, weil @extended nach DllStructCreate() geändert wird
    Local $stText = DllStructCreate("char[128]")
    DllCall("winmm.dll", "short", "mciGetErrorStringA", "str", $iExtended, "struct*", $tText, "int", 128)
    MsgBox($MB_SYSTEMMODAL, "Fehler", "Öffnen fehlgeschlagen." & @CRLF & "Fehlernummer: " & $iExtended & @CRLF & "Fehlerbeschreibung: " & DllStructGetData($stText, 1) & @CRLF & "Beachte: Möglicherweise kann der Sound dennoch korrekt abgespielt werden.")
Else
    MsgBox($MB_SYSTEMMODAL, "Erfolgsmeldung", "Die Datei wurde erfolgreich geöffnet.")
EndIf

_SoundPlay($sound)
Sleep(1000)

_SoundPause($sound)
Sleep(1000)

_SoundResume($sound)

While 1
    Sleep(100)
    If _SoundPos($sound, 2) >= _SoundLength($sound, 2) Then ExitLoop
WEnd

_SoundClose($sound)
