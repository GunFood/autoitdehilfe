_NamedPipes_CreatePipe



Beschreibung anzeigen in

Erzeugt eine anonyme Pipe

#include <NamedPipes.au3>
_NamedPipes_CreatePipe ( ByRef $hReadPipe, ByRef $hWritePipe [, $tSecurity = 0 [, $iSize = 0]] )

Parameter

$hReadPipe Variable, in die das Lesehandle für die Pipe geschrieben werden soll.
$hWritePipe Variable, in die das Schreibhandle für die Pipe geschrieben werden soll.
$tSecurity [optional] tagSECURITY_ATTRIBUTES Struktur die ermittelt, ob das zurückgegebene Handle durch den child Prozess übernommen werden kann
Falls 0, kann das Handle nicht übernommen werden.
$iSize [optional] Die Größe des Puffers für die Pipe in Bytes.
Falls 0, benutzt das System die Standardgröße.

Rückgabewert

Erfolg: True
Fehler: False

Verwandte Funktionen

$tagSECURITY_ATTRIBUTES, _NamedPipes_CreateNamedPipe
