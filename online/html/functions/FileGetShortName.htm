FileGetShortName

Beschreibung anzeigen in

Gibt den angegebenen Dateinamen und -pfad im Format 8.3 zurück.

FileGetShortName ( "filename" [, flag = 0] )

Parameter

filename Vollständiger Pfad und Dateiname, der umgewandelt werden soll.
flag [optional]
    $FN_FULLPATH (0) - Standard
    $FN_RELATIVEPATH (1) - wenn im ersten Parameter ("filename") ein relativer Pfad verwendet wird, z. B. "..\..\file.txt".

Die Konstanten sind in FileConstants.au3 definiert.

Rückgabewert

Erfolg: Der angegebenen Dateinamen und -pfad im Format 8.3.
Fehler: Der Parameter "filename". Setzt @error auf 1.

Bemerkungen

Die Datei muss existieren, da es keine Möglichkeit gibt, den exakten "~n" zu kennen, wenn mehrere Dateien die gleichen 8 ersten Zeichen haben.

Verwandte Funktionen

FileGetLongName

Beispiel

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Erstellt eine Konstante im lokalen Bereich des Dateipfads um den Dateinamen im Format 8.3 zu ermitteln.
    Local Const $sFilePath = @ProgramFilesDir

    ; Zeigt den Pfad und Dateiname im Format 8.3 des Programm-Ordners.
    MsgBox($MB_SYSTEMMODAL, "", FileGetShortName($sFilePath))
EndFunc   ;==>Example
