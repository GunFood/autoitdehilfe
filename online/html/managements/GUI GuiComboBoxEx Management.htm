GuiComboBoxEx Benutzerdefinierte Funktionsreferenz

Es folgt eine Liste der benutzerdefinierten Funktionen die in AutoIt verfügbar sind. Ein Klick auf einen Funktionsnamen liefert eine detaillierte Beschreibung.

Wenn diese Funktion verwendet wird, so muss #include <GuiComboBoxEx.au3> hinzugefügt werden.

_GUICtrlComboBoxEx_GetComboControl Ermittelt das Handle des Unter-Combobox Controls
