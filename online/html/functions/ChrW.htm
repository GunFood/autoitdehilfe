ChrW

Beschreibung anzeigen in

Gibt ein Zeichen entsprechend eines Unicode-Codes wieder.

ChrW ( UNICODEcode )

Parameter

UNICODEcode Ein Unicode-Code im Bereich von 0-65535 (z.B.: 65 gibt den Großbuchstaben "A" wieder).

Rückgabewert

Erfolg: Einen String mit entsprechendem Unicode-Code-Zeichen.
Fehler: Einen leeren String und setzt das @error Flag auf ungleich null wenn der UNICODE-Wert größer als 65535 ist.

Bemerkungen

Siehe die ASCII Character Code Tabelle für eine komplette Liste der verfügbaren Werte.

Verwandte Funktionen

Asc, AscW, Chr, String

Beispiel

#include <MsgBoxConstants.au3>

Local $sText = ""
For $i = 256 To 512
    $sText = $sText & ChrW($i) ; Oder $sText &= ChrW($i) kann genauso verwendet werden
Next
MsgBox($MB_SYSTEMMODAL, "Unicode Zeichen 256 bis 512", $sText) ;Zeigt die Unicode Zeichen zwischen 256 und 512
