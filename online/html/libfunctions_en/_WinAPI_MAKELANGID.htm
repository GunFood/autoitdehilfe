Construct language id from a primary language id and a sublanguage id

#include <WinAPIConv.au3>
_WinAPI_MAKELANGID ( $iLngIDPrimary, $iLngIDSub )

Parameters

$iLngIDPrimary Primary Language id
$iLngIDSub Sub-Language id

Return Value

Returns the Language identifier

Remarks

Constants for $iLngIDPrimary or $iLngIDSub are defined in WinAPILangConstants.au3.
