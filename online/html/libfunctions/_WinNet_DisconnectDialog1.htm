_WinNet_DisconnectDialog1

Beschreibung anzeigen in


Startet eine allgemeine Browsing-Dialog-Box für das Trennen von Netzwerk-Ressourcen

#include <WinNet.au3>
_WinNet_DisconnectDialog1 ( $hWnd, $sLocalName [, $sRemoteName = "" [, $iFlags = 1]] )

Parameter

$hWnd Handle zu dem Eigentümerfenster der Dialogbox
$sLocalName Name eines lokalen Gerätes, wie z.B. "F:" oder "LPT1"
$sRemoteName [optional] Name der Netzwerk-Ressource, die getrennt werden soll. Dieser Wert kann 0 sein, wenn sLocalName angegeben wird.
Ist $sLocalName angegeben, wird die Verbindung zu der Netzwerk-Ressource, die von $sLocalName umgeleitet wurde, getrennt.
$iFlags [optional] Flags beschreiben die Verbindung. Kann eine Kombination der folgenden sein:
    1 - Wird dieser Wert gesetzt, ist die angegebene Verbindung nicht länger eine dauerhafte Verbindung. Dieses Flag ist nur gültig, wenn
    sLocalName ein lokales Gerät spezifiziert.
    2 - Wenn dieser Wert NICHT gesetzt ist, wird das System das Trennen erzwingen

Rückgabewert

Erfolg: Gibt 1 zurück. Falls der Benutzer die Dialogbox abbricht ist der Rückgabewert -1.
Fehler: Gibt 0 zurück und setzt @error auf einen Wert ungleich 0.
