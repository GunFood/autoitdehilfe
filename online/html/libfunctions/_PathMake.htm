_PathMake

Beschreibung anzeigen in

Erstellt einen Pfad aus einem Laufwerk, Verzeichnis, Dateinamen und einer Dateierweiterung

#include <File.au3>
_PathMake ( $sDrive, $sDir, $sFileName, $sExtension )

Parameter

$sDrive Laufwerk (Kann auch UNC sein). Falls es sich um einen Laufwerksbuchstaben handelt, wird der : automatisch angehängt
$sDir Verzeichnis. Ein abschließender und vorhergehender Slash werden automatisch hinzugefügt, falls dieser nicht existiert.
$sFileName Der Dateiname
$sExtension Die Dateierweiterung. Der Punkt wird vorangestellt, falls die Erweiterung diesen nicht enthält

Rückgabewert

Gibt den String mit dem kompletten Pfad zurück

Bemerkungen

Der Pfad wird mit den Elementen erstellt, so wie sie übergeben werden. Die Gültigkeit des resultierenden Pfades wird nicht geprüft, es könnten also ungültige Zeichen enthalten sein.
Es muss für jeden Teil des Pfads "" (ein leerer String) übergeben werden, wenn dieser Teil nicht in der Ausgabe erscheinen soll.

Verwandte Funktionen

_PathFull, _PathSplit

Beispiel

#include <File.au3>
#include <MsgBoxConstants.au3>

Local $sTestPath = _PathMake("c:", "\temp", "boot", "ini")
MsgBox($MB_SYSTEMMODAL, "demo _PathMake", $sTestPath)
