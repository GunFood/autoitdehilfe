_WinAPI_DeregisterShellHookWindow



Beschreibung anzeigen in

Deregistriert ein gegebenes Shellfenster welches registriert wurde um Shellhooknachrichten zu erhalten.

#include <WinAPISysWin.au3>
_WinAPI_DeregisterShellHookWindow ( $hWnd )

Parameter

$hWnd Das Handle eines Fensters welches von dem Erhalten von Shellhooknachrichten deregistriert werden soll.

Rückgabewert

Erfolg: True.
Fehler: False.
