_WinAPI_GetSystemWow64Directory



Beschreibung anzeigen in

Ermittelt den Pfad des von WOW64 verwendeten Systemverzeichnisses

#include <WinAPISys.au3>
_WinAPI_GetSystemWow64Directory ( )

Rückgabewert

Erfolg: Der Pfad des Systemverzeichnisses. Dieser Pfad endet nicht mit einem Backslash.
Fehler: Setzt das @error Flag auf ungleich null, das @extended Flag kann den letzter Fehlercode enthalten.

Bemerkungen

Die Funktion schlägt unter 32-Bit-Windows immer fehl, @extended ist auf ERROR_CALL_NOT_IMPLEMENTED gesetzt.
