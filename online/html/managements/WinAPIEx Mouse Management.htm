Mouse Benutzerdefinierte Funktionsreferenz

Es folgt eine Liste der benutzerdefinierten Funktionen die in AutoIt verfügbar sind. Ein Klick auf einen Funktionsnamen liefert eine detaillierte Beschreibung.

Wenn diese Funktion verwendet wird, so muss #include <WinAPIMisc.au3> hinzugefügt werden.

 

Benutzerdefinierte Funktion Beschreibung
_WinAPI_GetMousePos Gibt die derzeitige Mausposition zurück
_WinAPI_GetMousePosX Gibt die aktuelle X-Koordinate der Maus zurück
_WinAPI_GetMousePosY Gibt die aktuelle Y-Koordinate der Maus zurück
