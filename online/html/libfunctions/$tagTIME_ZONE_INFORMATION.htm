$tagTIME_ZONE_INFORMATION

Legt Informationen für eine bestimmte Zeitzone fest

Global Const $tagTIME_ZONE_INFORMATION = "struct; long Bias;wchar StdName[32];word StdDate[8];long StdBias;wchar DayName[32];word DayDate[8];long DayBias; endstruct"

Parameter

Bias Die aktuelle Zeitverschiebung für die lokale Zeitübersetzung auf diesem Computer in Minuten
StdName Eine Beschreibung für die Standardzeit. Zum Beispiel: "EST" könnte für Eastern Standard Time stehen.
StdDate Eine SYSTEMTIME-Struktur, welche ein Datum und eine lokale Zeit enthält, wann die Umstellung von der Sommerzeit auf die Standardzeit auf diesem Betriebssystem auftritt.
StdBias Die Zeitverschiebung, die bei der Umstellung der lokalen Zeit auf die Standardzeit verwendet wird
DayName Eine Beschreibung für die Sommerzeit. Zum Beispiel: "PDT" könnte für Pacific Daylight Time stehen.
DayDate Eine SYSTEMTIME-Struktur, welche ein Datum und eine lokale Zeit enthält, wann die Umstellung von der Standardzeit auf die Sommerzeit auf diesem Betriebssystem auftritt.
DayBias Die Zeitverschiebung, die bei der Umstellung der lokalen Zeit verwendet wird, welcher während der Sommerzeit auftritt
