Beschreibung anzeigen in


Die FarbMatrix Struktur beinhaltet eine 5x5 Matrix mit Realen Zahlen. Mehrere Methoden der ImageAttributes Klasse beeinflussen Bildfarben durch nutzen einer Farbmatrix

Global Const $tagGDIPCOLORMATRIX = "float m[25]"

Parameter

m beinhaltet Informationen über die Farbmatrix

Bemerkungen

Eine 5x5 Farbmatrix ist eine homogene Matrix für eine 4D Transformation.
Das Element in der fünften Zeile und Spalte der homogenen 5x5 Matrix muss 1 sein und alle anderen Elemente in der fünften Reihe und Spalte muss 0 sein.
x x x x 0
x x x x 0
x x x x 0
x x x x 0
0 0 0 0 1
Die Farbmatrizen werden genutzt um Farb-Vektoren zu Transformieren.
Die ersten 4 Komponenten eines Farb-Vektors enthalten die Rot, Grün, Blau und Alpha Werte (in der Reihenfolge) von einer Farbe.
Die fünfte Komponente eines Farbvektors ist immer 1.
