_Word_DocAdd

Beschreibung anzeigen in

Fügt ein neues oder leeres Dokument zur angegebenen Word Anwendung hinzu

#include <Word.au3>
_Word_DocAdd ( $oAppl [, $iDocumentType = $WdNewBlankDocument [, $sDocumentTemplate = "" [, $bNewTemplate = False]]] )

Parameter

$oAppl Word Objekt wie es von einem vorherigen Aufruf von _Word_Create() zurückgegeben wurde
$iDocumentType [optional] Kann eine der folgenden Konstanten der WdNewDocumentType Aufzählung sein:
    $WdNewBlankDocument, $WdNewEmailMessage, $WdNewFrameset oder $WdNewWebPage (Standard = $WdNewBlankDocument)
$sDocumentTemplate [optional] Der Name der Vorlage die für das neue Dokument verwendet werden soll.
Wenn dieses Argument ausgelassen wurden, dann wird die Normal-Vorlage verwendet (Standard = "")
$bNewTemplate [optional] True wenn das Dokument als Vorlage geöffnet werden soll (Standard = False)

Rückgabewert

Erfolg: Eine Variable die auf das neue Dokumenten-Objekt verweist.
Fehler: 0 und setzt das @error Flag auf ungleich null.
@error: 1 - $oAppl ist kein Objekt
2 - Die angegebene Vorlage existiert nicht
3 - Ein Fehler wird von der Methode Documents.Add zurückgegeben. @extended wird auf den COM Fehlercode gesetzt

Verwandte Funktionen

_Word_DocClose, _Word_DocSave, _Word_DocSaveAs

Beispiel

#include <MsgBoxConstants.au3>
#include <Word.au3>

; Erzeugt ein neues oder stellt eine Verbindung zu einem bestehenden Word-Objekt her
Local $oWord = _Word_Create()
If @error <> 0 Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocAdd Beispiel", _
        "Fehler beim Erstellen eines neuen Word-Anwendungsobjektes." & @CRLF & "@error = " & @error & ", @extended = " & @extended)

; Ein neues leeres Dokument hinzufügen
_Word_DocAdd($oWord)
If @error Then Exit MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocAdd Beispiel", "Fehler beim Erstellen eines neuen Word-Dokuments." _
         & @CRLF & "@error = " & @error & ", @extended = " & @extended)
MsgBox($MB_SYSTEMMODAL, "Word UDF: _Word_DocAdd Beispiel", "Ein neues leeres Dokument wurde erfolgreich hinzugefügt.")
