SplashImageOn



Beschreibung anzeigen in

Erzeugt ein einstellbares Bild-Anzeigefenster.

SplashImageOn ( "title", "file" [, width [, height [, x pos [, y pos [, opt]]]]] )

Parameter

title Titel des Splash Fensters.
file Kompletter Pfad\Dateiname der Bilddatei (BMP, GIF oder JPG).
width [optional] Breite des Fensters in Pixel (Standard ist 500).
height [optional] Höhe des Fensters in Pixel (Standard ist 400).
x pos [optional] Position von links (in Pixel - Standard ist zentriert).
y pos [optional] Position von oben (in Pixel - Standard ist zentriert).
opt [optional] Standard ist "immer im Vordergrund/mit Titel"
Addieren Sie die benötigten Optionen:
    $DLG_NOTITLE (1) = Fenster ohne Titel mit schmalen Außenlinien.
    $DLG_NOTONTOP (2) = Ohne "immer im Vordergrund" Attribut.
    $DLG_MOVEABLE (16) = Fenster kann bewegt werden.

Die Konstanten sind in AutoItConstants.au3 definiert.

Rückgabewert

Keine.

Bemerkungen

Um einen optionalen Parameter zu überspringen, also den Standardwert zu belassen, ist der Wert -1 einzusetzen.

Das Bild ist auf die angegebene Breite und Höhe skaliert und sollte eine Bitmap, GIF oder JPEG Bilddatei sein. Icons (*.ico) werden nicht angezeigt, obwohl das Fenster trotzdem erscheint. PNG Bilddateien geben eine Fehlermeldung mit "Fehler beim Laden" aus. Diese wird ebenfalls bei ungültigen, fehlerhaften Dateien, Dateiname und Pfaden ausgegeben.

Bilddateien werden nicht in Exe-Dateien eingebunden, wenn nicht FileInstall() benutzt wird.

Nur ein SplashImage/Text Fenster ist zur selben Zeit erlaubt. Sollen verschiedene Image/Text Fenster durchgegangen werden, ist SplashImageOn()/SplashTextOn() wiederholt mit den neuen Informationen aufzurufen.

Falls opt auf 1 gesetzt wird, kann das Fenster nicht bewegt und auch nicht durch einen Klick aktiviert werden.

Verwandte Funktionen

SplashOff, SplashTextOn

Beispiel

Local $sDestination = "..\GUI\mslogo.jpg"

SplashImageOn("Splash Fenster", $sDestination, 250, 50)
Sleep(3000)
SplashOff()
