_WinAPI_ShellGetIconOverlayIndex

Beschreibung anzeigen in

Ruft den Index des Overlay-Symbols in der Systembildliste ab

#include <WinAPIShellEx.au3>
_WinAPI_ShellGetIconOverlayIndex ( $sIcon, $iIndex )

Parameter

$sIcon Der vollständig qualifizierte Pfad zu der Datei, die das Symbol enthält.
$iIndex Der Index des Symbols. Um ein Standard-Overlay-Symbol anzufordern, setzt man den Pfad auf einen leeren String und den Index auf einen der folgenden Werte.
    $IDO_SHGIOI_LINK
    $IDO_SHGIOI_SHARE
    $IDO_SHGIOI_SLOWFILE

Windows 7 oder höher
    $IDO_SHGIOI_DEFAULT

Rückgabewert

Erfolg: Der Index des Overlay-Symbols.
Fehler: (-1).

Bemerkungen

Um diese Funktion zu verwenden, muss man _WinAPI_FileIconInit() beim Start der Anwendung aufrufen.

Verwandte Funktionen

_WinAPI_FileIconInit
