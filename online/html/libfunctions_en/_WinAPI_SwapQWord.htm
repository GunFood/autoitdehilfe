Function Reference

_WinAPI_SwapQWord

Converts a ULONGLONG from little-endian to big-endian, and vice versa

#include <WinAPIConv.au3>
_WinAPI_SwapQWord ( $iValue )

Parameters

$iValue The ULONGLONG value to convert.

Return Value

Returns the converted value.

Example

#include <WinAPIConv.au3>

Local $iValue = _WinAPI_MakeQWord(0x55667788, 0x11223344)

ConsoleWrite('0x' & Hex($iValue) & @CRLF)
ConsoleWrite('0x' & Hex(_WinAPI_SwapQWord($iValue)) & @CRLF)
