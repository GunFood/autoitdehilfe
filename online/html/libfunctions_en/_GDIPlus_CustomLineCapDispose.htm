#include <GDIPlus.au3>
_GDIPlus_CustomLineCapDispose ( $hCap )

Parameters

$hCap Handle to a custom line cap object

Return Value

Success: True.
Failure: False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code (GDIP_ERR*).
