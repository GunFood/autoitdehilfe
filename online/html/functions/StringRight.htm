StringRight



Beschreibung anzeigen in

Gibt von dem String eine Anzahl von Zeichen von rechts ausgehend zurück.

StringRight ( "string", count )

Parameter

string Der auszuwertende String.
count Die Anzahl der Zeichen, die zurückgegeben werden sollen.

Rückgabewert

Gibt einen String zurück, der die von rechts ausgehende Anzahl an Zeichen enthält.

Bemerkungen

Wenn count die Länge des Strings überschreitet, wird der gesamte String zurückgegeben.
Ist count negativ, wird ein leerer String zurückgegeben.

Verwandte Funktionen

StringCompare, StringInStr, StringLeft, StringLen, StringLower, StringMid, StringReplace, StringSplit, StringTrimLeft, StringTrimRight, StringUpper

Beispiel

#include <MsgBoxConstants.au3>

Local $sString = StringRight("Dies ist ein Satz mit Leerzeichen.", 5) ; Erfasst 5 Zeichen von der rechten Seite des Strings.
MsgBox($MB_SYSTEMMODAL, "", "Die 5 Zeichen von rechts lauten: " & $sString)
