Function Reference

_WinAPI_IsIconic

Determines whether the specified window is minimized (iconic)

#include <WinAPISysWin.au3>
_WinAPI_IsIconic ( $hWnd )

Parameters

$hWnd Handle to the window to test.

Return Value

True: the window is iconic.
False: the window is not iconic.
