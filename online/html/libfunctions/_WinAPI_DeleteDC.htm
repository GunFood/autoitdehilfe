_WinAPI_DeleteDC



Beschreibung anzeigen in

Löscht den festgelegten Gerätekontext

#include <WinAPIGdiDC.au3>
_WinAPI_DeleteDC ( $hDC )

Parameter

$hDC Identifiziert den zu löschenden Gerätekontext

Rückgabewert

Erfolg: True
Fehler: False

Bemerkungen

Eine Anwendung muss einen Gerätekontext nicht löschen dessen Handle durch die _WinAPI_GetDC() Funktion entstand.
Stattdessen muss _WinAPI_ReleaseDC() aufgerufen werden um den Gerätekontext wieder freizugeben.

Verwandte Funktionen

_WinAPI_CreateCompatibleDC, _WinAPI_GetDC, _WinAPI_ReleaseDC
