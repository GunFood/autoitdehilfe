Process Benutzerdefinierte Funktionsreferenz

Es folgt eine Liste der benutzerdefinierten Funktionen die in AutoIt verfügbar sind. Ein Klick auf einen Funktionsnamen liefert eine detaillierte Beschreibung.

Wenn diese Funktion verwendet wird, so muss #include <Process.au3> hinzugefügt werden.

 

Benutzerdefinierte Funktion Beschreibung
_ProcessGetName Gibt einen String zurück, der den Prozessnamen der angegebenen PID beinhaltet
_ProcessGetPriority Gibt die Priorität eines Prozesses zurück
_RunDos Führt einen DOS-Befehl in einem nicht sichtbaren Fenster aus
