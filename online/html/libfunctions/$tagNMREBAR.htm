Schlüsselwortreferenz

$tagNMREBAR

Beschreibung anzeigen in

Enthält Informationen, die in der Behandlung von diversen Rebar Benachrichtigungen genutzt werden

Global Const $tagNMREBAR = $tagNMHDR & ";dword dwMask;uint uBand;uint fStyle;uint wID;lparam lParam"

Parameter

$tagNMHDR Enthält Informationen über den Benachrichtigungscode
dwMask Satz von Flags, die definieren welche Mitglieder dieser Struktur gültige Informationen enthalten.
Dies kann einer oder mehrere der folgenden Werte sein:
    $RBNM_ID - Das wID Mitglied enthält gültige Informationen.
    $RBNM_LPARAM - Das lParam Mitglied enthält gültige Informationen.
    $RBNM_STYLE - Das fStyle Mitglied enthält gültige Informationen.
uBand 0-basierter Index der von der Benachrichtigung betroffenen Gruppe. Dies ist -1 wenn keine Gruppe betroffen ist.
fStyle Der Stil der Gruppe. Dies ist einer oder mehrere $RBBS_ Stile detailliert in dem fStyle Mitglied der $tagREBARBANDINFO Struktur.
Dieses Mitglied ist nur gültig, wenn dwMask $RBNM_STYLE enthält.
wID Anwendungsdefinierter Identifizierer der Gruppe. Dieses Mitglied ist nur gültig, wenn dwMask $RBNM_ID enthält.
lParam Anwendungsdefinierter Wert, verknüpft mit der Gruppe. Dieses Mitglied ist nur gültig, wenn dwMask $RBNM_LPARAM enthält.

Verwandte Funktionen

$tagNMHDR, $tagREBARBANDINFO
