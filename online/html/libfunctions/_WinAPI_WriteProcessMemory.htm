Schreibt Speicherinhalt in einen bestimmten Prozess

#include <WinAPIMem.au3>
_WinAPI_WriteProcessMemory ( $hProcess, $pBaseAddress, $pBuffer, $iSize, ByRef $iWritten [, $sBufferType = "ptr"] )

Parameter

$hProcess Identifiziert ein offenes Prozesshandle, in dessen Speicher geschrieben werden soll
$pBaseAddress Zeigt auf die Basisadresse des Prozesses, in dessen Speicher geschrieben werden soll
$pBuffer Zeigt auf den Puffer, der die zu schreibenden Daten bereitstellt
$iSize Gibt die Anzahl der zu schreibenden Bytes an
$iWritten Die aktuelle Anzahl der Bytes, welche in den Prozess transferiert wurden
$sBufferType [optional] Enthält den Datentyp, den $pBuffer repräsentiert

Rückgabewert

Erfolg: True
Fehler: False. Es ist _WinAPI_GetLastError() aufzurufen um erweiterte Fehlerinformationen zu erhalten.

Verwandte Funktionen
