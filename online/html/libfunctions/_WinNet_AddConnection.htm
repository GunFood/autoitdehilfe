_WinNet_AddConnection

Verbindet ein lokales Laufwerk mit einer Netzwerk Ressource

#include <WinNet.au3>
_WinNet_AddConnection ( $sLocalName, $sRemoteName [, $sPassword = 0] )

Parameter

$sLocalName Name des lokalen Laufwerkes das umgeleitet wird, wie zum Beispiel "F:" oder "LPT1".
Bei dem String wird keine Groß-Kleinschreibung beachtet.
Wenn der Parameter 0 ist, wird eine Verbindung aufgebaut, ohne das lokale Laufwerk umzuleiten.
$sRemoteName Name der Netzwerkressource, zu der verbunden werden soll.
$sPassword [optional] Passwort, das benutzt werden soll, um eine Verbindung herzustellen.
Dieser Parameter ist normalerweise das Passwort, das mit dem aktuellen Benutzer verknüpft ist.
Wenn der Parameter 0 ist, wird das Standard Passwort verwendet.
Ein leerer String bedeutet, dass kein Passwort verwendet wird.

Rückgabewert

Erfolg: True
Fehler: False

Bemerkungen

Diese Funktion wird nur noch aus Gründen der Kompatibilität mit 16-Bit Versionen von Windows angeboten.
Anwendungen sollten die Funktionen $tRECT oder _WinNet_AddConnection3() aufrufen.
Eine erfolgreiche Verbindung besteht dauerhaft, d.h. das System stellt die Verbindung während nachfolgender Anmeldeoperationen automatisch wieder her.

Verwandte Funktionen

_WinNet_AddConnection2, _WinNet_AddConnection3
