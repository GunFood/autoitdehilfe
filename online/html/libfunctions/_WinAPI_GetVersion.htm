Funktionreferenz

_WinAPI_GetVersion



Beschreibung anzeigen in

Ermittelt die Version des aktuellen Betriebssystems

#include <WinAPISys.au3>
_WinAPI_GetVersion ( )

Rückgabewert

Gibt einen String mit der aktuellen OS Version zurück.

    "6.3" - Windows 8.1
    "6.2" - Windows 8 / Windows Server 2012
    "6.1" - Windows 7 / Windows Server 2008 R2
    "6.0" - Windows Vista / Windows Server 2008
    "5.2" - Windows XP 64-bit Edition / Windows Server 2003 / Windows Server 2003 R2
    "5.1" - Windows XP

Beispiel

#include <WinAPISys.au3>

ConsoleWrite('Windowsversion: ' & _WinAPI_GetVersion() & @CRLF)
