_DebugOut



Beschreibung anzeigen in

Schreibt die Ausgabe einer Debugging-Session, welche durch _DebugSetup() gestartet wurde

#include <Debug.au3>
_DebugOut ( $sOutput )

Parameter

$sOutput Der String (oder ein anderer druckbarer Wert), der an die Debugging-Session ausgegeben werden soll.

Rückgabewert

Erfolg: 1 (siehe Bemerkungen)
Fehler: gibt 0 zurück und setzt das @error Flag auf ungleich null
@Error: 1 - $sOutput Typ ist inkompatibel
3 - _DebugSetup() kann nicht richtig ausgeführt werden.

Bemerkungen

gibt immer @error und @extended zurück.

Immer wenn _DebugOut() aufgerufen wird, wird die Ausgabe in eine eigenen Zeile gesetzt.

Verwandte Funktionen

_DebugSetup

Beispiel

#cs ----------------------------------------------------------------------------
    
    AutoIt Version:     3.2.8.1
    Autor:              David Nuttall
    
    Skriptfunktion:     Dieses Grundlagenskript zeigt die Funktionalität der Debug-Funktionen
    
#ce ----------------------------------------------------------------------------

#include <Debug.au3>

_DebugSetup("Teste Excel")
For $i = 1 To 4
    WinActivate("Microsoft Excel")
    ; Mit Excel arbeiten
    Send("{Down}")
    _DebugOut("Maus wurde nach unten bewegt.")
