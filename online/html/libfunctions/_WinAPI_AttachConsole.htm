_WinAPI_AttachConsole

Beschreibung anzeigen in

Heftet den aufgerufenen Prozess an der Console des festgelegten Prozesses an

#include <WinAPIProc.au3>
_WinAPI_AttachConsole ( [$iPID = -1] )

Parameter

$iPID [optional] ID des Prozesses. Auf -1 setzen, um es dem aktuellen Prozess anzuheften.

Rückgabewert

Erfolg: True
Fehler: False
