_WinNet_GetNetworkInformation

Beschreibung anzeigen in

Gibt erweiterte Informationen zu einem bestimmten Netzwerk-Provider

#include <WinNet.au3>
_WinNet_GetNetworkInformation ( $sName )

Parameter

$sName Der Netzwerk-Provider zu welchem Informationen benötigt werden

Rückgabewert

Erfolg: Ein Array mit dem folgenden Format:
$aInfo[0] - Versionsnummer der Netzwerk-Providersoftware
$aInfo[1] - Aktueller Status der Netzwerk-Providersoftware:
 0 - Das Netzwerk läuft
 1 - Das Netzwerk ist unerreichbar
 2 - Das Netzwerk ist momentan nicht fähig Anfragen zu bearbeiten
$aInfo[2] - Instanz-Handle für den Netzwerk-Provider
$aInfo[3] - Höherwertiges WORD des Netzwerktyps, einzigartig zu dem laufenden Netzwerk
$aInfo[4] - Eine Reihe von Bit-Flags aus denen sich die gültigen Drucknummern für das Weiterleiten der lokalen Drucker ergeben, mit dem niederwertigen Bit entsprechend zu LPT1.
$aInfo[5] - Satz von Bit-Flags hinweisend auf die gültigen lokalen Disketten-Laufwerke, die für umleitenden Disketten-Laufwerke genutzt werde können, mit dem niederwertigen Bit entsprechend zu A:.
Fehler: Setzt das @error Flag auf ungleich null
