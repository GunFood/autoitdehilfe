_WinAPI_PointFromRect



Beschreibung anzeigen in

Gibt die obere und linken Koordinate einer $tagRECT Struktur als eine $tagPOINT Struktur zurück

#include <WinAPIConv.au3>
_WinAPI_PointFromRect ( ByRef $tRECT [, $bCenter = True] )

Parameter

$tRECT $tagRECT Struktur
$bCenter [optional] Falls True, wird ein Punkt in der Mitte des Rechtecks zurückgegeben, sonst werden die linke/obere
Koordinate zurückgegeben.

Rückgabewert

Gibt eine $tagPOINT Struktur zurück

Bemerkungen

Diese Funktion wird verwendet um die Klickposition für viele der Klickfunktionen zu ermitteln

Verwandte Funktionen

$tagPOINT, $tagRECT
