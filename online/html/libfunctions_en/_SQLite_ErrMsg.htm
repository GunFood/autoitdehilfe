Function Reference

_SQLite_ErrMsg



Show description in

Returns a string describing in english the error condition for the most recent sqlite3_* API call

#include <SQLite.au3>
_SQLite_ErrMsg ( [$hDB = -1] )

Parameters

$hDB [optional] An open database, default is the last opened database

Return Value

Success: an error message string.
Failure: sets the @error flag to non-zero.
@error: 1 - error calling SQLite API 'sqlite3_errmsg16'
2 - Call Prevented by SafeMode

Related

_SQLite_ErrCode

Example

#include <MsgBoxConstants.au3>
#include <SQLite.au3>
#include <SQLite.dll.au3>

_SQLite_Startup()
ConsoleWrite("_SQLite_LibVersion=" & _SQLite_LibVersion() & @CRLF)
_SQLite_Open()
