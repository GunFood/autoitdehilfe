_WinAPI_CoTaskMemRealloc

Beschreibung anzeigen in

Ändert die Größe eines zuvor zugewiesenen Blocks des Aufgabenspeichers

#include <WinAPICom.au3>
_WinAPI_CoTaskMemRealloc ( $pMemory, $iSize )

Parameter

$pMemory Der Zeiger auf den Speicherblock, der neu zugewiesen werden soll.
$iSize Die Größe des neu zuzuweisenden Speicherblocks in Bytes.

Rückgabewert

Erfolg: Der neu zugewiesene Speicherblock.
Fehler: 0.

Bemerkungen

Wenn $pMemory 0 ist, weist die Funktion einen neuen Speicherblock auf die gleiche Weise wie die Funktion _WinAPI_CoTaskMemAlloc() zu.
Wenn $pMemory ungleich 0 ist, sollte es ein Zeiger sein, der von einem vorherigen Aufruf der Funktion _WinAPI_CoTaskMemAlloc() zurückgegeben wird.

Der Inhalt des Blocks bleibt bis zur kürzeren der beiden Größen unverändert, obwohl der neue Block an einem anderen Ort stehen kann.
Wenn $pMemory nicht 0 ist und $iSize 0 ist, wird der Speicher, auf den $pMemory zeigt, freigegeben.
