_WinAPI_CreateGUID

Beschreibung anzeigen in

Erstellt eine globale eindeutige Kennung (globally unique identifier GUID).

#include <WinAPICom.au3>
_WinAPI_CreateGUID ( )

Rückgabewert

Erfolg: Die Stringrepräsentation einer GUID.
Fehler: Leerer String und setzt das @error Flag auf ungleich 0, das @extended Flag kann den HRESULT-Fehlercode enthalten.

Siehe auch

Suche nach CoCreateGuid in der MSDN Bibliothek.

Beispiel

#include <WinAPICom.au3>

For $i = 1 To 3
    ConsoleWrite(_WinAPI_CreateGUID() & @CRLF)
