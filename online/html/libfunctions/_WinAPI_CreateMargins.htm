_WinAPI_CreateMargins

Beschreibung anzeigen in

Erzeugt eine $tagMARGINS Struktur mit der angegebenen linken, rechten, oberen und unteren Grenze.

#include <WinAPIMisc.au3>
_WinAPI_CreateMargins ( $iLeftWidth, $iRightWidth, $iTopHeight, $iBottomHeight )

Parameter

$iLeftWidth Die Breite der linken Grenze welches seine Größe behält.
$iRightWidth Die Breite der rechten Grenze welche seine Größe behält.
$iTopHeight Die Höhe der oberen Grenze welches seine Größe behält.
$iBottomHeight Die Höhe der unteren Grenze welches seine Größe behält.

Rückgabewert

Gibt eine $tagMARGINS Struktur zurück welches die angegebenen Grenzen beinhaltet.

Verwandte Funktionen

$tagMARGINS
