_WinAPI_ShowLastError

Beschreibung anzeigen in

Zeigt den letzten Fehlercode und die letzte Meldung

#include <WinAPIError.au3>
_WinAPI_ShowLastError ( [$sText = '' [, $bAbort = False [, $iLanguage = 0]]] )

Parameter

$sText [optional] Der Text des Benutzers, der mit der Nachricht angezeigt werden soll.
$bAbort [optional] Gibt an, ob das Skript nach der Anzeige einer Fehlermeldung beendet werden soll, gültige Werte sind:
True - Beendet das Skript nach der Anzeige einer Meldung, wenn es einen Fehler anzeigt.
False - mmer normal zurückkehren (Standard).
$iLanguage [optional] Der Sprachidentifikator für die Nachricht.

Rückgabewert

Keine.

Bemerkungen

@error und @extended bleiben bei der Rückkehr erhalten.
