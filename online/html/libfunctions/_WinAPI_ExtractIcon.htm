_WinAPI_ExtractIcon

Beschreibung anzeigen in

Extrahiert ein Symbol aus der angegebenen ausführbaren Datei, DLL oder Symboldatei

#include <WinAPIIcons.au3>
_WinAPI_ExtractIcon ( $sIcon, $iIndex [, $bSmall = False] )

Parameter

$sIcon Der Name einer ausführbaren Datei, DLL oder Symboldatei, aus der die Symbole extrahiert werden sollen.
$iIndex Der 0-basierte Index des zu extrahierenden Symbols.
Wenn dieser Wert eine negative Zahl ist, extrahiert die Funktion das Symbol, dessen Ressourcenkennung gleich dem absoluten Wert von $iIndex ist.
$bSmall [optional] Gibt an, ob ein kleines Symbol extrahiert werden soll, gültige Werte sind:
    True - Extrahiert ein kleines Symbol.
    False - Extrahiert ein großes Symbol (Standard).

Rückgabewert

Erfolg: das Handle zum extrahierten Symbol.
Fehler: 0.

Bemerkungen

Wenn das Icon nicht mehr benötigt wird, sollte man _WinAPI_DestroyIcon() aufrufen um die Ressource im Speicher wieder freizugeben.

Verwandte Funktionen

_WinAPI_DestroyIcon
