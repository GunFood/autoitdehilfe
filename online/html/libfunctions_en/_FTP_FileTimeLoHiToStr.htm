_FTP_FileTimeLoHiToStr

Show description in


Converts filetime Loword and Hiword to a string

#include <FTPEx.au3>
_FTP_FileTimeLoHiToStr ( $iLoDWORD, $iHiDWORD [, $iFmt = 0] )

Parameters

$iLoDWORD FileTime Low
$iHiDWORD File Time Hi
$iFmt [optional]
    0 returns mm/dd/yyyy hh:mm:ss (Default)
    1 returns yyyy/mm/dd hh:mm:ss

Return Value

Success: a datetime according to $iFmt.
Failure: an empty string.
