_WinAPI_ShellCreateDirectory

Erzeugt einen neuen Dateisystemordner

#include <WinAPIShellEx.au3>
_WinAPI_ShellCreateDirectory ( $sFilePath [, $hParent = 0 [, $tSecurity = 0]] )

Parameter

$sFilePath Der vollständig qualifizierte Pfad des Verzeichnisses.
$hParent [optional] Ein Handle zu einem übergeordneten Fenster.
$tSecurity [optional] $tagSECURITY_ATTRIBUTES-Struktur mit dem Sicherheitsattribut des Verzeichnisses. Ist dieser Parameter gleich 0 (Standard), werden keine Sicherheitsattribute gesetzt.

Rückgabewert

Erfolg: 1.
Fehler: 0 und setzt das @error-Flag auf ungleich Null, kann @extended-Flag den Systemfehlercode enthalten.

Bemerkungen

Wenn einer oder mehrere der Zwischenordner nicht vorhanden sind, werden sie ebenfalls erstellt.
Außerdem prüft diese Funktion, ob die Dateien im Verzeichnis sichtbar sind, wenn der Pfad ein Netzwerkpfad ist, um mit Netzwerktreibern umzugehen, die eine begrenzte, aber unbekannte maximale Pfadlänge haben können.
