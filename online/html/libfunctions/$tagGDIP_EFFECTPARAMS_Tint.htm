Parameter Struktur für einen Farbton Effekt

Global Const $tagGDIP_EFFECTPARAMS_Tint = "int Hue; int Amount"

Parameter

Hue Integer im Bereich von -180 bis 180, welche den Farbton festlegt, welcher verstärkt oder abgeschwächt werden soll.
Amount Integer im Bereich von -100 bis 100, welcher festlegt, wie stark der Farbton (gegeben vom Farbtons-Parameter) verstärkt oder abgeschwächt werden soll.
