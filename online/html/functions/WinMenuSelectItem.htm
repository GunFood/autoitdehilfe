WinMenuSelectItem



Beschreibung anzeigen in

Aufruf eines Menüpunktes aus einem Fenster.

WinMenuSelectItem ( "title", "text", "item" [, "item" [, "item" [, "item" [, "item" [, "item" [, "item"]]]]]] )

Parameter

title Der Titel / hWnd / Klasse des Fensters, um das Item zu wählen. Siehe Erweiterte Fenstertiteldefinition.
text Der Text des Fensters, um das Item zu wählen. Siehe Erweiterte Fenstertiteldefinition.
item Der Text des Menüpunktes.
item [optional] Text eines Untermenüpunktes.
item [optional] Text eines Untermenüpunktes.
item [optional] Text eines Untermenüpunktes.
item [optional] Text eines Untermenüpunktes.
item [optional] Text eines Untermenüpunktes.
item [optional] Text eines Untermenüpunktes.

Rückgabewert

Erfolg: 1
Fehler: 0 wenn das Menü nicht gefunden werden konnte

Bemerkungen

Unterstrichene Menüpunkte enthalten ein vorangestelltes "&" Zeichen zum Initialisieren der Unterstreichung. Der Menüpunkt Datei z.B. erfordert den Text "&Datei". Extras erfordert "E&xtras". Auf bis zu 6 verschachtelte Menüpunkte kann zugegriffen werden. Es kann mit inaktiven, minimierten und/oder auch versteckten Fenstern gearbeitet werden.

WinMenuSelectItem() arbeitet nur mit Standard-Menüs. Unglücklicherweise sind viele Menüs heute speziell geschrieben oder Symbolleisten, die nur so tun also ob sie Menüs wären. Dies gilt z.B. für die meisten Programme von Microsoft.

Verwandte Funktionen

ControlCommand, Send

Beispiel

; Dies wählt Datei und dann Seite einrichten im Editor aus

WinActivate("[CLASS:Notepad]", "")
WinMenuSelectItem("[CLASS:Notepad]", "", "&Datei", "Seite ein&richten...")
