Unregisters the client's window

#include <WinAPIShellEx.au3>
_WinAPI_ShellChangeNotifyDeregister ( $iID )

Parameter

$iID The value that specifies the registration ID returned by _WinAPI_ShellChangeNotifyRegister().

Rückgabewert

Success: True.
Failure: False.
