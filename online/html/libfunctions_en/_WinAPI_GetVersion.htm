Function Reference

_WinAPI_GetVersion

Show description in

Retrieves version of the current operating system

#include <WinAPISys.au3>
_WinAPI_GetVersion ( )

Return Value

Returns The string containing the current OS version.

    "6.3" - Windows 8.1
    "6.2" - Windows 8 / Windows Server 2012
    "6.1" - Windows 7 / Windows Server 2008 R2
    "6.0" - Windows Vista / Windows Server 2008
    "5.2" - Windows XP 64-bit Edition / Windows Server 2003 / Windows Server 2003 R2
    "5.1" - Windows XP

Example

#include <WinAPISys.au3>

ConsoleWrite('Windows version: ' & _WinAPI_GetVersion() & @CRLF)
