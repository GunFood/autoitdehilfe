_DebugBugReportEnv



Show description in

Outputs a string containing information for Bug report submission

#include <Debug.au3>
_DebugBugReportEnv ( )

Return Value

Returns a string containing all information needed to submit.

Remarks

Must be used to submit a Bug Report.

Example

#include <Debug.au3>
#include <MsgBoxConstants.au3>

MsgBox($MB_SYSTEMMODAL, "Bug Report Environment", _DebugBugReportEnv())
