_GDIPlus_GraphicsGetDC



Beschreibung anzeigen in

Erhält ein Handle zu dem Gerätekontext des Grafik-Objekts

#include <GDIPlus.au3>
_GDIPlus_GraphicsGetDC ( $hGraphics )

Parameter

$hGraphics Handle zu einem Grafik-Objekt

Rückgabewert

Erfolg: Gerätekontext des Grafik-Objekts
Fehler: 0 und setzt das @error Flag auf ungleich null, das @extended Flag kann den GPSTATUS-Fehlercode ($GPID_ERR* siehe GDIPlusConstants.au3) enthalten.

Bemerkungen

Jedem Aufruf von _GDIPlus_GraphicsGetDC sollte ein Aufruf von _GDIPlus_GraphicsReleaseDC folgen. Es sollte keine andere Methode des Grafik-Objekts zwischen den Aufrufen stattfinden.

Verwandte Funktionen

_GDIPlus_GraphicsReleaseDC
