_GUICtrlMenu_LoadMenu

Beschreibung anzeigen in

Lädt die angegebene Menüressource von der ausführbaren Datei, welche mit der Instanz einer Anwendung verknüpft ist

#include <GuiMenu.au3>
_GUICtrlMenu_LoadMenu ( $hInst, $sMenuName )

Parameter

$hInst Handle zu dem Modul welches die zu ladende Menüressource enthält
$sMenuName String, welcher den Namen der Menüressource enthält.
Alternativ kann dieser Parameter aus Ressourcen-IDs im niederwertigen Wort und 0 im höherwertigen Wort bestehen.

Rückgabewert

Erfolg: das Handle zu der Menüressource
Fehler: 0
