_WinAPI_DeleteObjectID



Beschreibung anzeigen in

Entfernt den Objektidentifizierer von einer angegebenen Datei oder Verzeichnis.

#include <WinAPIFiles.au3>
_WinAPI_DeleteObjectID ( $sFilePath )

Parameter

$sFilePath Der Pfad zu einer Datei oder Verzeichnis von dem Objektidentifizierer welches gelöscht werden soll.

Rückgabewert

Erfolg: 1.
Fehler: 0 und setzt das @error Flag auf ungleich null.

Bemerkungen

Der aufrufende Prozess muss das $SE_BACKUP_NAME Privileg haben, ansonsten schlägt die Funktion fehl und _WinAPI_GetLastError() gibt ERROR_ACCESS_DENIED (5) zurück.
