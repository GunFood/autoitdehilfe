_WinNet_GetConnection

Beschreibung anzeigen in

Ermittelt den Namen einer Netzwerkressource, mit der ein lokales Gerät verbunden ist

#include <WinNet.au3>
_WinNet_GetConnection ( $sLocalName )

Parameter

$sLocalName Der Name des lokalen Gerätes für das die Netzwerkressource ermittelt werden soll

Rückgabewert

Erfolg: Der Name der Netzwerkressource, mit der das lokale Gerät verbunden ist.
Fehler: Setzt das @error Flag auf ungleich 0
