_WinAPI_StringLenA



Beschreibung anzeigen in

Berechnet die Länge eines ANSI-Strings

#include <WinAPIMisc.au3>
_WinAPI_StringLenA ( Const ByRef $tString )

Parameter

$tString Stringstruktur zum berechnen

Rückgabewert

Erfolg: Stringlänge in Zeichen
Fehler: 0

Bemerkungen

Der zurückgegebene String enthält keinen abschliessenden Null-Char.

Verwandte Funktionen

_WinAPI_StringLenW

Siehe auch

Suche nach lstrlenA in der MSDN Bibliothek.

Beispiel

#include <MsgBoxConstants.au3>
#include <WinAPIMisc.au3>

; Erstellt den String-Buffer mit der Größe 64 als char-Struct
Local $tStringBuffer = DllStructCreate("char Data[64]")
; Den Buffer mit Daten füllen
DllStructSetData($tStringBuffer, "Data", "AutoIt")

MsgBox($MB_SYSTEMMODAL, "_WinAPI_StringLenA", "Die Länge des String-Buffers beträgt " & _WinAPI_StringLenA($tStringBuffer) & " Zeichen.")
