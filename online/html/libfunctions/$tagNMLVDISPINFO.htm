$tagNMLVDISPINFO



Beschreibung anzeigen in

Enthält Informationen über eine $LVN_GETDISPINFO oder $LVN_SETDISPINFO Benachrichtigung

Global Const $tagNMLVDISPINFO = $tagNMHDR & ";int Mask;int Item;int SubItem;int State;int StateMask;" & _
"ptr Text;int TextMax;int Image;int Param;int Indent;int GroupID;int Columns;ptr pColumns"

Parameter

$tagNMHDR Enthält Informationen über den Benachrichtigungscode
Mask Satz von Flags, die festlegen, welche Mitglieder dieser Struktur zu setzende Daten enthalten oder welche Mitglieder angefordert werden. Dieses Mitglied kann einen oder mehrere der folgenden Flags gesetzt haben:
    $LVIF_COLUMNS - Das Columns-Mitglied ist gültig
    $LVIF_DI_SETITEM - Das Betriebssystem soll die angeforderten Elementinformationen speichern
    $LVIF_GROUPID - Das GroupID-Mitglied ist gültig
    $LVIF_IMAGE - Das Image-Mitglied ist gültig
    $LVIF_INDENT - Das Indent-Mitglied ist gültig
    $LVIF_NORECOMPUTE - Das Control wird keine LVN_GETDISPINFO-Nachricht erzeugen, um Textinformationen zu ermitteln
    $LVIF_PARAM - Das Param-Mitglied ist gültig
    $LVIF_STATE - Das State-Mitglied ist gültig
    $LVIF_TEXT - Das Text-Mitglied ist gültig
Item Der 0-basierte Index des Controls, auf das diese Struktur verweist
SubItem Der Eins-basierte Index eines Subelements, auf das diese Struktur verweist
State Weist auf den Status des Elements, das Statusbild und das Overlaybild hin
StateMask Wert, der festlegt, welche Bits des State-Mitglieds ermittelt oder verändert werden sollen.
Text Zeiger auf einen String, die den Text des Elements enthält
TextMax Anzahl der Bytes in dem Puffer, auf den Text verweist, inklusive Nullzeichen
Image Index des Element-Icons in der Imagelist des Controls
Param Wert, spezifisch zu dem Element.
Indent Anzahl der Bilderbreiten, um die das Element eingerückt wird
GroupID Identifizierer der Gruppe, zu der das Element gehört
Columns Anzahl der Datenspalten (Subelemente), die für dieses Element dargestellt werden sollen
pColumns Zeiger auf ein Array von Spalten-Indizes
