HttpSetUserAgent

Beschreibung anzeigen in


Setzt den Http User-Agent String, der bei allen Internetanfragen mitgesendet wird.

HttpSetUserAgent ( "user agent" )

Parameter

user agent Der String, welcher als User Agent gesetzt wird. Übergabe eines leeren Strings wird den User Agent zurück auf Standard (="AutoIt") stellen.

Rückgabewert

Der vorherige User Agent.

Verwandte Funktionen

InetGet, InetGetSize, InetRead

Beispiel

HttpSetUserAgent("MyUserAgent")
