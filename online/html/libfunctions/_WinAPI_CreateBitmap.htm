_WinAPI_CreateBitmap

Beschreibung anzeigen in

Erstellt eine Bitmap mit den angegebenen Daten für Breite, Höhe und Farbformat

#include <WinAPIGdi.au3>
_WinAPI_CreateBitmap ( $iWidth, $iHeight [, $iPlanes = 1 [, $iBitsPerPel = 1 [, $pBits = 0]]] )

Parameter

$iWidth Gibt die Breite der Bitmap in Pixel an
$iHeight Gibt die Höhe der Bitmap in Pixel an
$iPlanes [optional] Gibt die Anzahl der Farbebenen an (konstant 1, sofern nicht gerade eine EGA-Karte benutzt wird)
$iBitsPerPel [optional] Gibt die Anzahl der Bits an, um die Farbtiefe zu bestimmen
$pBits [optional] Zeiger auf ein Array mit Farbdaten die benutzt werden sollen, um die Farben in einem Pixel-Rechteck zu setzen.
Jede aus dem Rechteck eingelesene Zeile muss der Länge von einem oder mehreren Wörtern (WORD = 2 bytes) entsprechen (andernfalls sind die Zeilen entsprechend mit Nullen zu füllen).
Wird ein NULL-Zeiger als Parameter gesetzt, ist der Inhalt der Bitmap undefiniert.

Rückgabewert

Erfolg: Das Handle zu dem Bitmap
Fehler: 0
