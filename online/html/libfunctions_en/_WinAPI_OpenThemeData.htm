Function Reference

_WinAPI_OpenThemeData


Opens the theme data for a window and its associated class

#include <WinAPITheme.au3>
_WinAPI_OpenThemeData ( $hWnd, $sClass )

Parameters

$hWnd Handle of the window for which theme data is required.
$sClass The string that contains a semicolon-separated list of classes.
This parameter may contain a list, not just a single name,
to provide the class an opportunity to get the best match between the class and the current visual style.

Return Value

Success: The handle to the theme.
Failure: 0.
