Funktionreferenz

_GUICtrlToolbar_SetParent



Beschreibung anzeigen in

Legt das Fenster fest, an welches das Control Nachrichten sendet

#include <GuiToolBar.au3>
_GUICtrlToolbar_SetParent ( $hWnd, $hParent )

Parameter

$hWnd Handle des Controls
$hParent Handle zum Fenster um Nachrichten zu empfangen

Rückgabewert

Erfolg: das Handle zum vorherigen Nachrichtenfenster
Fehler: 0

Bemerkungen

Die $TB_SETPARENT Nachricht ändert nicht das Hauptfenster welches festgelegt wurde als das Control erstellt wurde.
Durch den Aufruf von _WinAPI_GetParent() gibt das Control das aktuelle Hauptfenster und nicht das in $TB_SETPARENT festgelegte Fenster zurück. Um das Hauptfenster des Controls zu ändern ist _WinAPI_SetParent() aufzurufen.
