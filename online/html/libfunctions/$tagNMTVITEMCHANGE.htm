$tagNMTVITEMCHANGE

Beschreibung anzeigen in


Beinhaltet Information einer TreeView item (Eintrags) Änderung, gesendet von $TVN_ITEMCHANGED und $TVN_ITEMCHANGING notifications (Benachrichtigungen)

Global Const $tagNMTVITEMCHANGE = $tagNMHDR & ";uint Changed;handle hItem;uint StateNew;uint StateOld;lparam lParam"

Parameter

$tagNMHDR Beinhaltet Informationen über die Benachrichtigungs-Nachricht
Changed Legt das Attribut fest. Changed muss immer den Folgenden Wert haben:
    TVIF_STATE - Die Änderung ist das Status-Attribut
hItem Handle zu dem geänderten TreeView Item.
StateNew Flag welche den neuen Item Status festlegt.
StateOld Flag das den vorherigen Itemstatus festlegt.
lParam Vorbehalte für Programmspezifische Daten. Zum Beispiel, ein Wert der mit dem Item zusammenhängt.

Bemerkungen

Diese Benachrichtigung (notification) ist nur ab Windows Vista vorhanden.

Verwandte Funktionen

$tagNMHDR
