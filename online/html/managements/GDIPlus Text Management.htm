Text Benutzerdefinierte Funktionsreferenz

Es folgt eine Liste der benutzerdefinierten Funktionen die in AutoIt verfügbar sind. Ein Klick auf einen Funktionsnamen liefert eine detaillierte Beschreibung.

Wenn diese Funktion verwendet wird, so muss #include <GDIPlus.au3> hinzugefügt werden.

 

Benutzerdefinierte Funktion Beschreibung
_GDIPlus_GraphicsDrawString Zeichnet einen String
_GDIPlus_GraphicsDrawStringEx Zeichnet einen String
_GDIPlus_GraphicsMeasureCharacterRanges Gets a set of region objects each of which bounds a range of character positions within a string
_GDIPlus_GraphicsMeasureString Misst die Größe eines Strings
