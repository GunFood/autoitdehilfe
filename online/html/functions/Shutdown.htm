Shutdown

Beschreibung anzeigen in

Fährt das System herunter.

Shutdown ( code )

Parameter

code Eine Kombination von Herunterfahr-Codes. Siehe Anmerkungen.

Rückgabewert

Erfolg: 1.
Fehler: 0 und @error wird auf Windows API GetLastError() gesetzt.

Bemerkungen

Der Herunterfahr-Code ist eine Kombination der folgenden Werte:
    $SD_LOGOFF (0) = Abmelden / Logoff
    $SD_SHUTDOWN (1) = Herunterfahren
    $SD_REBOOT (2) = Neu Starten / Reboot
    $SD_FORCE (4) = Herunterfahren erzwingen / Force
    $SD_POWERDOWN (8) = Strom abschalten / Power down
    $SD_FORCEHUNG (16) = Erzwingen, falls das System hängt
    $SD_STANDBY (32) = Bereitschaftsmodus / Standby
    $SD_HIBERNATE (64) = Ruhezustand / Hibernate

Die Konstanten sind in AutoItConstants.au3 definiert.

Die benötigten Wert sind mit BitOR() "ODER" zu verknüpfen. Zum Herunterfahren mit Stromabschaltung wäre der Wert z.B. BitOR($SD_SHUTDOWN, $SD_POWERDOWN).

Standby oder Hibernate werden ignoriert, wenn andere Werte gesetzt werden.

Verwandte Funktionen

ProcessClose

Beispiel

 ; Einen Neustart erzwingen

 Shutdown(6)
