_WinAPI_AttachThreadInput



Beschreibung anzeigen in

Verknüpft oder trennt den Eingabe-Prozessmechanismus von einem Thread mit dem eines anderen Threads

#include <WinAPIProc.au3>
_WinAPI_AttachThreadInput ( $iAttach, $iAttachTo, $bAttach )

Parameter

$iAttach Identifizierer des Threads, der mit einem anderen verknüpft werden soll
$iAttachTo Identifizierer des Threads, mit dem verknüpft werden soll
$bAttach Verknüpfungsmodi:
    True - Die Threads sind verknüpft
    False - Die Threads sind getrennt

Rückgabewert

Erfolg: True
Fehler: False
