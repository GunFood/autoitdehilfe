_WinAPI_GetDesktopWindow

Beschreibung anzeigen in

Ermittelt das Handle des Windows Desktop Fensters

#include <WinAPISysWin.au3>
_WinAPI_GetDesktopWindow ( )

Rückgabewert

Gibt das Handle zum Desktop Fenster zurück

Siehe auch

Suche nach GetDesktopWindow in der MSDN Bibliothek.

Beispiel

#include <MsgBoxConstants.au3>
#include <WinAPISysWin.au3>

Example()

Func Example()
    Local $hWin = _WinAPI_GetDesktopWindow()
    MsgBox($MB_SYSTEMMODAL, "", WinGetTitle($hWin))
    MsgBox($MB_SYSTEMMODAL, "", $hWin)
EndFunc   ;==>Example
