_Net_Share_ShareEnum



Beschreibung anzeigen in

Gibt Informationen über alle freigegebenen Ressourcen auf einem Server zurück

#include <NetShare.au3>
_Net_Share_ShareEnum ( [$sServer = ""] )

Parameter

$sServer [optional] String, der den DNS oder NetBIOS-Namen des Remoteservers angibt, auf dem die Funktion ausgeführt werden soll.
Falls dieser Parameter weggelassen wird, wird der lokale Computer verwendet.

Rückgabewert

Erfolg: Ein Array mit dem folgenden Format:
    [0][0] - Anzahl der Einträge im Array
    [1][0] - Name der Ressource
    [1][1] - Typ der Ressource. Kann eine Kombination der folgenden sein:
  $STYPE_DISKTREE - Festplatte
  $STYPE_PRINTQ - Druckerwarteschlange
  $STYPE_DEVICE - Kommunikationsgerät
  $STYPE_IPC - IPC
  $STYPE_SPECIAL - spezielle Freigabe für IPC$ oder ferngesteuerte Administration des Servers
  $STYPE_TEMPORARY - temporäre Freigabe
    [1][2] - Enthält einen optionalen Kommentar über die Ressource
    [1][3] - Gibt die Berechtigungen der freigegebenen Ressource an:
   1 - Berechtigung zum Lesen und Ausführen einer Ressource.
   2 - Berechtigung zum Schreiben einer Ressource.
   4 - Berechtigung zum Erstellen einer Ressource.
   8 - Berechtigung zum Ausführen.
  16 - Berechtigung zum Löschen.
  32 - Berechtigung zum Ändern der Eigenschaften der Ressource.
  64 - Ändern der Berechtigungen der Ressource.
    [1][4] - maximale Anzahl gleichzeitiger Verbindungen zur Ressource
    [1][5] - momentane Anzahl der Verbindungen zur Ressource
    [1][6] - lokaler Pfad zur freigegebenen Ressource
    [1][7] - Passwort der Freigabe
Fehler: Setzt das @error Flag auf ungleich null.

Bemerkungen

Die Mitgliedschaft in einer der folgenden Gruppen ist notwendig, um die Funktion ausführen zu können:
Administratoren, Hauptbenutzer, Druck-Operatoren oder Server-Operatoren.

Verwandte Funktionen

_Net_Share_ConnectionEnum, _Net_Share_FileEnum, _Net_Share_SessionEnum

Beispiel

#include <GUIConstantsEx.au3>
#include <MsgBoxConstants.au3>
#include <NetShare.au3>
#include <WindowsConstants.au3>

#RequireAdmin ; wird für _Net_Share_ShareAdd() benötigt

Global $g_idMemo

Example()

Func Example()
    Local $iI, $aInfo
    Local Const $sShareName = "AutoIt Share"

    ; Erstellt eine GUI
    GUICreate("NetShare", 400, 300)

    ; Erstellt ein Memo Control
    $g_idMemo = GUICtrlCreateEdit("", 2, 2, 396, 296, $WS_VSCROLL)
    GUICtrlSetFont($g_idMemo, 9, 400, 0, "Courier New")
    GUISetState(@SW_SHOW)

    ; Prüfen, ob die Freigabe existiert
    If _Net_Share_ShareCheck(@ComputerName, $sShareName) = -1 Then
        ; Eine Freigabe auf dem lokalen Computer erstellen
        _Net_Share_ShareAdd(@ComputerName, $sShareName, 0, "C:\", "AutoIt Share Comment")
        If @error Then MsgBox($MB_SYSTEMMODAL, "Information", "Fehler beim Hinzufügen der Freigabe : " & @error)
        MemoWrite("Freigabe hinzugefügt")
    Else
        MemoWrite("Freigabe existiert")
    EndIf

    ; Listet Informationen über alle lokalen Freigaben auf
    $aInfo = _Net_Share_ShareEnum(@ComputerName)
    MemoWrite("Einträge lesen ............: " & $aInfo[0][0])
    For $iI = 1 To $aInfo[0][0]
        MemoWrite("Freigabename .............: " & $aInfo[$iI][0])
        MemoWrite("Freigabetyp...............: " & _Net_Share_ResourceStr($aInfo[$iI][1]))
        MemoWrite("Kommentar ................: " & $aInfo[$iI][2])
        MemoWrite("Benutzerrechte ...........: " & _Net_Share_PermStr($aInfo[$iI][3]))
        MemoWrite("Maximale Verbindungen ....: " & $aInfo[$iI][4])
        MemoWrite("Aktuelle Verbindungen ....: " & $aInfo[$iI][5])
        MemoWrite("Lokaler Pfad .............: " & $aInfo[$iI][6])
        MemoWrite("Passwort .................: " & $aInfo[$iI][7])
        MemoWrite()
    Next

    ; Freigabe entfernen
    _Net_Share_ShareDel(@ComputerName, $sShareName)
    If @error Then MsgBox($MB_SYSTEMMODAL, "Information", "Fehler beim Entfernen der Freigabe : " & @error)
    MemoWrite("Freigabe entfernt")

    ; Die Schleife wiederholt sich, bis der Benutzer die Beenden-Aktion der GUI auslöst.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example

; Schreibt eine Nachricht in das Memo
Func MemoWrite($sMessage = "")
    GUICtrlSetData($g_idMemo, $sMessage & @CRLF, 1)
EndFunc   ;==>MemoWrite
