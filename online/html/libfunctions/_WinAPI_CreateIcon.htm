_WinAPI_CreateIcon



Beschreibung anzeigen in

Erzeugt ein Icon mit der angegebenen Größe, Farbe und Bitmuster.

#include <WinAPIIcons.au3>
_WinAPI_CreateIcon ( $hInstance, $iWidth, $iHeight, $iPlanes, $iBitsPixel, $pANDBits, $pXORBits )

Parameter

$hInstance Das Handle zu der Instanz welches das Modul erzeugt.
$iWidth Die Breite des Icons in Pixeln.
$iHeight Die Höhe des Icons in Pixeln.
$iPlanes Die Anzahl der Flächen in der XOR-Bitmaske des Icons.
$iBitsPixel Die Anzahl der Bits pro Pixel in der XOR-Bitmaske des Icons.
$pANDBits Ein Array von Bytes welches die Bitwerte für die AND-Bitmaske des Icons enthält. Die Bitmaske beschreibt eine monochromatische Bitmap.
$pXORBits Ein Array von Bytes welches die Bitwerte für die XOR-Bitmaske des Icons enthält. Die Bitmaske beschreibt eine monochromatische oder geräteunabhängige Farbenbitmap.

Rückgabewert

Erfolg: Das Handle zu dem Icon welches erzeugt wurde.
Fehler: 0. Es ist _WinAPI_GetLastError() aufzurufen um erweiterte Fehlerinformationen zu erhalten.

Bemerkungen

Wenn die Arbeit mit dem Icon fertig ist, sollte man sie mit der _WinAPI_DestroyIcon() Funktion zerstören.

Verwandte Funktionen

_WinAPI_DestroyIcon

; Create AND bitmask structure
