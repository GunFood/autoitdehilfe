Ändert den Text in der Titelleiste des angegebenen Fensters

#include <WinAPISysWin.au3>
_WinAPI_SetWindowText ( $hWnd, $sText )

Parameter

$hWnd Handle zu dem Fenster oder Control dessen Text geändert werden soll
$sText String der den neuen Text des Fensters oder Controls darstellen soll

Rückgabewert

Erfolg: True
Fehler: False. Ruft _WinAPI_GetLastError() auf, um erweiterte Fehlerinformationen zu erhalten.

Bemerkungen

Wenn das Zielfenster vom aktuellen Prozess besessen wird, wird SetWindowText veranlasst eine $WM_SETTEXT Nachricht an das angegebene Fenster oder Control zu senden. Ist das Control eine ListBox, erzeugt mit dem $WS_CAPTION Fensterstil, setzt SetWindowText den Text für das Control, jedoch nicht für die Einträge in der ListBox. Um den Text eines Controls in einem anderen Prozess zu setzen, muss eine $WM_SETTEXT Nachricht mit dem Text direkt an das Fenster gesendet werden,
anstatt mit SetWindowText.
