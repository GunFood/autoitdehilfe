Parameter Struktur für Farbkurven

Global Const $tagGDIP_EFFECTPARAMS_ColorCurve = "int Adjustment; int Channel; int AdjustValue"

Parameter

Adjustment Kurvenanpassungs Konstante, welche die Änderung der Kurve festlegt, die angewendet werden soll ($GDIP_Adjust*).
Channel Die KurvenKanal-Konstante die den Farbkanal festlegt, der von Anpassung beeinflusst werden soll ($GDIP_CurveChannel*).
AdjustValue Integer welche die Intensität der Anpassung festlegt
Der Bereich für akzeptable Werte hängt von der Anpassung ab, die genutzt werden soll (Siehe $GDIP_Adjust* Konstanten).
