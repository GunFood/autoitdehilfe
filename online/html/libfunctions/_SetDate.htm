_SetDate

Beschreibung anzeigen in

Setzt das aktuelle Systemdatum

#include <Date.au3>
_SetDate ( $iDay [, $iMonth = 0 [, $iYear = 0]] )

Parameter

$iDay Tag des Monats. Werte: 1-31
$iMonth [optional] Monat. Werte: 1-12
$iYear [optional] Jahr. Werte: > 0 (Windows könnte dies später einschränken!!)

Rückgabewert

Erfolg: 1
Fehler: 0 und setzt das @error Flag auf ungleich 0
@Error: 1 - Fehler
@extended: _WinAPI_GetLastError()
Fehlercodes: MSDN: ms681381.aspx

Bemerkungen

Falls die optionalen Parameter ($iMonth, $iYear) nicht definiert werden, wird die Funktion die aktuellen Werte nicht ändern!

Verwandte Funktionen

_SetTime

Siehe auch

Suche nach GetLastError in der MSDN Bibliothek.

Beispiel

#include <Date.au3>

_SetDate(20) ; Setzt das Datum auf den 20. Tag des aktuellen Monates des aktuellen Jahres (20-xx-xxxx)
_SetDate(20, 10) ; Setzt das Datum auf den 20. Tag des 10. Monats des aktuellen Jahres (20-10-xxxx)
_SetDate(20, 10, 2004) ; Setzt das Datum auf den 20. Oktober 2004
