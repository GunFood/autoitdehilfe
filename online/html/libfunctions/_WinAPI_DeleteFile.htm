_WinAPI_DeleteFile



Beschreibung anzeigen in

Löscht eine bereits existierende Datei.

#include <WinAPIFiles.au3>
_WinAPI_DeleteFile ( $sFilePath )

Parameter

$sFilePath Der Name der Datei welche gelöscht werden soll.

Rückgabewert

Erfolg: True.
Fehler: False. Ruft _WinAPI_GetLastError() auf, um erweiterte Fehlerinformationen zu erhalten.
