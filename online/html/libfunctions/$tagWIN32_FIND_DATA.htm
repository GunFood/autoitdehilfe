$tagWIN32_FIND_DATA

Beschreibung anzeigen in


Enthält die gefundenen Daten wenn eine Datei ermittelt wird

Global Const $tagWIN32_FIND_DATA = "dword dwFileAttributes;dword ftCreationTime[2];dword ftLastAccessTime[2];dword ftLastWriteTime[2];dword nFileSizeHigh;dword nFileSizeLow;dword dwReserved0;dword dwReserved1;wchar cFileName[260];wchar cAlternateFileName[14]"

Parameter

dwFileAttributes Die Dateiattribute einer Datei.
ftCreationTime Eine FILETIME Struktur, die angibt wann eine Datei oder ein Verzeichnis erstellt wurde.
ftLastAccessTime Eine FILETIME Struktur, die angibt wann eine Datei zuletzt gelesen, in ihr geschrieben, oder für ausführbare Dateien, gestartet wurde.
ftLastWriteTime Eine FILETIME Struktur, die angibt wann eine Datei zuletzt gekürzt, in ihr geschrieben, oder überschrieben wurde.
nFileSizeHigh Der höherwertige DWORD-Wert der Dateigröße, in Bytes.
nFileSizeLow Der niederwertige DWORD-Wert der Dateigröße, in Bytes.
dwReserved0 Falls das dwFileAttributes Mitglied das $FILE_ATTRIBUTE_REPARSE_POINT Attribut enthält. Dieses Mitglied gibt das "Reparse Tag".
dwReserved1 Reserviert.
cFileName Der Name der Datei.
cAlternateFileName Ein Alternativname für die Datei, das klassische 8.3 (Dateiname.erw) Dateinamenformat.

Bemerkungen

Unicode Version muss genutzt werden.
