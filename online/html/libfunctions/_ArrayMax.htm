_ArrayMax



Beschreibung anzeigen in

Gibt den höchsten Wert aus einem 1D oder 2D Array zurück

#include <Array.au3>
_ArrayMax ( Const ByRef $aArray [, $iCompNumeric = 0 [, $iStart = -1 [, $iEnd = -1 [, $iSubItem = 0]]]] )

Parameter

$aArray Das zu durchsuchende Array.
$iCompNumeric [optional] Vergleichsmethode:
    0 = Alphanumerischer Vergleich
    1 = Numerischer Vergleich
$iStart [optional] Start Array-Index; gewöhnlich auf 0 oder 1 gesetzt (Standard = 0)
$iEnd [optional] Stopp Array-Index (Standard = 0)
$iSubItem [optional] zu durchsuchende Spalte des Arrays

Rückgabewert

Erfolg: der höchste Wert in dem Array
Fehler: Setzt @error auf ungleich 0 (siehe _ArrayMaxIndex() Beschreibung für @error).

Verwandte Funktionen

_ArrayMaxIndex, _ArrayMin, _ArrayMinIndex, _ArrayUnique

Beispiel

#include <Array.au3>
#include <MsgBoxConstants.au3>

Local $aArray = StringSplit("4,2,06,8,12,5", ",")

MsgBox($MB_SYSTEMMODAL, 'Maximaler String Wert', _ArrayMax($aArray, 0, 1))
MsgBox($MB_SYSTEMMODAL, 'Maximaler numerischer Wert', _ArrayMax($aArray, 1, 1))

Local $aArray[4][4]
For $i = 0 To 3
    For $j = 0 To 3
        $aArray[$i][$j] = Random(0, 99, 1)
    Next
Next
_ArrayDisplay($aArray, "2D Array")
MsgBox($MB_SYSTEMMODAL, 'Maximaler numerischer Wert in Spalte 2', _ArrayMax($aArray, 1, 0, 0, 2))
