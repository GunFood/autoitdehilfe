_WinAPI_GetHGlobalFromStream



Beschreibung anzeigen in

Ruft das globale Speicherhandle zu einem Stream ab

#include <WinAPICom.au3>
_WinAPI_GetHGlobalFromStream ( $pStream )

Parameter

$pStream Zeiger auf das Stream-Objekt, das zuvor durch einen Aufruf der Funktion _WinAPI_CreateStreamOnHGlobal() erzeugt wurde.

Rückgabewert

Erfolg: Zeiger auf das aktuelle Speicherhandle, das von dem angegebenen Stream-Objekt verwendet wird.
Fehler: 0 und setzt @error auf ungleich 0. @extended kann den HRESULT Fehlercode enthalten.

Bemerkungen

Diese Funktion kann nur innerhalb desselben Prozesses aufgerufen werden, von dem aus das Byte-Array erstellt wurde.

Verwandte Funktionen

_WinAPI_CreateStreamOnHGlobal
