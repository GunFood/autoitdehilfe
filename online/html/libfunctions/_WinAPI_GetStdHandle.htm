_WinAPI_GetStdHandle

Beschreibung anzeigen in

Ermittelt ein Handle für das Standard Eingabe-, Ausgabe- oder Fehlerausgabegerät

#include <WinAPIHObj.au3>
_WinAPI_GetStdHandle ( $iStdHandle )

Parameter

$iStdHandle Standardgerät für welches ein Handle zurückgegeben werden soll. Dies kann einer der folgenden Werte sein:
0 - Handle zu dem Standard Eingabegerät
1 - Handle zu dem Standard Ausgabegerät
2 - Handle zu dem Standard Fehlerausgabegerät

Rückgabewert

Erfolg: Handle zum festgelegten Gerät
Fehler: (-1). Es ist _WinAPI_GetLastError() aufzurufen um erweiterte Fehlerinformationen zu erhalten.

Bemerkungen

Das Handle hat GENERIC_READ und GENERIC_WRITE Zugriffsrechte, sofern die Anwendung nicht mit SetStdHandle ein Standard Handle mit weniger Zugriffsrechten festgelegt hat.
Falls eine Anwendung keine zugehörigen Standard Handles besitzt, ist der Rückgabewert 0.
