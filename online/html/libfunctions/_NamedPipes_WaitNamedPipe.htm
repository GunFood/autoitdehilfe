_NamedPipes_WaitNamedPipe

Beschreibung anzeigen in


Wartet, bis eine Instanz einer benannten Pipe verfügbar ist

#include <NamedPipes.au3>
_NamedPipes_WaitNamedPipe ( $sPipeName [, $iTimeOut = 0] )

Parameter

$sPipeName Der Name der benannten Pipe. Der String muss den Computername enthalten, auf dem der Serverprozess ausgeführt wird.
Ein Punkt kann für den Servername verwendet werden, wenn die Pipe lokal angelegt wurde.
$iTimeout [optional] Anzahl der Millisekunden, die auf die Verfügbarkeit der benannten Pipe gewartet wird.
Es können auch folgende Werte verwendet werden:
    -1 - Die Funktion kehrt erst zurück, wenn die Pipe verfügbar ist.
    0 - Die Wartezeit entspricht dem vom Serverprozess vorgegebenen Standardwert.

Rückgabewert

Erfolg: True
Fehler: False

Bemerkungen

Wenn keine Instanz der angegebenen benannten Pipe existiert, kehrt die _NamedPipes_WaitNamedPipe()-Funktion sofort zurück.

Verwandte Funktionen

_NamedPipes_CallNamedPipe
