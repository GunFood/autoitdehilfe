$tagNMHDDISPINFO

Beschreibung anzeigen in

Enthält Informationen die bei der Verarbeitung von $HDN_GETDISPINFO Benachrichtigungen gebraucht werden

Global Const $tagNMHDDISPINFO = "hwnd WndFrom;int IDFrom;int Code;int Item;int Mask;ptr Text;int TextMax;int Image;int lParam"

Parameter

WndFrom Ein Fensterhandle zu dem Control welches die Nachricht sendet
IDFrom Ein Identifizierer des Controls welches die Nachricht sendet
Code Ein Benachrichtigungscode
Item 0-basierter Index des Elements in dem Header-Control
Mask Satz von Bit-Flags, die festlegen welche Mitglieder der Struktur von dem Eigner des Controls ausgefüllt werden müssen. Dieser Wert kann eine Kombination der folgenden Werte sein:
    $HDI_TEXT - Das Text Feld muss ausgefüllt werden
    $HDI_IMAGE - Das Image Feld muss ausgefüllt werden
    $HDI_LPARAM - Das lParam Feld muss ausgefüllt werden
    $HDI_DI_SETITEM - Ein Rückgabewert. Weist darauf hin, dass das Control die Elementinformationen speichern soll und nicht wieder danach fragt.
Text Zeiger auf einen null-terminierten String, die den Text enthält der für das Element angezeigt wird
TextMax Größe des Puffers auf den Text verweist
Image 0-basierter Index eines Bildes innerhalb der Imagelist. Das festgelegte Bild wird mit dem Element angezeigt, aber es nimmt nicht den Platz der Bitmap des Elements ein. Wenn Image auf $I_IMAGECALLBACK gesetzt ist, erfragt das Control die Bildinformationen für dieses Element indem es eine $HDN_GETDISPINFO Benachrichtigung sendet.
lParam Anwendungsspezifischer Wert des Elements
