_WinAPI_WriteConsole



Beschreibung anzeigen in

Schreibt einen String in einen Konsolen-Bildschirmpuffer

#include <WinAPIProc.au3>
_WinAPI_WriteConsole ( $hConsole, $sText )

Parameter

$hConsole Handle zu dem Konsolen-Bildschirmpuffer
$sText Text welcher in den Konsolen-Bildschirmpuffer geschrieben werden soll

Rückgabewert

Erfolg: True
Fehler: False. Es ist _WinAPI_GetLastError() aufzurufen um erweiterte Fehlerinformationen zu erhalten.
