$tagGDIPSTARTUPINPUT Struktur

Global Const $tagGDIPSTARTUPINPUT = "int Version;ptr Callback;int NoThread;int NoCodecs"

Parameter

Version Legt die Version der Microsoft Windows GDI+ fest
Callback Zeiger auf eine Callback-Funktion, welche von der GDI+ in Debug Builds, für Aussagen und Warnungen, aufgerufen werden kann.
NoThread Wahrheitswert (BOOL) der bestimmt, ob der GDI+ Hintergrund-Thread unterbunden werden soll.
Wenn dieses Mitglied auf TRUE gesetzt wird, gibt GdiplusStartup einen Zeiger auf eine Hook-Funktion und einen Zeiger auf eine Unhook-Funktion zurück.
Diese Funktionen müssen entsprechend aufgerufen werden um den Hintergrund-Thread zu ersetzen.
Wenn man für den Aufruf der Hook- und Unhook-Funktionen nicht verantwortlich sein möchte, setzt man dieses Mitglied auf FALSE.
NoCodecs Wahrheitswert (BOOL) der bestimmt, ob die GDI+ die externen Image-Codecs unterbinden soll.
GDI+ Version 1.0 unterstützt keine externen Image-Codecs, daher wird dieser Parameter ignoriert.
