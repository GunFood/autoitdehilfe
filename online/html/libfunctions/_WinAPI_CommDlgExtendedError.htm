_WinAPI_CommDlgExtendedError

Beschreibung anzeigen in

Gibt die Fehlermeldung einer Standard-Dialogbox zurück. Dieser String beschreibt den zuletzt aufgetretenen Fehler beim Aufruf der Standard-Dialogbox Funktionen

#include <WinAPIDlg.au3>
_WinAPI_CommDlgExtendedError ( )

Rückgabewert

Erfolg: String mit Fehlermeldung
Fehler: Ein Hexadezimalstring des erweiterten Fehlers

Bemerkungen

Kann allgemeine Fehlermeldungen von jeder der Standard-Dialogbox Funktionen zurückgeben.

Verwandte Funktionen

_WinAPI_GetOpenFileName, _WinAPI_GetSaveFileName
