Chr



Beschreibung anzeigen in

Gibt ein Zeichen entsprechend eines ASCII-Codes wieder.

Chr ( ASCIIcode )

Parameter

ASCIIcode Ein ASCII-Code aus dem Bereich 0-255 (z.B.: 65 gibt den Großbuchstaben "A" wieder).

Rückgabewert

Erfolg: Einen String mit entsprechendem ASCII-Zeichen.
Fehler: Einen leeren String und setzt @error auf 1, wenn der ASCII-Code größer als 255 ist.

Bemerkungen

Siehe die ASCII Character Code Tabelle für eine komplette Liste der verfügbaren Werte.

Chr(48) == "0", Chr(57) == "9", Chr(65) == "A", Chr(90) == "Z", Chr(97) == "a", Chr(122) = "z", etc.

Verwandte Funktionen

Asc, AscW, ChrW, String

Beispiel

#include <MsgBoxConstants.au3>

Local $sText = ""
For $i = 65 To 90
    $sText = $sText & Chr($i) ; Oder $sText &= Chr($i) kann genauso verwendet werden
Next
MsgBox($MB_SYSTEMMODAL, "Alphabet der Großbuchstaben", $sText) ;Zeigt die Zeichen zwischen 65 und 90
