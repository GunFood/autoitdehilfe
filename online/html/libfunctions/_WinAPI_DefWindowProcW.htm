_WinAPI_DefWindowProcW



Beschreibung anzeigen in

Ruft die Standardfensterprozedur auf um eine Standardverarbeitung für jegliche Fensternachrichten bereitzustellen.

#include <WinAPISysWin.au3>
_WinAPI_DefWindowProcW ( $hWnd, $iMsg, $wParam, $lParam )

Parameter

$hWnd Das Handle einer Fensterprozedur welches die Nachricht erhält.
$iMsg Die Nachricht.
$wParam Zusätzliche nachrichtenspezifische Informationen. Der Inhalt diesen Parameters hängt von der Nachricht ab.
$lParam Zusätzliche nachrichtenspezifische Informationen. Der Inhalt diesen Parameters hängt von der Nachricht ab.

Rückgabewert

Gibt das Ergebnis der Nachrichtenverarbeitung zurück welches von der gesendeten Nachricht abhängt.
