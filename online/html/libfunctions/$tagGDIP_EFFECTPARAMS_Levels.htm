Parameter Struktur für Level Effekte

Global Const $tagGDIP_EFFECTPARAMS_Levels = "int Highlight; int Midtone; int Shadow"

Parameter

Highlight Integer im Bereich von 0 bis 100, welche festlegt, welche Pixel aufgehellt werden sollen.
Midtone Integer im Bereich von -100 bis 100, welche festlegt, wie stark ein Bild aufgehellt oder abgedunkelt werden soll.
Shadow Integer im Bereich von 0 bis 100, welche festlegt, welche Pixel abgedunkelt werden sollen.
