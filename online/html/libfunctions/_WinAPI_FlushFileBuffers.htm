_WinAPI_FlushFileBuffers



Beschreibung anzeigen in

Leert den (Speicher) Puffer einer bestimmten Datei und bewirkt, dass alle gepufferten Daten geschrieben werden

#include <WinAPIFiles.au3>
_WinAPI_FlushFileBuffers ( $hFile )

Parameter

$hFile Handle zu einer geöffneten Datei. Das Dateihandle muss $GENERIC_WRITE Zugriffsrechte besitzen.
Falls hFile ein Handle zu einem Kommunikationsgerät ist, leert die Funktion nur den übertragenen Puffer.
Ist hFile ein Handle zu einem Server-End einer Named Pipe kehrt die Funktion nicht zurück bis der Client alle aus der Leitung gepufferten Daten gelesen hat.

Rückgabewert

Erfolg: True
Fehler: False. Ruft _WinAPI_GetLastError() auf, um erweiterte Fehlerinformationen zu erhalten.

Verwandte Funktionen

_WinAPI_CloseHandle, _WinAPI_CreateFile, _WinAPI_GetFileSizeEx, _WinAPI_ReadFile, _WinAPI_SetEndOfFile, _WinAPI_SetFilePointer, _WinAPI_WriteFile
