_EventLog__RegisterSource



Beschreibung anzeigen in

Stellt einen registrierten Handle zum angegebenen Event Log wieder her

#include <EventLog.au3>
_EventLog__RegisterSource ( $sServerName, $sSourceName )

Parameter

$sServerName Der UNC-Name des Servers auf dem der Eventlog geöffnet werden soll. Wenn dieser leer ist, wird die Anweisung auf dem lokalem Computer ausgeführt.
$sSourceName Der Name der Eventquelle deren Handle wieder hergestellt werden soll. Der Sourcename muss ein Unterschlüssel vom Eventlog Registry Eintrag sein.

Rückgabewert

Erfolg: Der Handle des Eventlog
Fehler: 0

Bemerkungen

Wird der Quellname nicht gefunden, verwendet der "logging" Service den Eintrag log; es wird keine neue Quelle angelegt.
Events werden nach Ihrer Quelle bezeichnet, wenn keine Message und Kategoriemessages-Files zur genaueren Beschreibung der Eventquelle vorliegen.
Um den Handle des Eventlog zu beenden muss _EventLog__DeregisterSource() benutzt werden.

Verwandte Funktionen

_EventLog__DeregisterSource
