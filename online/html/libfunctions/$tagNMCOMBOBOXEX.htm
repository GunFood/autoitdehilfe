$tagNMCOMBOBOXEX

Beschreibung anzeigen in


Enthält Informationen bezüglich der ComboBoxEx Elemente für die Nutzung von Benachrichtigungen

Global Const $tagNMCOMBOBOXEX = $tagNMHDR & ";int Mask;int Item;ptr Text;int TextMax;int Image;" & _
"int SelectedImage;int OverlayImage;int Indent;int Param"

Parameter

$tagNMHDR Enthält Informationen über den Benachrichtigungscode
Mask Ein Satz von Bit-Flags, welche die Attribute festlegen. Kann eine Kombination der folgenden Werte sein.
    CBEIF_DI_SETITEM - Dieses Flag wird gesetzt, wenn CBEN_GETDISPINFO durchgeführt werden soll
    CBEIF_IMAGE - Das Image-Mitglied ist gültig oder muss ausgefüllt sein.
    CBEIF_INDENT - Das Indent-Mitglied ist gültig oder muss ausgefüllt sein.
    CBEIF_LPARAM - Das Param-Mitglied ist gültig oder muss ausgefüllt sein.
    CBEIF_OVERLAY - Das Overlay-Mitglied ist gültig oder muss ausgefüllt sein.
    CBEIF_SELECTEDIMAGE - Das SelectedImage-Mitglied ist gültig oder muss ausgefüllt sein.
    CBEIF_TEXT - Das Text-Mitglied ist gültig oder muss ausgefüllt sein.
Item Der 0-basierte Index des Elements.
Text Ein Zeiger auf einen Zeichen-Puffer, der den Text des Elements enthält oder aufnimmt.
TextMax Die Länge von Text, in TCHARs. Wenn Textinformation gesetzt ist, wird dieses Mitglied ignoriert.
Image Der 0-basierte Index eines Bildes innerhalb der Imagelist.
SelectedImage Der 0-basierte Index eines Bildes innerhalb der Imagelist.
OverlayImage Der Eins-basierte Index eines Overlay-Bildes innerhalb der Imagelist.
Indent Die Anzahl der Einrückungen zur Anzeige des Elements. Jede Einrückung entspricht 10 Pixel.
Param Ein Wert spezifisch zu dem Element.
