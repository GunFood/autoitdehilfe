_WinAPI_DefWindowProc



Beschreibung anzeigen in

Ruft die Standard Fenster-Prozedur auf, um eine korrekte Weiterverarbeitung der unbearbeiteten Nachrichten zu gewährleisten

#include <WinAPISysWin.au3>
_WinAPI_DefWindowProc ( $hWnd, $iMsg, $wParam, $lParam )

Parameter

$hWnd Fenster-Handle der Fenster-Prozedur, welche die Nachricht erhalten hat
$iMsg Legt die Nachricht fest
$wParam Gibt zusätzliche Informationen der Nachricht an
$lParam Gibt zusätzliche Informationen der Nachricht an

Rückgabewert

Gibt den Rückgabewert zurück der das Ergebnis der Nachrichtenverarbeitung ist. Dieser hängt von der Nachricht ab.
