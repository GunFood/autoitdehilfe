$tagNMTBHOTITEM

Beschreibung anzeigen in

Enthält Informationen, die in $TBN_HOTITEMCHANGE Benachrichtigungen genutzt werden

Global Const $tagNMTBHOTITEM = $tagNMHDR & ";int idOld;int idNew;dword dwFlags"

Parameter

$tagNMHDR Enthält Informationen über den Benachrichtigungscode
idOld Befehls-Identifizierer des zuvor hervorgehobenen Elements
idNew Befehls-Identifizierer des Elements, welches hervorgehoben werden soll.
dwFlags Flags die darauf hinweisen warum das hervorgehobene Element geändert wurde. Dies kann einer oder mehrere der folgenden Werte sein:
    $HICF_ACCELERATOR - Die Änderung des hervorgehobenen Elements wurde durch einen Abkürzungsbefehl herbeigeführt
    $HICF_ARROWKEYS - Die Änderung des hervorgehobenen Elements wurde durch eine der Pfeiltasten herbeigeführt
    $HICF_DUPACCEL - Modifiziert $HICF_ACCELERATOR. Wenn dieses Flag gesetzt ist, haben mehrere Elemente denselben Abkürzungsbefehl
    $HICF_ENTERING - Modifiziert die anders begründeten Flags. Wenn dieses Flag gesetzt ist, gibt es kein zuvor hervorgehobenes Element und idOld enthält keine gültigen Informationen
    $HICF_LEAVING - Modifiziert die anders begründeten Flags. Wenn dieses Flag gesetzt ist, gibt es kein neu hervorzuhebendes Element und idNew enthält keine gültigen Informationen
    $HICF_LMOUSE - Die Änderung des hervorgehobenen Elements resultiert aus einem Klick der linken Maustaste
    $HICF_MOUSE - Die Änderung des hervorgehobenen Elements resultiert aus einem Mausereignis
    $HICF_OTHER - Die Änderung des hervorgehobenen Elements resultiert aus einem nicht ermittelbaren Ereignis. Dies wird meistens durch eine Änderung des Fokus oder der $TB_SETHOTITEM Nachricht herrühren
    $HICF_RESELECT - Die Änderung des hervorgehobenen Elements resultiert aus der Eingabe eines Abkürzungsbefehls durch den Benutzer für ein Element, dass bereits hervorgehoben ist
    $HICF_TOGGLEDROPDOWN - Version 5.80. Bewirkt, dass der Button zwischen dem Status hin-/herschaltet

Bemerkungen

Benötigt Abgleich zu x64.

Verwandte Funktionen

$tagNMHDR
