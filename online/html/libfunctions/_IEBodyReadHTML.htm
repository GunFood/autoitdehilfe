_IEBodyReadHTML



Beschreibung anzeigen in

Gibt den HTML-Code innerhalb des <body>-Tags des Dokumentes zurück

#include <IE.au3>
_IEBodyReadHTML ( ByRef $oObject )

Parameter

$oObject Objektvariable einer InternetExplorer.Application, Window- oder Frame-Objekt

Rückgabewert

Erfolg: HTML-Code innerhalb des <body>-Tags des Dokumentes.
Fehler: Setzt das @error Flag auf ungleich null
@error: 3 ($_IEStatus_InvalidDataType) - Ungültiger Datentyp
@Extended: Nummer des ungültigen Parameters.

Bemerkungen

Diese Funktion verwendet die Eigenschaft .innerHTML des <body>-Tags des Dokumentes.
.innerHTML ist für fast alle DOM-Elemente mittels _IEPropertyGet() auslesbar.

Verwandte Funktionen

_IEBodyReadText, _IEBodyWriteHTML, _IEDocInsertHTML, _IEDocInsertText, _IEDocReadHTML, _IEPropertyGet

Beispiel

#include <IE.au3>

; Öffnet einen Browser mit dem Grundbeispiel, liest den HTML-Code innerhalb des <body>-Tags des Dokumentes.
; Dann wird eine neue HTML-Quelltextzeile dem Original hinzugefügt und zurück in den Browser geschrieben.

Local $oIE = _IE_Example("basic")
Local $sHTML = _IEBodyReadHTML($oIE)
$sHTML = $sHTML & "<p><font color=red size=+5>Big RED text!</font>"
_IEBodyWriteHTML($oIE, $sHTML)
