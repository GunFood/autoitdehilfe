_IE_Example

Beschreibung anzeigen in

Zeigt ein Browserfenster mit vorgefertigten HTML-Seiten an, die in der IE.au3 verwendet werden und die man für eigene Tests nutzen kann

#include <IE.au3>
_IE_Example ( [$sModule = "basic"] )

Parameter

$sModule [optional] Gibt an, welches Modul aufgerufen werden soll
    "basic" = (Standard) einfache HTML-Seite mit Text, Links und Bildern
    "form" = einfache HTML-Seite mit mehreren Forumularen
    "frameset" = einfache HTML-Seite mit Frames
    "iframe" = einfache HTML-Seite mit Iframes
    "table" = einfache HTML-Seite mit Tabellen

Rückgabewert

Erfolg: Gibt eine Objektvariable zurück, die auf ein InternetExplorer.Application-Objekt verweist
Fehler: Setzt das @error Flag auf ungleich 0
@error: 5 ($_IEStatus_InvalidValue) = Ungültiger Wert
@extended: Beinhaltet Nummer des ungültigen Parameters

Bemerkungen

Diese Funktion zeigt ein Browserfenster mit vorgefertigten HTML-Seiten an, die in der IE.au3 verwendet werden und die man als Basis für eigene Tests mit der IE.au3 nutzen kann.

Verwandte Funktionen

_IE_Introduction

Beispiel

#include <IE.au3>

; Erstellt ein Browserfenster und stellt jede der möglichen Beispielseiten dar.
; Die zurückgegebene Objektvariable kann als Objektvariable behandelt werden, die von _IECreate oder _IEAttach zurückgegeben wurde.

Local $oIE_basic = _IE_Example("basic")
Local $oIE_form = _IE_Example("form")
Local $oIE_table = _IE_Example("table")
Local $oIE_frameset = _IE_Example("frameset")
Local $oIE_iframe = _IE_Example("iframe")
