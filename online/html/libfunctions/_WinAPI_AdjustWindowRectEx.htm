_WinAPI_AdjustWindowRectEx



Beschreibung anzeigen in

Berechnet die benötigte Größe des Fensterrechtecks basierend auf der gewünschten Größe des Clientrechtecks

#include <WinAPISysWin.au3>
_WinAPI_AdjustWindowRectEx ( ByRef $tRECT, $iStyle [, $iExStyle = 0 [, $bMenu = False]] )

Parameter

$tRECT Die $tagRECT Struktur welche die Koordinaten der gewünschten Clientrechtecks beinhaltet. Diese Struktur mmuss vor dem Funktionsaufruf erzeugt werden.
$iStyle Der Fensterstil des Fensters wessen benötigte Größe berechnet werden soll. Der $WS_OVERLAPPED-Stil kann nicht angegeben werden.
$iExStyle [optional] Der erweiterte Fensterstil des Fensters wessen benötigte Größe berechnet werden soll.
$bMenu [optional] Gibt an ob das Fenster ein Menu hat, gültige Werte sind:
True - Das Fenster hat ein Menu.
False - Das Fenster hat kein Menu (Standard).

Rückgabewert

Erfolg: True.
Fehler: False. Ruft _WinAPI_GetLastError() auf, um erweiterte Fehlerinformationen zu erhalten.
