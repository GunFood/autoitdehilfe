_MemMoveMemory

Beschreibung anzeigen in

Bewegt Speicher entweder nach vorne oder nach hinten, ausgerichtet oder unausgerichtet

#include <Memory.au3>
_MemMoveMemory ( $pSource, $pDest, $iLength )

Parameter

$pSource Pointer auf die Quelle der Bewegung
$pDest Pointer auf das Ziel der Bewegung
$iLength Legt die Anzahl der zu kopierenden Bytes fest.

Rückgabewert

Keine, das @error Makro könnte gesetzt werden, wenn der interne DllCall() fehlschlägt (siehe AutoIt oder Windows Fehler)
