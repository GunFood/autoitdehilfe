_WinAPI_CreateDIBitmap

Beschreibung anzeigen in

Erzeugt eine kompatible Bitmap (DDB) von einer DIB und setzt, optional, die Bitmapbits.

#include <WinAPIGdi.au3>
_WinAPI_CreateDIBitmap ( $hDC, $tBITMAPINFO, $iUsage [, $pBits = 0] )

Parameter

$hDC Das Handle eines Gerätekontextes.
$tBITMAPINFO Die $tagBITMAPINFO Struktur welche diverse Attribute der DIB angibt, einschließlich der Dimensionen und Farben der Bitmap.
Man sollte beachten, dass ein positiver Wert für die Höhe eine von unten nach oben DIB erzeugt während hingegen ein negativer Wert eine von oben nach unten DIB erzeugt.
$iUsage Der Typ welcher für die Farben verwendet werden soll. Dieser Parameter muss einer der folgenden Werte annehmen:
    $DIB_PAL_COLORS
    $DIB_RGB_COLORS
$pBits [optional] Ein Zeiger welches auf ein Array an Bytes zeigt welches die ursprünglichen Bitmapdaten beinhaltet.

Rückgabewert

Erfolg: Das Handle einer kompatiblen Bitmap welche die selbe Farbtiefe besitzen wird wie ihr angegebener Gerätekontext.
Fehler: 0.

Bemerkungen

Wenn die Arbeit mit der Bitmap abgeschlossen ist, so sollte man sie mit der _WinAPI_DeleteObject() Funktion zerstören.

Verwandte Funktionen

_WinAPI_DeleteObject
