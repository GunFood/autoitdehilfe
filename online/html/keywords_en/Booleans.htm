Keyword Reference

False / True



Show description in

Boolean values for use in logical expressions.

$var = False
$var = True

Remarks

These keywords should not be used in expressions as AutoIt will not detect this 'misuse' and the results will be unpredictable.

Related

IsBool

Example

#include <MsgBoxConstants.au3>

Local $bBoolean = False
If Not ($bBoolean = True) Then MsgBox($MB_SYSTEMMODAL, "Bool comparison", "OK") ; If $bBoolean is NOT equal to True then display the messagebox
