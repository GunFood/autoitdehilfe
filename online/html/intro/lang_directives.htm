Sprachreferenz - Richtlinien

Pre-Prozessor-Richtlinien sind eine Art um ein Skript (oder Compiler) darzustellen, die sich auf eine besondere Weise verhalten. Sie beginnen mit einem Hash (#) und sind am Anfang des Skripts platziert..

 

Es folgt eine Liste der alphabetischen Pre-Prozessor-Richtlinien, die von AutoIt unterstützt werden.

 

Richtlinie Beschreibung
#comments... Gibt an, dass ein ganzer Teil eines Skriptes auskommentiert werden soll.
#include Bindet eine Datei in das aktuelle Skript ein.
#include-once Legt fest, dass die aktuelle Datei nur einmal inkludiert wird.
#NoTrayIcon Bewirkt, dass das AutoIt-Icon nicht im Systeminfo-Bereich der Taskleiste erscheint, wenn das Skript gestartet wird.
#OnAutoItStartRegister Registriert eine Funktion, wenn AutoIt gestartet wird.
#pragma Changes how the script is compiled.
#RequireAdmin Legt fest, dass das aktuelle Skript die vollen Administratorrechte benötigt, um zu laufen.
