Trennt die Serverseite einer benannten Pipe-Instanz vom Client-Prozess

#include <NamedPipes.au3>
_NamedPipes_DisconnectNamedPipe ( $hNamedPipe )

Parameter

$hNamedPipe Handle zur Serverseite der Pipe-Instanz.

Rückgabewert

Erfolg: True
Fehler: False

Bemerkungen

Wenn die Clientseite der Pipe geöffnet ist, wird sie durch _NamedPipes_DisconnectNamedPipe() gewaltsam geschlossen.
Der Client erhält eine Fehlermeldung beim nächsten Zugriffsversuch auf die Pipe.
Der so von der Pipe getrennte Client muss trotzdem _WinAPI_CloseHandle() aufrufen, um sein Ende der Pipe ordnungsgemäß zu schließen.

Verwandte Funktionen

_WinAPI_CloseHandle
