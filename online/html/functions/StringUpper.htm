StringUpper

Beschreibung anzeigen in


Wandelt einen String in Großbuchstaben um.

StringUpper ( "string" )

Parameter

string Der umzuwandelnde String.

Rückgabewert

Gibt den neuen String in Großbuchstaben aus.

Verwandte Funktionen

StringCompare, StringInStr, StringIsAlNum, StringIsAlpha, StringIsASCII, StringIsLower, StringIsSpace, StringIsUpper, StringIsXDigit, StringLeft, StringLen, StringLower, StringMid, StringReplace, StringRight, StringSplit, StringTrimLeft, StringTrimRight

Beispiel

#include <MsgBoxConstants.au3>

Local $sString = StringUpper("Dies ist ein String") ; Wandelt den folgenden String in Großbuchstaben.
MsgBox($MB_SYSTEMMODAL, "", "StringUpper gibt folgendes zurück: " & $sString)
