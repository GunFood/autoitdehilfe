$tagPROCESS_INFORMATION

Beschreibung anzeigen in

Enthält Informationen über einen kürzlich erstellten Prozess und dessen primären Thread

Global Const $tagPROCESS_INFORMATION = "hwnd hProcess;hwnd hThread;int ProcessID;int ThreadID"

Parameter

hProcess Ein Handle zu dem kürzlich erstellten Prozess
hThread Ein Handle zu dem primären Thread des kürzlich erstellen Prozesses
ProcessID Ein Wert der verwendet werden kann um einen Prozess zu identifizieren
ThreadID Ein Wert der verwendet werden kann um einen Thread zu identifizieren
