_FTP_FileTimeLoHiToStr

Beschreibung anzeigen in

Konvertiert Lo-und HiDWORD einer FileTime-Struktur zu einem String

#include <FTPEx.au3>
_FTP_FileTimeLoHiToStr ( $iLoDWORD, $iHiDWORD [, $iFmt = 0] )

Parameter

$iLoDWORD FileTime Low-Wert
$iHiDWORD File Time Hi-Wert
$iFmt [optional]
    0 Gibt das Format mm/tt/jjjj hh:mm:ss zurück (Standard)
    1 gibt jjjj/mm/tt hh:mm:ss zurück (z.B. für Date.au3)

Rückgabewert

Erfolg: ein Datum und eine Zeit wie in $iFmt angegeben
Fehler: ein leerer String
