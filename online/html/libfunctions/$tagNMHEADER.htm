$tagNMHEADER

Beschreibung anzeigen in

Enthält Informationen über Control-Benachrichtigungen

Global Const $tagNMHEADER = $tagNMHDR & ";int Item;int Button;ptr pItem"

Parameter

$tagNMHDR Enthält Informationen über den Benachrichtigungscode
Item Der 0-basierte Index des Controls auf das diese Struktur verweist
Button Index der Maustaste die benutzt wird um diese Nachricht zu erzeugen. Dieses Mitglied kann einer der folgenden Werte sein:
    0 - Linke Taste
    1 - Rechte Taste
    2 - Mittlere Taste
pItem Optionaler Zeiger auf eine tagHDITEM Struktur welche Informationen über das Element in Item enthält.
Das mask Mitglied der tagHDITEM Struktur weist darauf hin, welche seiner Mitglieder gültig sind.

Verwandte Funktionen

$tagNMHDR
