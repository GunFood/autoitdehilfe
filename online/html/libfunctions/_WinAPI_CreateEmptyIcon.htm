_WinAPI_CreateEmptyIcon

Beschreibung anzeigen in

Erzeugt ein vollkommen transparentes Icon mit der angegebenen Länge, Höhe und Farbtiefe.

#include <WinAPIIcons.au3>
_WinAPI_CreateEmptyIcon ( $iWidth, $iHeight [, $iBitsPerPel = 32] )

Parameter

$iWidth Die Länge des Icons in Pixeln.
$iHeight Die Höhe des Icons in Pixeln.
$iBitsPerPel [optional] Die Anzahl der Bits pro Pixel in der XOR-Bitmaske des Icons. Standard ist 32.

Rückgabewert

Erfolg: Das Handle zum erzeugten Icon.
Fehler: 0 und setzt das @error Flag auf ungleich 0.

Bemerkungen

Diese Funktion erzeugt ein Icon mit einer schwarzen XOR- und weißen AND-Bitmasken Bitmaps
Das Icon hat keinen Alphakanal, es wird nur angezeigt wenn man ihre AND-Bitmaske verwendet.

Wenn die Arbeit mit der Bitmap abgeschlossen ist, so sollte man sie mit der _WinAPI_DeleteObject() Funktion zerstören.
