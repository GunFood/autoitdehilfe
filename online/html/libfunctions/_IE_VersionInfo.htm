Funktionreferenz

_IE_VersionInfo

Beschreibung anzeigen in

Gibt ein Array mit Informationen über die aktuelle IE.au3-Version zurück

#include <IE.au3>
_IE_VersionInfo ( )

Rückgabewert

Gibt ein Array mit den Informationen zurück ($__gaIEAU3VersionInfo)
    $__gaIEAU3VersionInfo[0] = Versionstyp (T=Test oder V=Stabil)
    $__gaIEAU3VersionInfo[1] = Hauptversion
    $__gaIEAU3VersionInfo[2] = Kleine Version
    $__gaIEAU3VersionInfo[3] = Unterversion
    $__gaIEAU3VersionInfo[4] = Herausgabe Datum (JJMMTT) (Jahr Monat Tag)
    $__gaIEAU3VersionInfo[5] = Anzeigeversion (z. B. V3 1-0)

Bemerkungen

Der Inhalt des Arrays erlaubt eine einfache Überprüfung, ob die IE.au3 Version ausreicht, um bestimmte Funktionen auszuführen.

Beispiel

#include <IE.au3>
#include <MsgBoxConstants.au3>

; Abrufen und anzeigen von IE.au3 Versionsinformationen

Local $aVersion = _IE_VersionInfo()
MsgBox($MB_SYSTEMMODAL, "IE.au3 Version", $aVersion[5] & " veröffentlicht " & $aVersion[4]) 4])
