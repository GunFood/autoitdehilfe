Parameter Struktur für Schärfungseffekt

Global Const $tagGDIP_EFFECTPARAMS_Sharpen = "float Radius; float Amount"

Parameter

Radius Reale Zahl welche den Scharfzeichen-Radius (den Radius des Convolution Kernel) in pixeln festlegt.
Der Radius muss im Bereich von 0.0 bis 255.0 sein.
Amount Reale Zahl im Bereich von 0.0 bis 100.0, welche den Schärfungsgrad festlegt.
