_GDIPlus_GraphicsReleaseDC

Beschreibung anzeigen in

Gibt den Gerätekontext des Grafik-Objekts frei

#include <GDIPlus.au3>
_GDIPlus_GraphicsReleaseDC ( $hGraphics, $hDC )

Parameter

$hGraphics Handle zu einem Grafik-Objekt
$hDC Handle zu dem Gerätekontext der Grafik

Rückgabewert

Erfolg: True
Fehler: False und setzt das @error Flag auf ungleich null, das @extended Flag kann den GPSTATUS-Fehlercode ($GPID_ERR* siehe GDIPlusConstants.au3) enthalten.

Verwandte Funktionen

_GDIPlus_GraphicsGetDC
