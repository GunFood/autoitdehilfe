_WinNet_CancelConnection

Beschreibung anzeigen in

Beendet eine bestehende Netzwerkverbindung

#include <WinNet.au3>
_WinNet_CancelConnection ( $sName [, $bForce = True] )

Parameter

$sName Name des umgeleiteten lokalen Geräts oder der fernen Netzwerkressource, das/die getrennt werden soll.
Wenn ein umgeleitetes lokales Gerät angegeben ist, wird nur diese Umleitung beendet.
Wenn eine ferne Netzwerkressource angegeben ist, wird nur die Verbindung zu diesem Netzwerk getrennt.
$bForce [optional] Legt fest, ob die Trennung auch erfolgen soll, wenn noch Dateien oder Aufträge in Arbeit sind.
Wenn dieser Parameter False ist, bricht die Funktion mit einem Fehler ab, falls noch Dateien oder Aufträge geöffnet sind.

Rückgabewert

Erfolg: True
Fehler: False

Bemerkungen

Diese Funktion ist nur für die Abwärtskompatibilität zu den 16bit-Versionen von Windows zu verwenden.
Sonst sollte die _WinNet_CancelConnection2()-Funktion verwendet werden.

Verwandte Funktionen
