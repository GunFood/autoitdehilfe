Binary



Beschreibung anzeigen in

Gibt die binäre Darstellung eines Ausdruckes zurück.

Binary ( expression )

Parameter

expression Ausdruck welcher in binäre oder Byte Daten konvertiert werden soll.

Rückgabewert

Gibt eine binäre Variante zurück.

Bemerkungen

Siehe Sprachreferenz - Datentypen für eine genaue Beschreibung.

Verwandte Funktionen

BinaryLen, BinaryMid, BinaryToString, Int, IsBinary, Number, StringToBinary

Beispiel

#include <MsgBoxConstants.au3>

Local $dBinary = Binary("0x00204060") ; konvertiert einen String in eine binäre Darstellung

MsgBox($MB_SYSTEMMODAL, "Gibt 1 zurück, wenn die Variable binär ist", IsBinary($dBinary))
