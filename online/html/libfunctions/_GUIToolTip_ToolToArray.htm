_GUIToolTip_ToolToArray



Beschreibung anzeigen in

Schreibt die ToolInfo Struktur in ein Array

#include <GuiToolTip.au3>
_GUIToolTip_ToolToArray ( $hWnd, ByRef $tToolInfo, $iError )

Parameter

$hWnd Handle des Controls (welches von _GUIToolTip_Create zurückgegeben wird.)
$tToolInfo $tagTOOLINFO Struktur
$iError Zurückzugebender Fehler Code

Rückgabewert

Erfolg: Array mit dem folgenden Format:
    [0] - Flags welche die ToolTip Anzeige steuern:
        TTF_IDISHWND = Zeigt an, dass $iID das Fenster Handle des Tools ist, anstatt der ID
        TTF_CENTERTIP = Zentriert das Fenster, zugehörig zu dem durch $iID festgelegten Tool
        TTF_RTLREADING = Zeigt an, dass Text in entgegengesetzter Richtung dargestellt wird
        TTF_SUBCLASS = Zeigt an, dass das Control das Fenster des Tools unterklassifizieren soll, um Nachrichten abzufangen
        TTF_TRACK = Positioniert das Control neben dem zugehörigen Tool
        TTF_ABSOLUTE = Positioniert das Fenster bei den gleichen Koordinaten wie TTM_TRACKPOSITION
        TTF_TRANSPARENT = Veranlasst das Control, Mausnachrichten an das Parent-Fenster weiterzuleiten
        TTF_PARSELINKS = Zeigt an, dass Links im Text des Controls geparst werden sollen
    [1] - Handle zum Fenster, welches das Tool enthält
    [2] - ID des Tools
    [3] - X Koordinate der oberen linken Ecke des Rechtecks
    [4] - Y Koordinate der oberen linken Ecke des Rechtecks
    [5] - X Koordinate der unteren rechten Ecke des Rechtecks
    [6] - Y Koordinate der unteren rechten Ecke des Rechtecks
    [7] - Handle zu der Instanz, welche die Stringressource für das Tool enthält
    [8] - Text für das Tool
    [9] - anwendungsspezifischer Wert, der mit dem Tool verknüpft ist

Bemerkungen

$iError ist der Rückgabecode, den die Funktion an den aufrufenden Prozess zurückgeben soll.
Diese Funktion wird normalerweise zusammen mit anderen UDF-Funktionen verwendet, um die $tagTOOLINFO-Struktur zu Anzeigezwecken in ein Array zu zerlegen.
