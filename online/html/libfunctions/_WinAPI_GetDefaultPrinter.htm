_WinAPI_GetDefaultPrinter

Beschreibung anzeigen in

Ermittelt den Namen des Standarddruckers für den aktuellen Benutzer auf dem lokalen Computer

#include <WinAPISys.au3>
_WinAPI_GetDefaultPrinter ( )

Rückgabewert

Erfolg: String, welcher den Namen des Standarddruckers enthält.
Fehler: Leerer String und setzt das @error Flag auf ungleich null, @extended kann Systemfehler enthalten:
2 - Kein Standarddrucker
