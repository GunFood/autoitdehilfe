BinaryLen

Beschreibung anzeigen in

Gibt die Anzahl von Bytes einer Binärvariablen zurück.

BinaryLen ( binary )

Parameter

binary Die zu überprüfenden Binärdaten.

Rückgabewert

Gibt die Länge der Binärdaten in Bytes zurück.

Verwandte Funktionen

Binary, BinaryMid

Beispiel

#include <MsgBoxConstants.au3>

Local $dBinary = Binary("0x10203040") ; Erstellt binäre Daten aus einem String
MsgBox($MB_SYSTEMMODAL, "Die Länge der binären Daten ist:", BinaryLen($dBinary))
