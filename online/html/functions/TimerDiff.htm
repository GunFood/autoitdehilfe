Funktionreferenz

TimerDiff

Beschreibung anzeigen in

Gibt den Zeitunterschied zu einem vorherigen Aufruf der Funktion TimerInit() zurück.

TimerDiff ( handle )

Parameter

handle Handle zurückgegeben durch einen vorherigen Aufruf von TimerInit().

Rückgabewert

Gibt in Millisekunden den Zeitunterschied zu einem vorherigen Aufruf von TimerInit() zurück.

Bemerkungen

Bei einigen Prozessoren liefert diese Funktion falsche Ergebnisse.

Verwandte Funktionen

TimerInit

Beispiel

#include <MsgBoxConstants.au3>

Local $hTimer = TimerInit() ; Beginnt den Timer speichert das Handle in einer Variable.
Sleep(3000) ; Sleep für 3 Sekunden.
Local $fDiff = TimerDiff($hTimer) ; Findet den Zeitunterschied zum vorherigen Aufruf von TimerInit. Die Variable wird im TimerInit Handle gespeichert und als Handle an TimerDiff übergeben.
MsgBox($MB_SYSTEMMODAL, "Zeitunterschied", $fDiff)
