_GUIToolTip_Update

Beschreibung anzeigen in

Erzwingt das aktuelle Tool zum Neuaufbau

#include <GuiToolTip.au3>
_GUIToolTip_Update ( $hTool )

Parameter

$hTool Handle des Controls (welches von _GUIToolTip_Create zurückgegeben wird.)

Rückgabewert

Keine.

Bemerkungen

Diese Funktion wird das aktuelle ToolTip neu zeichnen. Es wird keinen Effekt auf ein Tool haben welches nicht angezeigt wird während diese Funktion ausgeführt wird.
