_INetExplorerCapable



Beschreibung anzeigen in

Konvertiert einen String in eine IE(Internet Explorer) ausführbare Zeile

#include <INet.au3>
_INetExplorerCapable ( $sIEString )

Parameter

$sIEString String der konvertiert werden soll

Rückgabewert

Erfolg: der konvertierte String
Fehler: ein leeren String und setzt @error auf ungleich 0

Bemerkungen

Dies konvertiert IE unausführbare Buchstaben zu '%' und ihren Hex-Werten.

Beispiel

#include <Inet.au3>
#include <MsgBoxConstants.au3>

MsgBox($MB_SYSTEMMODAL, "", _INetExplorerCapable(InputBox("", "Gib einen String ein, der in einen Text umgewandelt werden soll, der vom Browser gelesen werden kann:")))
