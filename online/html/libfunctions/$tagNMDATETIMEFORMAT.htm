Schlüsselwortreferenz

$tagNMDATETIMEFORMAT

Beschreibung anzeigen in


Enthält Informationen über einen Teilbereich des Strings, die das Rückruffeld innerhalb eines DTP-Controls definiert

Global Const $tagNMDATETIMEFORMAT = $tagNMHDR & ";ptr Format;" & $tagSYSTEMTIME & ";ptr pDisplay;wchar Display[64]"

Parameter

$tagNMHDR Enthält Informationen über den Benachrichtigungscode
Format Zeiger auf einen null-terminierten String, welche das Rückruffeld des Controls definiert. Der String umfasst ein oder mehrere "X" Zeichen.
$tagSYSTEMTIME Datum und Uhrzeit die formatiert wird.
pDisplay Zeiger auf einen null-terminierten String, die den Anzeigetext des Controls enthält. Standardmäßig ist dies die Adresse des pDisplay-Mitglieds dieser Struktur.
Ein existierender String in pDisplay als Verweis wird akzeptiert. In diesem Fall muss kein Wert mit pDisplay verknüpft werden.
Jedoch muss der String der pDisplay verweist, solange gültig bleiben, bis eine andere $DTN_FORMAT-Benachrichtigung gesendet wird oder bis das Control zerstört wird.
Display 64 Zeichen Puffer, dazu dienlich, um den Null-terminierten String zu ermitteln, welche den Text des Controls anzeigen soll. Es ist nicht nötig, den gesamten Puffer zu füllen.

Bemerkungen

Sie überträgt den String, welcher das Rückruffeld definiert und enthält einen Puffer, um den String zu ermitteln, der in dem Rückruffeld angezeigt wird.
Diese Struktur wird bei der $DTN_FORMAT-Benachrichtigung verwendet.

Verwandte Funktionen

$tagNMHDR, $tagSYSTEMTIME
