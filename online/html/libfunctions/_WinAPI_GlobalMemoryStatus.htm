_WinAPI_GlobalMemoryStatus



Beschreibung anzeigen in

Ermittelt Informationen über den aktuell verfügbaren Speicher

#include <WinAPIMem.au3>
_WinAPI_GlobalMemoryStatus ( )

Rückgabewert

Erfolg: Array mit dem folgenden Format:
$aMem[0] - Prozent vom Speicher in Gebrauch
$aMem[1] - Physischer Speicher: Gesamt
$aMem[2] - Physischer Speicher: Frei
$aMem[3] - Auslagerungsdatei: Gesamt
$aMem[4] - Auslagerungsdatei: Frei
$aMem[5] - Benutzer Speicher: Gesamt
$aMem[6] - Benutzer Speicher: Frei
Fehler: Setzt das @error Flag auf ungleich null. Es ist _WinAPI_GetLastError() aufzurufen um erweiterte Fehlerinformationen zu erhalten.

Bemerkungen

Anders als die AutoIt MemGetStats() Funktion, gibt diese Funktion die Werte in Bytes zurück
