_GUICtrlSlider_SetTipSide



Beschreibung anzeigen in

Positioniert ein ToolTip-Control

#include <GuiSlider.au3>
_GUICtrlSlider_SetTipSide ( $hWnd, $iLocation )

Parameter

$hWnd Control-ID / Handle des Controls
$iLocation Den Ort an dem das ToolTip-Control dargestellt wird. Dieser kann einer der folgenden Werte sein:
    $TBTS_TOP - wird über dem Slider positioniert. Dieses Flag ist für horizontale Slider zu verwenden.
    $TBTS_LEFT - wird links vom Slider positioniert. Dieses Flag ist für vertikale Slider zu verwenden.
    $TBTS_BOTTOM - wird unter dem Slider positioniert. Dieses Flag ist für horizontale Slider zu verwenden.
    $TBTS_RIGHT - wird rechts vom Slider positioniert. Dieses Flag ist für vertikale Slider zu verwenden.

Rückgabewert

Gibt den Ort des obigen ToolTip Controls zurück.
Der Rückgabewert entspricht einem der möglichen Werte von $iLocation

Bemerkungen

Es ist der Stil $TBS_TOOLTIPS zu verwenden um ToolTips anzuzeigen

- - - - - - - - Erklärung der Controls - - - - - - - -

Beispiel

#include <GuiConstantsEx.au3>
#include <GuiSlider.au3>

Example()

Func Example()
    Local $idRdoBottom, $idRdoLeft, $idRdoRight, $idRdoTop, $idSlider, $idSlider2

    ; Erstellt eine GUI
    GUICreate("Slider: Positioniert ein ToolTip-Control", 400, 296)
    $idSlider = GUICtrlCreateSlider(2, 2, 375, 20, BitOR($TBS_TOOLTIPS, $TBS_AUTOTICKS))
    $idSlider2 = GUICtrlCreateSlider(380, 2, 20, 292, BitOR($TBS_TOOLTIPS, $TBS_AUTOTICKS, $TBS_VERT))
    GUISetState(@SW_SHOW)

    GUICtrlCreateGroup("ToolTip horizontal", 2, 25, 120, 120)
    $idRdoBottom = GUICtrlCreateRadio("Unten", 5, 40, 108, 20)
    $idRdoTop = GUICtrlCreateRadio("Oben", 5, 115, 108, 20)
    GUICtrlCreateGroup("", -99, -99, 1, 1)
    GUICtrlSetState($idRdoTop, $GUI_CHECKED)

    GUICtrlCreateGroup("ToolTip vertikal", 130, 25, 120, 120)
    $idRdoLeft = GUICtrlCreateRadio("Links", 132, 65, 108, 20)
    $idRdoRight = GUICtrlCreateRadio("Rechts", 132, 90, 108, 20)
    GUICtrlCreateGroup("", -99, -99, 1, 1)
    GUICtrlSetState($idRdoLeft, $GUI_CHECKED)

    ; Die Schleife wiederholt sich, bis der Benutzer die Beenden-Aktion der GUI auslöst
    While 1
        Switch GUIGetMsg()
            Case $GUI_EVENT_CLOSE
                ExitLoop
            Case $idRdoBottom
                _GUICtrlSlider_SetTipSide($idSlider, $TBTS_BOTTOM)
            Case $idRdoLeft
                _GUICtrlSlider_SetTipSide($idSlider2, $TBTS_LEFT)
            Case $idRdoRight
                _GUICtrlSlider_SetTipSide($idSlider2, $TBTS_RIGHT)
            Case $idRdoTop
                _GUICtrlSlider_SetTipSide($idSlider, $TBTS_TOP)
        EndSwitch
    WEnd
    GUIDelete()
EndFunc   ;==>Example
