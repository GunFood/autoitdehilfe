_WinAPI_ReleaseStream

Beschreibung anzeigen in

Gibt ein Stream-Objekt frei

#include <WinAPICom.au3>
_WinAPI_ReleaseStream ( $pStream )

Parameter

$pStream Zeiger auf das Stream-Objekt, das zuvor durch einen Aufruf der Funktion _WinAPI_CreateStreamOnHGlobal() erzeugt wurde.

Rückgabewert

Erfolg: 1.
Fehler: 0 und setzt das @error Flag auf ungleich 0, das @extended Flag kann den HRESULT-Fehlercode enthalten.

Verwandte Funktionen

_WinAPI_CreateStreamOnHGlobal
