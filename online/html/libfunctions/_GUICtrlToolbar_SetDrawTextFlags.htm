_GUICtrlToolbar_SetDrawTextFlags

Beschreibung anzeigen in

Setzt das Textzeichen Flag für die Toolbar

#include <GuiToolBar.au3>
_GUICtrlToolbar_SetDrawTextFlags ( $hWnd, $iMask, $iDTFlags )

Parameter

$hWnd Handle des Controls
$iMask Eine oder mehrere DT_ Flags festgelegt in DrawText. Dies zeigt an, welche Bits in $iDTFlags verwendet werden, wenn der Text gezeichnet wird.
$iDTFlags Eine oder mehrere DT_ Flags festgelegt in DrawText. Dies zeigt an wie Text gezeichnet wird. Dieser Wert wird zu dem DrawText API übergeben wenn der Button Text gezeichnet wird.

Rückgabewert

Gibt die vorherigen Text Drawing Flags zurück

Bemerkungen

Der $iMask Parameter erlaubt es dem Benutzer festzulegen, welche Flags verwendet werden wenn der Text gezeichnet wird, auch wenn diese Flags abgeschaltet sind.
Zum Beispiel, wenn man nicht will, dass das $DT_CENTER Flag verwendet wird wenn Text gezeichnet wird, sollte man das $DT_CENTER Flag in iMask hinzufügen und das $DT_CENTER Flag in iDTFlags nicht angeben.
Dies verhindert das übergeben des $DT_CENTER Flag an das DrawText API.
