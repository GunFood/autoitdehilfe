_WinAPI_StrFormatByteSizeEx

Beschreibung anzeigen in

Wandelt einen numerischen Wert in eine Zeichenfolge um, die die links vom Dezimalzeichen ausgedrückte Zahl als (durch Punkt) getrennte Gruppen von Ziffern darstellt (Tausenderblöcke)

#include <WinAPIConv.au3>
_WinAPI_StrFormatByteSizeEx ( $iSize )

Parameter

$iSize Der umzuwandelnde numerische Wert.

Rückgabewert

Erfolg: Die umgewandelte Zeichenfolge.
Fehler: Leerstring und setzt das @error Flag auf ungleich Null.

Beispiel

#include <WinAPIConv.au3>

Local $aVal[5] = [532, 1340, 23506, 2400016, 2400000000]

For $i = 0 To 4
    ConsoleWrite(StringFormat('%10s %19s', $aVal[$i], _WinAPI_StrFormatByteSizeEx($aVal[$i])) & @CRLF)
