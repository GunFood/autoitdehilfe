_WinAPI_SetErrorMode

Beschreibung anzeigen in

Kontrolliert, ob das System die angegebenen Arten von schweren Fehlern behandelt oder ob der Prozess sie behandelt

#include <WinAPIDiag.au3>
_WinAPI_SetErrorMode ( $iMode )

Parameter

$iMode Der Prozessfehlermodus. Dieser Parameter kann einer oder mehrere der folgenden Werte sein.
$SEM_FAILCRITICALERRORS
$SEM_NOALIGNMENTFAULTEXCEPT
$SEM_NOGPFAULTERRORBOX
$SEM_NOOPENFILEERRORBOX

Rückgabewert

Gibt den vorherigen Zustand der Fehlermodus-Bitflags ($SEM_*) zurück.
