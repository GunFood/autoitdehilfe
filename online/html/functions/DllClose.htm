DllClose

Beschreibung anzeigen in

Schließt eine zuvor geöffnete DLL.

DllClose ( dllhandle )

Parameter

dllhandle Das Handle einer DLL, wie von einem früheren Aufruf von DllOpen() zurückgegeben.

Rückgabewert

Keine.

Bemerkungen

Beim Beenden des Skriptes schließt AutoIt normalerweise automatisch alle offenen DLLs; der Aufruf von DllClose() ist aber dennoch eine gute Idee.

Verwandte Funktionen

DllCall, DllOpen

Beispiel

Local $hDLL = DllOpen("user32.dll")
DllCall($hDLL, "int", "MessageBox", "hwnd", 0, "str", "Etwas Text", "str", "Ein Titel", "int", 0)
DllClose($hDLL)
