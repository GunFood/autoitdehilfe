_WinAPI_DrawThemeParentBackground



Beschreibung anzeigen in

Zeichnet den Bereich des Parentcontrols welcher von einem semi-transparenten oder alpha-überblendeten Childcontrol überdeckt wird

#include <WinAPITheme.au3>
_WinAPI_DrawThemeParentBackground ( $hWnd, $hDC [, $tRECT = 0] )

Parameter

$hWnd Das Handle eines Childcontrols.
$hDC Das Handle zum Gerätekontext eines Childcontrols.
$tRECT [optional] Die $tag$tagRECT Struktur welche den zu zeichnenden Bereich im Childfenster definiert.

Rückgabewert

Erfolg: 1.
Fehler: 0 und setzt @error auf ungleich 0. @extended kann den HRESULT Fehlercode enthalten.
