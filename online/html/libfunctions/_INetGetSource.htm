_INetGetSource

Beschreibung anzeigen in

Ermittelt den Quelltext von einer URL ohne eine temporäre Datei zu schreiben

#include <INet.au3>
_INetGetSource ( $sURL [, $bString = True] )

Parameter

$sURL Die URL der Seite, z. B. 'https://www.autoit.de'
$bString [optional] Falls True werden die Daten im Stringformat zurückgegeben, sonst im Binärformat.

Rückgabewert

Erfolg: den Quelltext und setzt @extended auf die zurückgegebenen Bytes.
Fehler: Leerer String und setzt das @error Flag auf ungleich 0

Bemerkungen

Bei $sURL muß am Anfang unbedingt "http://" bzw. "https://" stehen, sonst ist der Rückgabewert leer.

Verwandte Funktionen

_IEDocReadHTML

Beispiel

#include <Inet.au3>

ConsoleWrite(_INetGetSource('http://www.autoitscript.com') & @CRLF)
