$tagCOLORSCHEME

Beschreibung anzeigen in

Enthält Informationen für das Zeichnen von Buttons in einer Toolbar oder Rebar

Global Const $tagCOLORSCHEME = "dword Size;dword BtnHighlight;dword BtnShadow"

Parameter

Size Größe dieser Struktur in Bytes
BtnHighlight Der COLORREF Wert der die Farbe für das Hervorheben der Buttons darstellt. Für die Standardfarbe ist $CLR_DEFAULT zu verwenden.
BtnShadow Der COLORREF Wert der die Schattenfarbe für die Buttons darstellt. Für die Standardfarbe ist $CLR_DEFAULT zu verwenden.
