_NowTime

Beschreibung anzeigen in


Gibt die aktuelle Zeit im angegebenen Format zurück

#include <Date.au3>
_NowTime ( [$sType = 3] )

Parameter

$sType [optional]
    3 = Zeigt die Zeit im Format an, welches in den Regions- und Sprachoptionen auf dem PC eingestellt ist. (Standard)
    4 - Zeigt die Zeit im 24-Stunden Format an (HH:MM).
    5 - Zeigt die Zeit mit Sekunden im 24-Stunden Format an (HH:MM:SS).

Rückgabewert

Gibt die aktuelle Zeit im angegebenen Format zurück.

Bemerkungen

Falls das Zeitformat auf dem Computer nicht zugänglich ist, wird die Zeit im Format "h:mm:ss tt" zurückgegeben.

Verwandte Funktionen

_DateTimeFormat, _Now, _NowCalc, _NowCalcDate, _NowDate

Beispiel

#include <Date.au3>
#include <MsgBoxConstants.au3>

MsgBox($MB_SYSTEMMODAL, '', "Es ist jetzt " & _NowTime() & " Uhr.")
