_WinAPI_ArrayToStruct

Beschreibung anzeigen in

Konvertiert ein Array von Strings in die Struktur

#include <WinAPIMisc.au3>
_WinAPI_ArrayToStruct ( Const ByRef $aData [, $iStart = 0 [, $iEnd = -1]] )

Parameter

$aData Das zu konvertierende Array.
$iStart [optional] Der Index des Arrays, bei dem die Konvertierung beginnen soll.
$iEnd [optional] Der Index des Arrays, bei dem die Konvertierung beendet werden soll.

Rückgabewert

Erfolg: die Struktur vom Typ "string1;{0};string2;{0}; ... ;stringN;{0};{0}".
Fehler: setzt das @error-Flag auf einen Wert ungleich Null.

Bemerkungen

Diese Funktion funktioniert nur für Unicode-Strings.
