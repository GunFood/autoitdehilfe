Schlüsselwortreferenz

Es folgt eine komplette Liste der Schlüsselwörter die in AutoIt verfügbar sind. Ein Klick auf einen Funktionsnamen liefert eine detaillierte Beschreibung.

 

Schlüsselwörter Beschreibung
False / True Bool'sche Werte zur Verwendung in logischen Ausdrücken.
#comments-start Gibt an, dass ein ganzer Teil eines Skriptes auskommentiert werden soll.
ContinueCase Bricht den Code im aktuellen Case-Tree ab und setzt den nächsten Case-Tree in einer Select oder Switch Struktur fort.
ContinueLoop Setzt eine While-, Do- oder For-Schleife fort.
Default Schlüsselwort Wert verwendet im Funktionsaufruf.
Dim / Global / Local / Const Deklariert eine Variable, Konstante oder definiert ein Array.
Do...Until Schleife, die auf einem Ausdruck basiert.
Enum Zählt Konstanten auf.
Exit Beendet das Skript.
ExitLoop Beendet eine While-, Do- oder For-Schleife.
For...To...Step...Next Schleife, die auf einem Ausdruck basiert.
For...In...Next Durchläuft Elemente in einer Objektsammlung oder einem Array.
Func...Return...EndFunc Definiert eine Benutzerfunktion, die Null oder mehr Argumente aufnimmt und optional ein Ergebnis zurückgibt.
If...Then Führt eine einzige Anweisung konditionell aus.
If...ElseIf...Else...EndIf Führt Anweisungen konditionell aus.
#include-once Legt fest, dass die aktuelle Datei nur einmal inkludiert wird.
#include Bindet eine Datei in das aktuelle Skript ein.
#NoTrayIcon Bewirkt, dass das AutoIt-Icon nicht im Systeminfo-Bereich der Taskleiste erscheint, wenn das Skript gestartet wird.
Null Schlüsselwert welcher im Funktionsaufruf genutzt werden kann.
#OnAutoItStartRegister Registriert eine Funktion, wenn AutoIt gestartet wird.
#pragma A special directive for controlling aspects of how the script is compiled.
ReDim Verändert die Größe eines bestehenden Arrays.
#RequireAdmin Legt fest, dass das aktuelle Skript die vollen Administratorrechte benötigt, um zu laufen.
Select...Case...EndSelect Führt Anweisungen konditionell aus.
Static Deklariert eine statische Variable oder erzeugt ein statisches Array.
Switch...Case...EndSwitch Führt bedingt Anweisungen aus.
Volatile Function qualifier.
While...WEnd Schleife, die auf einem Ausdruck basiert.
With...EndWith Wird benutzt um lange Referenzen zu punktlastigen Objekttypvariablen zu reduzieren.
