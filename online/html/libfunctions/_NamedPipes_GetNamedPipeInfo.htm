_NamedPipes_GetNamedPipeInfo

Beschreibung anzeigen in

Ermittelt Informationen über die angeführte Benannte Pipe

#include <NamedPipes.au3>
_NamedPipes_GetNamedPipeInfo ( $hNamedPipe )

Parameter

$hNamedPipe Zugriff zu der Benannten Pipe Instanz. Der Zugriff muss GENERIC_READ Rechte zu der Benannten Pipe haben

Rückgabewert

Erfolg: ein Array mit dem folgenden Format:
    $aInfo[0] - True falls das Handle auf ein Server Ende verweist, andernfalls Client Ende
    $aInfo[1] - True für eine Nachrichten Pipe, andernfalls Byte Pipe
    $aInfo[2] - Größe des Puffers für ausgehende Daten, in Bytes
    $aInfo[3] - Größe des Puffers für eingehende Daten, in Bytes
    $aInfo[4] - Maximale Anzahl der Pipe Instanzen die erstellt werden können
Fehler: Setzt das @error Flag auf ungleich null
