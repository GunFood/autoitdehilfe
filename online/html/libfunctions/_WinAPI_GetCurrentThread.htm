_WinAPI_GetCurrentThread

Ermittelt ein Pseudo-Handle für den aufrufenden Prozess

#include <WinAPIProc.au3>
_WinAPI_GetCurrentThread ( )

Rückgabewert

Gibt das Pseudo-Handle für den aktuellen Prozess zurück

Bemerkungen

Ein Pseudo-Handle ist eine spezielle Konstante die als Handle des aktuellen Prozesses interpretiert wird.
Der aufrufende Thread kann Gebrauch von diesem Handle machen um sich selbst zu benennen, wann immer ein Thread-Handle benötigt wird.
