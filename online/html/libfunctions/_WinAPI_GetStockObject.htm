_WinAPI_GetStockObject



Beschreibung anzeigen in

Ermittelt ein Handle zu einem der vordefinierten Zeichenstifte, Füllmuster, Schriften oder Paletten des Systems

#include <WinAPIHObj.au3>
_WinAPI_GetStockObject ( $iObject )

Parameter

$iObject Legt den Typ des Objekts fest. Dieser Parameter kann einer der folgenden Werte sein:
$BLACK_BRUSH - Schwarzes Füllmuster
$DKGRAY_BRUSH - Dunkelgraues Füllmuster
$GREY_BRUSH - Graues Füllmuster
$HOLLOW_BRUSH - Transparentes Füllmuster (gleichbedeutend mit NULL_BRUSH)
$LTGRAY_BRUSH - Hellgraues Füllmuster
$NULL_BRUSH - Kein Füllmuster (gleichbedeutend mit HOLLOW_BRUSH)
$WITHE_BRUSH - Weißes Füllmuster
$BLACK_PEN - Schwarzer Zeichenstift
$NULL_PEN - Kein Zeichenstift
$WITHE_PEN - Weisser Zeichenstift
$ANSI_FIXED_FONT - Windows nicht-proportionale (feste Abstände) Systemschrift
$ANSI_VAR_FONT - Windows proportionale (proportionale Abstände) Systemschrift
$DEVICE_DEFAULT_FONT - Geräteabhängige Schrift
$DEFAULT_GUI_FONT - Standard Schrift für Benutzerschnittstellen-Objekte
$OEM_FIXED_FONT - OEM-Abhängige, nicht-proportionale (feste Abstände) Schrift (MSDOS)
$SYSTEM_FONT - Proportionale Systemschrift
$SYSTEM_FIXED_FONT - Nicht-proportionale (feste Abstände) Systemschrift, genutzt in Windows-Versionen vor 3.0
$DEFAULT_PALETTE - Standard Palette. Diese Palette besteht aus den statischen Farben in der System Palette

Rückgabewert

Erfolg: Das gewünschte logische Objekt
Fehler: 0
