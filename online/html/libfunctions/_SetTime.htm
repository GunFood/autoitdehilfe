_SetTime



Beschreibung anzeigen in

Setzt die aktuelle Systemzeit

#include <Date.au3>
_SetTime ( $iHour, $iMinute [, $iSecond = 0 [, $iMSeconds = 0]] )

Parameter

$iHour Stunde. Werte: 0-23
$iMinute Minute. Werte: 0-59
$iSecond [optional] Sekunden. Werte: 0-59
$iMSeconds [optional] Millisekunden. Werte: 0-999

Rückgabewert

Erfolg: 1
Fehler: 0 und setzt @error auf ungleich 0
@Error: 1 - Fehler
@extended: _WinAPI_GetLastError()
Fehlercodes: MSDN: ms681381.aspx

Bemerkungen

Falls die optionalen Parameter ($iSecond und iMSeconds) nicht definiert wird, wird die Funktion den aktuellen Wert nicht ändern.

Verwandte Funktionen

_SetDate

Siehe auch

Suche nach GetLastError in der MSDN Bibliothek.

Beispiel

#include <Date.au3>

_SetTime(20, 15) ; Setzt die Zeit auf 20:15
_SetTime(20, 15, 30) ; Setzt die Zeit auf 20:15:30
