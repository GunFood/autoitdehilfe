_WinNet_EnumResource



Beschreibung anzeigen in

Setzt die Auflistung der Netzwerk-Ressourcen fort

#include <WinNet.au3>
_WinNet_EnumResource ( $hEnum, ByRef $iCount, $pBuffer, ByRef $iBufSize )

Parameter

$hEnum Handle, das die Auflistungs-Instanz eindeutig identifiziert. Das Handle wird von der _WinNet_OpenEnum() Funktion zurückgegeben.
$iCount Anzahl der Einträge, die von der Funktion ermittelt werden sollen. Wird als Anzahl -1 angegeben, werden so viele Einträge wie möglich zurückgegeben.
Wenn die Funktion erfolgreich ausgeführt wurde, enthält die Variable die Anzahl der Einträge die derzeitig eingelesen wurden.
$pBuffer Zeiger auf den Puffer der die Auflistung aufnimmt. Das zurückgegebene Ergebnis, ist ein Array mit einer $tagNETRESOURCE Struktur.
Der Puffer muss groß genug sein, um die Struktur und die Strings aufzunehmen, auf welche die Strukturmitglieder verweisen.
Der Puffer ist solange gültig, bis der nächste Aufruf durch verwenden des Handles (festgelegt durch $hEnum) erfolgt.
Die Reihenfolge der $tagNETRESOURCE Struktur in dem Array ist nicht vorhersehbar.
$iBufSize Die Größe des Puffers für die Einträge, in Byte. Ist der Puffer zu klein um die Daten aufzunehmen, so wird die erforderliche Puffer-Größe zurückgegeben.

Rückgabewert

Erfolg: True
Fehler: False

Verwandte Funktionen

$tagNETRESOURCE, _WinNet_OpenEnum
