VarGetType



Beschreibung anzeigen in

Gibt den internen Datentyp eines Variants zurück.

VarGetType ( expression )

Parameter

expression Der Ausdruck, dessen Datentyp ermittelt werden soll.

Rückgabewert

Gibt einen String zurück, die den Datentyp des Ausdrucks beschreibt.

Bemerkungen

Siehe Sprachreferenz - Datentypen für eine detaillierte Beschreibung.
   
IsInt() kann abweichende Ergebnisse liefern, wenn eine String- oder Gleitkommaumwandlung erfolgt. Beispielsweise beim Aufruf von VarGetType() mit einem Wert von 2.0 wird "Double" zurückgegeben, aber IsInt() gibt für den gleichen Wert 1 zurück.

Verwandte Funktionen

IsArray, IsBinary, IsBool, IsDllStruct, IsFloat, IsFunc, IsHWnd, IsInt, IsKeyword, IsObj, IsPtr, IsString

Beispiel

#include <MsgBoxConstants.au3>

Local $aArray[2] = [1, "Beispiel"]
Local $mMap[]
Local $dBinary = Binary("0x00204060")
Local $bBoolean = False
Local $pPtr = Ptr(-1)
Local $hWnd = WinGetHandle("[CLASS:Shell_TrayWnd]")
Local $iInt = 1
Local $fFloat = 2.0
Local $oObject = ObjCreate("Scripting.Dictionary")
Local $sString = "Etwas Text"
Local $tStruct = DllStructCreate("wchar[256]")
Local $vKeyword = Default
Local $fuFunc = ConsoleWrite
Local $fuUserFunc = Test

MsgBox($MB_SYSTEMMODAL, "", _
        "Variablentypen" & @CRLF & @CRLF & _
        "$aArray ist ein " & @TAB & @TAB & VarGetType($aArray) & " Variablentyp." & @CRLF & _
