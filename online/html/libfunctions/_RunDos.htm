_RunDos

Beschreibung anzeigen in

Führt einen DOS-Befehl in einem nicht sichtbaren Fenster aus

#include <Process.au3>
_RunDos ( $sCommand )

Parameter

$sCommand Befehl der ausgeführt werden soll

Rückgabewert

Erfolg: der Exitcode des Befehles
Fehler: 0 und setzt das @error Flag auf ungleich null

Bemerkungen

Das Skript wartet bis der Befehl ausgeführt wurde.

Beispiel

#include <Process.au3>

Local $iRc = _RunDOS("start http://www.autoitscript.com")
