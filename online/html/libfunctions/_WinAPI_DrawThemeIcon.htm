_WinAPI_DrawThemeIcon

Beschreibung anzeigen in


Zeichnet ein Bild von einer Bilderliste mit dem Iconeffekt definiert durch den visuellen Stil

#include <WinAPITheme.au3>
_WinAPI_DrawThemeIcon ( $hTheme, $iPartID, $iStateID, $hDC, $tRECT, $hIL, $iIndex )

Parameter

$hTheme Das Handle, eines für ein Fenster, angegebener Themedaten.
$iPartID Der Bereich in welchem das Bild gezeichnet werden soll.
$iStateID Der Zustand des Bereiches.
$hDC Das Handle eines Gerätekontextes.
$tRECT Die $tag$tagRECT Struktur welche das Rechteck beinhaltet in welchem das Bild gezeichnet werden soll.
$hIL Das Handle einer Bilderliste welches das zu zeichnende Bild beinhaltet.
$iIndex Der Index des zu zeichnenden Bildes.

Rückgabewert

Erfolg: 1.
Fehler: 0 und setzt das @error Flag auf ungleich null, das @extended Flag kann den HRESULT-Fehlercode enthalten.
