SetExtended



Beschreibung anzeigen in

Manuelles Setzen des Wertes des @extended Makros.

SetExtended ( code [, return value] )

Parameter

code Der gewünschte Wert (als Ganzzahl), auf den @extended gesetzt werden soll.
return value [optional] Der Rückgabewert dieser Funktion - wird nichts angegeben ist der Rückgabewert undefiniert.

Rückgabewert

Gibt den Wert von Parameter "return value"(Rückgabewert) zurück. (Wird dieser Parameter nicht angegeben, ist der Rückgabewert undefiniert).
@extended wird auf den angegebenen Wert gesetzt. (Standard = 0)

Bemerkungen

Wenn eine Funktion gestartet wird, wird @extended auf 0 gesetzt. Sofern SetExtended() nicht aufgerufen wird, behält @extended den Wert 0, wenn die Funktion endet. Dies bedeutet, wenn @extended nach Ausführung der Funktion gesetzt sein soll, dies explizit in der Funktion erfolgen muss. Des Weiteren muss der Wert für @extended eventuell in einer Variablen zwischengespeichert werden, falls dieser Wert zur Abfrage einer While-Wend Schleife genutzt wird.
Der Parameter des Rückgabewerts ist optional. Es ist als eine Möglichkeit zu sehen, die zur Verwendung des Return SetExtended(...) Syntax bereitgestellt wird um den Wert zu definieren, der zur gleichen Zeit wie @extended gesetzt wird. Falls kein bestimmter Wert gesetzt wird, so wird der Rückgabewert undefiniert bleiben und sollte anschließend nicht verwendet werden.

@extended kann Werte von -2147483648 bis 2147483647 annehmen.

Verwandte Funktionen

SetError

Beispiel

#include <MsgBoxConstants.au3>

SetExtended(10)
MsgBox($MB_SYSTEMMODAL, "Der Wert von @Extended lautet", @extended)
