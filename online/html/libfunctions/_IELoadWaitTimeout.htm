Abfrage oder Setzen des _IELoadWait-Timeouts in Millisekunden

#include <IE.au3>
_IELoadWaitTimeout ( [$iTimeout = -1] )

Parameter

$iTimeout [optional] Abruf oder Setzen der Zahl in Millisekunden
    0 oder positive Ganzzahl setzt Timeout auf diesen Wert
    -1 = (Standard) Gibt den momentanen Timeout zurück (gespeichert in der globalen Variable $__IELoadWaitTimeout)

Rückgabewert

Wenn $iTimeout = -1, gibt die Funktion den momentanen Timeout zurück, ansonsten ist Rückgabe = 1

Bemerkungen

Es gibt keine Möglichkeit, den Timeout völlig abzustellen.
Funktionell kann man dies erreichen, indem man eine sehr hohe positive Ganzzahl angibt.
Der interne Anfangs-Timeout ist 300.000 ms (5 Minuten).

Verwandte Funktionen

_IELoadWait

Beispiel

Beispiel 1

#include <IE.au3>
#include <MsgBoxConstants.au3>

;Abfrage des momentanen Timeouts

Local $iCurrentTimeout = _IELoadWaitTimeout()
MsgBox($MB_SYSTEMMODAL, "_IELoadWaitTimeout", $iCurrentTimeout / 1000 & " sec")

Beispiel 2

#include <IE.au3>
#include <MsgBoxConstants.au3>

; Setzen des Timeouts auf 1 Minute (60000 Millisekunden)

_IELoadWaitTimeout(60000)

Local $iCurrentTimeout = _IELoadWaitTimeout()
MsgBox($MB_SYSTEMMODAL, "_IELoadWaitTimeout", $iCurrentTimeout / 1000 & " sec")
