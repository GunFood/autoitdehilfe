_WinAPI_DwmQueryThumbnailSourceSize



Beschreibung anzeigen in

Gibt die Quellgröße des Desktop-Fenster-Manager-Vorschaubildes zurück (DWM)

#include <WinAPIGdi.au3>
_WinAPI_DwmQueryThumbnailSourceSize ( $hThumbnail )

Parameter

$hThumbnail Das Handle des Vorschaubildes wessen Quellfenstergröße zurückgegeben werden soll.

Rückgabewert

Erfolg: Die $tagSIZE Struktur welche die Größe des Quellvorschaubildes beinhaltet.
Fehler: Setzt das @error Flag auf ungleich null, das @extended Flag kann den HRESULT-Fehlercode enthalten.

Bemerkungen

Diese Funktion benötigt Windows Vista oder höher.
