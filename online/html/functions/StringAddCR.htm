StringAddCR



Beschreibung anzeigen in

Nimmt einen String und setzt allen Zeilenwechsel-Zeichen ( Chr(10) ) ein Eingabe-Zeichen ( Chr(13) ) voran.

StringAddCR ( "string" )

Parameter

string String, der konvertiert werden soll.

Rückgabewert

Gibt den gleichen String zurück, wobei allen Zeilenwechsel Zeichen ( @LF ) ein Eingabe-Zeichen ( @CR ) vorangestellt wurde.

Verwandte Funktionen

StringReplace, StringStripCR

Beispiel

#include <MsgBoxConstants.au3>

; Setzt allen Zeilenwechsel-Zeichen ( Chr(10) ) ein Eingabe-Zeichen ( Chr(13) ) voran.
Local $sString = StringAddCR("Dies ist ein Satz " & @LF & "mit " & Chr(10) & "Leerzeichen.")
MsgBox($MB_SYSTEMMODAL, "", $sString)
