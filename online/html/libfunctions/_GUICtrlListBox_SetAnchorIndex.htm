Funktionreferenz

_GUICtrlListBox_SetAnchorIndex

Beschreibung anzeigen in


Setzt das erste Item von dem eine Mehrfachauswahl beginnt

#include <GuiListBox.au3>
_GUICtrlListBox_SetAnchorIndex ( $hWnd, $iIndex )

Parameter

$hWnd Control-ID / Handle des Controls
$iIndex legt den 0-basierenden Index des Items fest

Rückgabewert

Erfolg: True
Fehler: False

Bemerkungen

- - - - - - - - Erklärung der Controls - - - - - - - -

Verwandte Funktionen

_GUICtrlListBox_GetAnchorIndex

Beispiel

#include <GUIConstantsEx.au3>
#include <GuiListBox.au3>

Example()

Func Example()
    ; Erstellt eine GUI
    GUICreate("ListBox: Setzt und ermittelt den Anker-Index (v" & @AutoItVersion & ")", 400, 296)
    Local $idListBox = GUICtrlCreateList("", 2, 2, 396, 296)

    GUISetState(@SW_SHOW)

    ; Fügt Strings hinzu
    _GUICtrlListBox_BeginUpdate($idListBox)
    For $iI = 0 To 9
        _GUICtrlListBox_AddString($idListBox, StringFormat("%03d : String", $iI))
    Next
    _GUICtrlListBox_EndUpdate($idListBox)

    ; Setzt den Anker-Index
    _GUICtrlListBox_SetAnchorIndex($idListBox, 2)

    ; Liest den Anker-Index
    Local $iIndex = _GUICtrlListBox_GetAnchorIndex($idListBox)
    _GUICtrlListBox_SetCurSel($idListBox, $iIndex)

    ; Die Schleife wiederholt sich, bis der Benutzer die Beenden-Aktion der GUI auslöst.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>Example
