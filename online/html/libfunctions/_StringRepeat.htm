Funktionreferenz

_StringRepeat



Beschreibung anzeigen in

Ein String wird eine bestimmte Anzahl mal wiederholt

#include <String.au3>
_StringRepeat ( $sString, $iRepeatCount )

Parameter

$sString String, der wiederholt werden soll
$iRepeatCount Zahl der Wiederholungen vom String

Rückgabewert

Erfolg: ein String mit der bestimmten Anzahl
Fehler: "" (leerer String) und setzt @error auf ungleich 0
@error: 1 - Einer der Parameter ist ungültig

Beispiel

#include <MsgBoxConstants.au3>
#include <String.au3>

; Wiederholt den String "+-" 40 Mal
MsgBox($MB_SYSTEMMODAL, '', _StringRepeat("+-", 40))
