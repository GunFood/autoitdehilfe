_DateToDayOfWeekISO



Beschreibung anzeigen in

Gibt den Wochentag als Zahl im ISO-Format für ein vorgegebenes Datum zurück

#include <Date.au3>
_DateToDayOfWeekISO ( $iYear, $iMonth, $iDay )

Parameter

$iYear Ein gültiges Jahr im Format YYYY
$iMonth Ein gültiger Monat im Format MM
$iDay Ein gültiger Tag im Format DD

Rückgabewert

Erfolg: der Wochentag als Zahl (Bereich zwischen 1 und 7, wobei 1=Montag ist).
Fehler: 0 und setzt das @error Flag auf ungleich null
@error: 1 - ungültige Datumseingabe

Verwandte Funktionen

_DateAdd, _DateDayOfWeek, _DateDiff, _DateToDayOfWeek, _DayValueToDate

Beispiel

#include <Date.au3>
#include <MsgBoxConstants.au3>

; ISO Wochentagnummer des gegebenen Datums  1 = Montag - 7 = Sonntag
Local $iWeekday = _DateToDayOfWeekISO(@YEAR, @MON, @MDAY)
; Nicht gleich wie @WDAY
MsgBox($MB_SYSTEMMODAL, "", "Heute ist die ISO Wochentagsnummer: " & $iWeekday)
