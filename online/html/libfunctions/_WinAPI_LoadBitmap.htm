_WinAPI_LoadBitmap

Beschreibung anzeigen in

Lädt die angegebene Bitmap-Ressource von einem Modul einer ausführbaren Datei

#include <WinAPIRes.au3>
_WinAPI_LoadBitmap ( $hInstance, $sBitmap )

Parameter

$hInstance Handle zu der Instanz des Moduls dessen ausführbare Datei die zu ladende Bitmap enthält
$sBitmap Der Name der zu ladenden Bitmap. Alternativ kann dies der Identifizierer der Ressource
in dem niederwertigen Wort (WORD) und 0 in dem höherwertigen Wort sein.

Rückgabewert

Erfolg: Das Handle zu der angegebenen Bitmap
Fehler: 0

Bemerkungen

Wird die Bitmap nicht mehr benötigt, muss diese Ressource, so wie jedes GDI-Objekt, mit _WinAPI_DeleteObject wieder
freigegeben werden.
