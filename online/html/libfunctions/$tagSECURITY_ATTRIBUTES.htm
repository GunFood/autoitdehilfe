$tagSECURITY_ATTRIBUTES



Beschreibung anzeigen in

Enthält den Security Descriptor für ein Objekt und legt fest, ob das Handle, das durch Festlegung dieser Struktur, vererbbar ist

Global Const $tagSECURITY_ATTRIBUTES = "int Length;ptr Descriptor;int InheritHandle"

Parameter

Length Die Größe dieser Struktur, in Bytes
Descriptor Ein Zeiger auf einen Security Descriptor für das Objekt, der die Freigabe dessen steuert.
InheritHandle Falls True, erbt der neue Prozess das Handle.
