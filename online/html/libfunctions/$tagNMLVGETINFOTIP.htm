$tagNMLVGETINFOTIP



Beschreibung anzeigen in

Enthält und empfängt Informationen eines ListView Items, die gebraucht werden, um einen ToolTip für ein Element darzustellen

Global Const $tagNMLVGETINFOTIP = $tagNMHDR & ";int Flags;ptr Text;int TextMax;int Item;int SubItem;int lParam"

Parameter

$tagNMHDR Enthält Informationen über den Benachrichtigungscode
Flags Entweder Null oder $LVGIT_UNFOLDED
Text Adresse eines String-Puffers, der alle zusätzlichen Textinformationen aufnimmt.
Wenn Flags Null ist, wird dieses Mitglied den vorliegenden Elementtext enthalten.
Gesetzt diesem Falle, sollte jeglicher zusätzliche Text an das Ende dieses Strings angehängt werden.
TextMax Größe des Puffers in Druckzeichen, auf den Text verweist.
Trotzdem sollte man niemals annehmen, dass dieser Puffer jede bestimmte Größe annimmt, der Wert $INFOTIPSIZE kann für Gestaltungszwecke genutzt werden.
Item 0-basierter Index des Elements, auf das diese Struktur verweist.
SubItem 1-basierter Index des Subelements, auf das diese Struktur verweist.
Ist dieses Mitglied Null, verweist diese Struktur auf das Element und nicht auf ein Subelement.
Dieses Mitglied wird zur Zeit nicht genutzt und ist immer Null
lParam Anwendungsdefinierter Wert, der mit dem Element verknüpft ist.
Dieses Mitglied wird zur Zeit nicht genutzt und ist immer Null
