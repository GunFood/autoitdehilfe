_WinAPI_OemToChar



Beschreibung anzeigen in

Konvertiert eine Zeichenfolge aus dem OEM-definierten Zeichensatz in eine ANSI-Zeichenfolge

#include <WinAPIConv.au3>
_WinAPI_OemToChar ( $sStr )

Parameter

$sStr Die Zeichenfolge aus dem OEM-definierten Zeichensatz.

Rückgabewert

Erfolg: Die konvertierte Zeichenfolge.
Fehler: Setzt das @error Flag auf nicht Null, ruft _WinAPI_GetLastError() auf, um erweiterte Fehlerinformationen zu erhalten.
