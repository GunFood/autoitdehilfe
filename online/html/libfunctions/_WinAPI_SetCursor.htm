_WinAPI_SetCursor



Beschreibung anzeigen in

Setzt eine Cursorform

#include <WinAPIRes.au3>
_WinAPI_SetCursor ( $hCursor )

Parameter

$hCursor Identifiziert den Cursor

Rückgabewert

Erfolg: Das Handle des vorigen Cursors, sofern einer vorhanden war.
Fehler: 0, falls vorher kein Cursor vorhanden war.

Bemerkungen

Der Cursor wird nur geändert wenn man sich über einer selbsterstellten GUI befindet.

Siehe auch

Suche nach SetCursor in der MSDN Bibliothek.

Beispiel

#include <WinAPI.au3>

GUICreate("GUI", 300, 300)
GUISetState()
Sleep(1000)
_WinAPI_SetCursor(15)
Sleep(1000)
_WinAPI_SetCursor(17)
Sleep(1000)
