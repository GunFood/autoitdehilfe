Function Reference

_WinAPI_FlushViewOfFile


Writes to the disk a byte range within a mapped view of a file

#include <WinAPIFiles.au3>
_WinAPI_FlushViewOfFile ( $pAddress [, $iBytes = 0] )

Parameters

$pAddress A pointer to the base address of the byte range to be flushed to the disk representation of the mapped file.
$iBytes [optional] The number of bytes to be flushed.
If $iBytes is 0 (Default), the file is flushed from the base address to the end of the mapping.

Return Value

Success: True.
Failure: False, call _WinAPI_GetLastError() to get extended error information
