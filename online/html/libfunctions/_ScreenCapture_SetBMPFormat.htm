_ScreenCapture_SetBMPFormat



Beschreibung anzeigen in

Setzt das Bit-Format welches für BMP Screenshots verwendet wird

#include <ScreenCapture.au3>
_ScreenCapture_SetBMPFormat ( $iFormat )

Parameter

$iFormat Bit pro Pixel (bpp):
    0 = 16 bpp; 5 Bit für jede RGB Komponente
    1 = 16 bpp; 5 Bit für rot, 6 Bit für grün und 5 Bit für blau
    2 = 24 bpp; 8 Bit für jede RGB Komponente
    3 = 32 bpp; 8 Bit für jede RGB Komponente. Keine Alpha Komponente.
    4 = 32 bpp; 8 Bit für jede RGB und Alpha Komponente

Rückgabewert

Keine.

Bemerkungen

Falls nicht angegeben, werden BMP Screenshots standardmäßig in 24 bpp erstellt

Beispiel

#include <ScreenCapture.au3>

Example()

Func Example()
    ; Erstellt einen Screenshot vom kompletten Bildschirm
    _ScreenCapture_SetBMPFormat(0) ; Setzt den Qualitätslevel, der für JPEG Screenshots verwendet wird
    _ScreenCapture_Capture(@MyDocumentsDir & "\GDIPlus_Image.bmp")

    ShellExecute(@MyDocumentsDir & "\GDIPlus_Image.bmp")
EndFunc   ;==>Example
