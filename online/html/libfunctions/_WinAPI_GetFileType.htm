_WinAPI_GetFileType

Beschreibung anzeigen in

Ruft den Dateityp der angegebenen Datei ab

#include <WinAPIFiles.au3>
_WinAPI_GetFileType ( $hFile )

Parameter

$hFile Handle auf die Datei.

Rückgabewert

Erfolg: Dateityp ($FILE_TYPE_*).
Fehler: (-1) und setzt das @error Flag auf ungleich 0, das @extended Flag kann Systemfehlercodes enthalten.

Bemerkungen

$FILE_TYPE_* Konstanten benötigen #include <APIFilesConstants.au3>
