$tagCOMBOBOXEXITEM

Beschreibung anzeigen in


Enthält Informationen über ein Element in einem ComboBoxEx-Control

Global Const $tagCOMBOBOXEXITEM = "int Mask;int Item;ptr Text;int TextMax;int Image;int SelectedImage;int OverlayImage;" & _
"int Indent;int Param"

Parameter

Mask Ein Satz von Bit-Flags, die die Attribute festlegen. Kann eine Kombination der folgenden Werte sein.
    $CBEIF_DI_SETITEM - Dieses Flag wird gesetzt, wenn $CBEN_GETDISPINFO bearbeitet wird
    $CBEIF_IMAGE - Das iImage Mitglied ist gültig oder muss ausgefüllt sein.
    $CBEIF_INDENT - Das iIndent Mitglied ist gültig oder muss ausgefüllt sein.
    $CBEIF_LPARAM - Das lParam Mitglied ist gültig oder muss ausgefüllt sein.
    $CBEIF_OVERLAY - Das iOverlay Mitglied ist gültig oder muss ausgefüllt sein.
    $CBEIF_SELECTEDIMAGE - Das iSelectedImage Mitglied ist gültig oder muss ausgefüllt sein.
    $CBEIF_TEXT - Das pszText Mitglied ist gültig oder muss ausgefüllt sein.
Item Der 0-basierte Index des Elements.
Text Ein Zeiger auf einen Zeichen-Puffer, welcher den Text des Elements enthält oder aufnimmt.
TextMax Die Länge von pszText, in TCHARs. Wenn Text-Information gesetzt ist, wird dieses Mitglied ignoriert.
Image Der 0-basierte Index eines Bildes innerhalb der Imagelist.
SelectedImage Der 0-basierte Index eines Bildes innerhalb der Imagelist.
OverlayImage Der Eins-Basierte Index eines überlagernden Bildes innerhalb der Imagelist.
Indent Die Anzahl der Einrückungen zur Anzeige des Elements. Jede Einrückung entspricht 10 Pixel.
Param Ein Wert spezifisch zu dem Element.
