$tagGDIPIMAGECODECINFO Struktur

Global Const $tagGDIPIMAGECODECINFO = "byte CLSID[16];byte FormatID[16];ptr CodecName;ptr DllName;ptr FormatDesc;ptr FileExt;" & _
"ptr MimeType;dword Flags;dword Version;dword SigCount;dword SigSize;ptr SigPattern;ptr SigMask"

Parameter

CLSID Codec Identifizierer (GUID Struktur)
FormatID Dateiformat Identifizierer (GUID Struktur)
CodecName Zeiger auf einen Null-terminierten Unicode-String die den Codecnamen enthält
DllName Zeiger auf einen Null-terminierten Unicode-String die den Pfad zu der DLL enthält, in welcher sich der Codec befindet.
Wenn der Codec nicht in einer DLL enthalten ist, dann ist dieser Zeiger 0.
FormatDesc Zeiger auf einen Null-terminierten Unicode-String die den Namen des Dateiformats enthält, der von dem Codec genutzt wird.
FileExt Zeiger auf einen Null-terminierten Unicode-String die alle Dateinamenerweiterungen die mit dem Codec verknüpft sind enthält.
Die Erweiterungen werden durch ein Semikolon getrennt.
MimeType Zeiger auf einen Null-terminierten Unicode-String die den Mime Typ des Codecs enthält
Flags Kombination von $GDIP_ICF Flags
Version Bezeichnet die Version des Codecs
SigCount Bezeichnet die Anzahl der Signaturen die durch das Dateiformat verwendet werden und mit dem Codec verknüpft sind
SigSize Bezeichnet die Anzahl der Bytes in jeder Signatur
SigPattern Zeiger auf ein Array von Bytes welches das Muster für jede Signatur enthält
SigMask Zeiger auf ein Array von Bytes welches die Maske für jede Signatur enthält
