Skripteditoren

AutoIt-Skripte sind einfache Textdateien, die man mit Notepad bearbeiten kann. Aber es gibt viele kostenlose oder Shareware-Editoren, die bessere Editier-Funktionen und Syntax-Hervorhebung für AutoIt-Funktionen und Schlüsselwörter bieten, so dass die Arbeit stark erleichtert wird und so weniger Fehler passieren.

Der zur Zeit von AutoIt-Benutzern meistgenutzte Editor ist SciTE. Das AutoIt-Entwickler-Team hat eine eigene AutoIt-Version des Editors herausgebracht, die neben vollständiger Syntax-Hervorhebung auch einige Fremdtools wie den Skriptchecker und Tidyer integriert. Eine "lite" Version des SciTE Editors wird mit dem AutoIt Installationspaket mitgeliefert. Die komplette AutoIt Version von SciTE, die alle Tools enthält, kann unter http://www.autoitscript.com/autoit3/scite/ separat heruntergeladen werden.

 

Einige andere empfohlene Editoren sind:

  • TextPad
  • Crimson Editor (kostenlos)
  • Source Edit (kostenlos)
  • UltraEdit
  • Notepad++ (kostenlos)

AutoIt bringt für viele Editoren Syntax-Dateien mit, die im Unterordner Extras oder über das Startmenü zu finden sind. (Es gibt einen Link unter Startmenü / Programme / AutoIt v3 / Extras).
