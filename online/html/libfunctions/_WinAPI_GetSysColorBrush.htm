_WinAPI_GetSysColorBrush



Beschreibung anzeigen in

Ermittelt ein Handle welches einen logisches Füllmuster identifiziert, das einem festgelegten Farbindex entspricht

#include <WinAPIGdi.au3>
_WinAPI_GetSysColorBrush ( $iIndex )

Parameter

$iIndex Das Anzeige-Element dessen Farbe ermittelt werden soll

Rückgabewert

Erfolg: Ein logisches Füllmuster, falls $iIndex durch die aktuelle Plattform unterstützt wird
Fehler: 0
