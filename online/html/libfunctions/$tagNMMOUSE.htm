$tagNMMOUSE

Beschreibung anzeigen in


Enthält Informationen die mit Meldungen von Maus Nachrichten verwendet werden

Global Const $tagNMMOUSE = $tagNMHDR & ";dword ItemSpec;dword ItemData;int X;int Y;dword HitInfo"

Parameter

$tagNMHDR Enthält Informationen über den Benachrichtigungscode
ItemSpec Eine Controlspezifische Element ID
ItemData Controlsspezifische Elementdaten
X Legt die X-Koordinate des Punktes fest
Y Legt die Y-Koordinate des Punktes fest
HitInfo Enthält Informationen wo auf dem Element oder dem Control sich der Cursor befindet
