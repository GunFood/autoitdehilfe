_WinAPI_IsWow64Process



Beschreibung anzeigen in

Prüft, ob der angegebene Prozess unter WOW64 ausgeführt wird

#include <WinAPIProc.au3>
_WinAPI_IsWow64Process ( [$iPID = 0] )

Parameter

$iPID [optional] Die PID des Prozesses. Standard (0) ist der aktuelle Prozess.

Rückgabewert

Erfolg: True - Der PRozess läuft unter WOW64.
False - Sonstiges.
Fehler: Setzt @error auf ungleich 0. Es ist _WinAPI_GetLastError() aufzurufen um erweiterte Fehlerinformationen zu erhalten.
