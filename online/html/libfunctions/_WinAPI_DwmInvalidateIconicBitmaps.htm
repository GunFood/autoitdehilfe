_WinAPI_DwmInvalidateIconicBitmaps



Beschreibung anzeigen in

Deutet darauf hin, dass alle vorher bereitgestellen symbolischen Bitmaps eines Fensters, sowohl Vorschaubilder als auch Spitzenrepräsentation, aktualisiert werden sollten

#include <WinAPIGdi.au3>
_WinAPI_DwmInvalidateIconicBitmaps ( $hWnd )

Parameter

$hWnd Das Handle eines Fensters oder Tabs wessen Bitmaps durch diesen Aufruf ungültig gemacht werden.

Rückgabewert

Erfolg: 1.
Fehler: 0 und setzt @error auf ungleich 0. @extended kann den HRESULT Fehlercode enthalten.

Bemerkungen

Durch den Aufruf dieser Funktion bewirkt man, dass der Desktop-Fenster-Manager (DWM) die aktuellen Bitmaps für das Fenster ungültig macht und neue Bitmaps von dem Fenster anfordert wenn sie das nächste Mal benötigt werden.
_WinAPI_DwmInvalidateIconicBitmaps() sollte nicht regelmäßig aufgerufen werden.
Sollte man dies trotzdem tun, so kann das zu schlechter Performance führen, da neue Bitmaps erstellt und zurückgegeben werden.

Diese Funktion benötigt Windows 7 oder höher.
