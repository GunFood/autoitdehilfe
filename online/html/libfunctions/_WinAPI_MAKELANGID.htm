_WinAPI_MAKELANGID



Beschreibung anzeigen in

Erstellt eine Sprach-ID von einer Primär-Sprach-ID und einer Sub-Sprach-ID

#include <WinAPIConv.au3>
_WinAPI_MAKELANGID ( $iLngIDPrimary, $iLngIDSub )

Parameter

$iLngIDPrimary Primär-Sprach-ID
$iLngIDSub Sub-Sprach-ID

Rückgabewert

Gibt die Sprach-ID zurück

Bemerkungen

Die Konstanten für $iLngIDPrimary oder $iLngIDSub sind in WinAPILangConstants.au3 definiert.
