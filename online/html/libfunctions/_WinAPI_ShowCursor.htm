_WinAPI_ShowCursor

Beschreibung anzeigen in

Zeigt oder versteckt den Cursor

#include <WinAPIRes.au3>
_WinAPI_ShowCursor ( $bShow )

Parameter

$bShow Bei True wird der Cursor angezeigt, bei False versteckt

Rückgabewert

Gibt den neuen Anzeigenzähler zurück.

Bemerkungen

Diese Funktion setzt einen internen Anzeigenzähler, der entscheidet, ob die Maus angezeigt wird.
Der Cursor wird nur angezeigt, wenn der Anzeigenzähler größer oder gleich 0 ist.
Wenn eine Maus installiert ist, ist der anfängliche Wert des Anzeigenzählers 0.
Wenn keine Maus installiert ist, ist der Wert des Anzeigenzählers -1.
Der Cursor wird nur angezeigt bzw. versteckt wenn man sich über einer selbsterstellten GUI befindet.

Siehe auch

Suche nach ShowCursor in der MSDN Bibliothek.

Beispiel

#include <WinAPI.au3>

GUICreate("GUI", 300, 300)
GUISetState()
Sleep(1000)
_WinAPI_ShowCursor(0)
Sleep(1000)
_WinAPI_ShowCursor(1)
Sleep(1000)
