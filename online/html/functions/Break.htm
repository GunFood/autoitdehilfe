Break



Beschreibung anzeigen in

Ermöglicht oder blockiert das Abbrechen eines Skripts durch den Benutzer über das Tray-Icon-Menü.

Break ( mode )

Parameter

mode Setzt den Skript Breakmodus:
    $BREAK_ENABLE (1) = Break wird ermöglicht (der Benutzer kann das Skript abbrechen) (Standard)
    $BREAK_DISABLE (0) = Break wird blockiert (der Benutzer kann das Skript nicht abbrechen)

Die Konstanten sind in "AutoItConstants.au3" definiert.

Rückgabewert

Keine.

Bemerkungen

Bitte das Abbrechen nur aus guten Gründen blockieren.
AutoIt erzeugt normalerweise ein Tray-Icon, wenn ein Skript läuft, und ein Rechtsklick auf dieses Icon ermöglicht es dem Benutzer, das Skript anzuhalten (pause) oder abzubrechen (exit). Wenn Break() blockiert wird (0), kann der Benutzer das Skript nicht auf diese Weise beenden.

Beispiel

Break(0) ; Verhindert Unterbrechungen
