$tagNMCBEDRAGBEGIN

Beschreibung anzeigen in

Enthält Informationen die mit der $CBEN_DRAGBEGIN Benachrichtigung genutzt wird

Global Const $tagNMCBEDRAGBEGIN = $tagNMHDR & ";int ItemID;wchar szText[260]"

Parameter

$tagNMHDR Enthält Informationen über den Benachrichtigungscode
ItemID Der 0-basierte Index des Elements das gezogen ("Drag") werden soll. Dieser Wert ist immer -1, hinweisend darauf, dass das gezogene Element jenes Element ist das in dem Edit-Bereich des Controls angezeigt wird.
szText Der Zeichen-Puffer der den Text des gezogenen Elements enthält

Verwandte Funktionen

$tagNMHDR
