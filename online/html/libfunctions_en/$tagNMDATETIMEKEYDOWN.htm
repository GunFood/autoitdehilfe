$tagNMDATETIMEKEYDOWN



Show description in

Carries information used to describe and handle a $DTN_WMKEYDOWN notification message

Global Const $tagNMDATETIMEKEYDOWN = $tagNMHDR & ";int VirtKey;ptr Format;" & $tagSYSTEMTIME

Parameters

$tagNMHDR Contains information about a notification message
VirtKey Virtual key code that represents the key that the user pressed
Format Pointer to a null terminated string that defines the control callback field.
The string comprises one or more "X" characters.
$tagSYSTEMTIME Contains information about the current system date and time from the DTP control.
The owner of the control must modify the time information based on the user's keystroke.

Related

$tagNMHDR, $tagSYSTEMTIME
