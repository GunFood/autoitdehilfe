_WinAPI_GetErrorMode

Retrieves the error mode for the current process

#include <WinAPIDiag.au3>
_WinAPI_GetErrorMode ( )

Parameter

None.

Rückgabewert

Success: The process error mode ($SEM_*).
Failure: 0.

Bemerkungen

This function requires Windows Vista or later.
