ToolTip

Beschreibung anzeigen in


Erstellt einen "Tooltip" irgendwo auf dem Bildschirm.

ToolTip ( "text" [, x [, y [, "title" [, icon = 0 [, options]]]]] )

Parameter

text Der Text des Tooltips. (Ein Leerstring löscht einen angezeigten Tooltip)
x [optional] Die x-Position des Tooltips.
y [optional] Die y-Position des Tooltips.
title [optional] Der Titel des Tooltips
icon [optional] Vordefiniertes Icon, das in der Nähe des Titels angezeigt wird: Benötigt einen Titel.
    $TIP_NOICON (0) = Kein Icon
    $TIP_INFOICON (1) = Info Icon
    $TIP_WARNINGICON (2) = Warnung Icon
    $TIP_ERRORICON (3) = Fehler Icon

Die Konstanten sind in "AutoItConstants.au3" definiert.
options [optional] Setzt verschiedene Anzeigeoptionen (können addiert werden):
    $TIP_BALLOON (1) = Anzeige als Balloon-Tip
    $TIP_CENTER (2) = Zentriert den Tooltip an den x,y-Koordinaten, anstatt ihn in der oberen linken Ecke zu platzieren.
    $TIP_FORCEVISIBLE (4) = Zwingt den Tooltip immer innerhalb der Monitorgrenzen sichtbar zu sein, wenn nötig. Wenn mehrere Monitore benutzt werden, "schnappt" der Tooltip zum nächstgelegenen Monitor über.

Die Konstanten sind in "AutoItConstants.au3" definiert.

Rückgabewert

Erfolg: 1.
Fehler: 0 wenn der Titel mehr als 99 Zeichen enthält.

Bemerkungen

Um einen optionalen Parameter zu überspringen, seinen Standardwert also zu belassen, wird das Schlüsselwort Default verwendet.
Wenn die x- und y-Koordinaten ausgelassen werden, wird der Tooltip in der Nähe des Mauscursors gesetzt.
Ein Tooltip bleibt sichtbar, bis das Skript schließt oder ToolTip("") aufgerufen wird.
Man benötigt @CR oder @LF, um mehrzeilige Tooltips zu erstellen.
Um ein Icon zu zeigen, darf der Titel nicht leer sein. Das Icon erscheint in der selben Reihe wie der Titel und dies erfordert, dass dieser existiert. Wird die Zentrieroption verwendet, befindet sich die Mitte des Tooltipps an den festgelegten Koordinaten.
Wenn die Zentrieroption in Verbindung mit einem Balloon-Tip verwendet wird, so wird der Balloonhals zentriert und zeigt auf die festgelegten Koordinaten.

Verwandte Funktionen

MsgBox, SplashTextOn, TrayTip

Beispiel

; Dies erstellt einen Tooltip in der oberen linken Ecke des Bildschirms

ToolTip("Dies ist ein Tool-Tip", 0, 0)
Sleep(2000) ; Leerlaufzeit, in der der Tool-Tip angezeigt werden kann
