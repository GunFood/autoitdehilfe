_WinAPI_CreateDirectoryEx

Beschreibung anzeigen in

Erzeugt ein neues Verzeichnis mit den Attributen eines gegebenen Vorlageverzeichnisses.

#include <WinAPIFiles.au3>
_WinAPI_CreateDirectoryEx ( $sNewDir, $sTemplateDir [, $tSecurity = 0] )

Parameter

$sNewDir Der Pfad des Verzeichnisses welches erzeugt werden soll.
$sTemplateDir Der Pfad des Verzeichnisses welches als Vorlage, für das Erzeugen des neuen Verzeichnisses, dienen soll.
$tSecurity [optional] Die $tagSECURITY_ATTRIBUTES Struktur welche den Sicherheitsdeskriptor für das neue Verzeichnis angibt.
Wenn dieser Parameter 0 ist (Standard), so erhält das Verzeichnis einen Standardsicherheitsdeskriptor.
Die ACL im Standardsicherheitsdeskriptor werden von dem darüberliegenden Verzeichnis geerbt.

Rückgabewert

Erfolg: True.
Fehler: False. Es ist _WinAPI_GetLastError() aufzurufen um erweiterte Fehlerinformationen zu erhalten.

Bemerkungen

Die _WinAPI_CreateDirectoryEx() Funktion erlaubt es einem Verzeichnisse zu erzeugen, welche Streaminformationen von anderen Verzeichnissen erbt.
Diese Funktion ist hilfreich wenn man beispielsweise Macintosh Verzeichnisse verwendet welche einen Ressourcestream haben welcher benötigt wird um vernünftig die Verzeichnisinhalte als Attribute zu identifizieren.
