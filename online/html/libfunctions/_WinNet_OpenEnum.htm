_WinNet_OpenEnum

Beschreibung anzeigen in

Startet eine Auflistung der Netzwerkressourcen oder existierender Verbindungen

#include <WinNet.au3>
_WinNet_OpenEnum ( $iScope, $iType, $iUsage, $tResource, ByRef $hEnum )

Parameter

$iScope Bereich der Auflistung:
    0 - Listet alle aktuell verbundenen Ressourcen.
    1 - Listet alle Ressourcen im Netzwerk.
    2 - Listet alle gespeicherten (dauerhaften) Verbindungen.
    3 - Listet nur Ressourcen, die sich im Netzwerkkontext des Aufrufers befinden.
$iType Art der Ressourcen:
    0 - Alle Ressourcen
    1 - Laufwerk-Ressourcen
    2 - Drucker-Ressourcen
$iUsage Nutzungsart der Ressourcen:
    0 - Alle Ressource
    1 - Alle verknüpfbaren Ressourcen
    2 - Alle Container-Ressourcen
    4 - Führt zum Abbruch der Funktion, wenn der Benutzer nicht authentifiziert ist.
$tResource eine $tagNETRESOURCE-Struktur oder einen Zeiger auf diese, die den Container für die Auflistung spezifiziert.
Wenn $iScope nicht 1 ist, muss pResource 0 sein.
Wenn 0, wird das Stammverzeichnis des Netzwerks verwendet.
$hEnum Handle, das für spätere _WinNet_EnumResource()-Aufrufe benutzt werden kann.

Rückgabewert

Erfolg: True
Fehler: False

Verwandte Funktionen

$tagNETRESOURCE, _WinNet_CloseEnum, _WinNet_EnumResource
