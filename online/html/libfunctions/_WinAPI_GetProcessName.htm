_WinAPI_GetProcessName



Beschreibung anzeigen in

Ruft den Namen für den angegebenen Prozess ab

#include <WinAPIProc.au3>
_WinAPI_GetProcessName ( [$iPID = 0] )

Parameter

$iPID [optional] Die PID des Prozesses. Standard (0) ist der aktuelle Prozess.

Rückgabewert

Erfolg: Der Prozessname.
Fehler: Leerer String und setzt das @error-Flag auf ungleich Null.
