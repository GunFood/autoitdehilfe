Parameter Struktur für den Farbbalance Effekt

Global Const $tagGDIP_EFFECTPARAMS_ColorBalance = "int CyanRed; int MagentaGreen; int YellowBlue"

Parameter

CyanRed Integer im Bereich von -100 bis 100, der die Änderung im Rotbereich des Bildes festlegt.
MagentaGreen Integer im Bereich von -100 bis 100, der die Änderung im Grünbereich des Bildes festlegt.
YellowBlue Integer im Bereich von -100 bis 100, der die Änderung im Blaubereich des Bildes festlegt.
