_WinNet_RestoreConnection



Beschreibung anzeigen in

Stellt die Verbindung zu einer Netzwerkressource wieder her

#include <WinNet.au3>
_WinNet_RestoreConnection ( [$sDevice = "" [, $hWnd = 0 [, $bUseUI = True]]] )

Parameter

$sDevice [optional] Der lokale Name des Laufwerks, mit dem verbunden werden soll, z.B."Z:".
Wenn leer, werden alle Verbindungen wiederhergestellt, die dauerhaft als Laufwerk in der Registry des aktuellen Benutzers gespeichert sind.
$hWnd [optional] Handle zum Parent-Fenster, das die Funktion benutzt, um den Benutzer nach Name und Passwort für die Netzwerkverbindung zu fragen.
Wenn 0, hat das Fenster keinen Besitzer.
$bUseUI [optional] Wenn True, wird dem Aufrufer ein Login-Dialog angezeigt.

Rückgabewert

Erfolg: True
Fehler: False
