_Date_Time_DOSTimeToArray

Show description in

Decode a DOS time to an array

#include <Date.au3>
_Date_Time_DOSTimeToArray ( $iDosTime )

Parameters

$iDosTime MS-DOS date, packed as follows:
Bits 0- 4 Second divided by 2
Bits 5-10 Minute (0–59)
Bits 11-15 Hour (0–23 on a 24-hour clock)

Return Value

Returns an array with the following format:
    [0] - Hour
    [1] - Minute
    [2] - Second

Related

_Date_Time_DOSDateTimeToArray, _Date_Time_DOSDateToArray

Example

#include "Extras\HelpFileInternals.au3"

#include <Date.au3>
#include <GUIConstantsEx.au3>
#include <WindowsStylesConstants.au3>

Example()

Func Example()
    Local $aDate

    ; Create GUI
    GUICreate("Time", 400, 300)
    _MemoCreate(2, 2, 396, 296, $WS_VSCROLL)
    GUISetState(@SW_SHOW)

    ; Show FAT time
    $aDate = _Date_Time_DOSTimeToArray(0x944a) ; 18:34:20
    _MemoWrite("FAT time .: " & StringFormat("%02d:%02d:%02d", $aDate[0], $aDate[1], $aDate[2]))

    ; Loop until the user exits.
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>Example
