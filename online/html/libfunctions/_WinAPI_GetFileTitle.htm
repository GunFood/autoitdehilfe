_WinAPI_GetFileTitle

Beschreibung anzeigen in

Ermittelt den Namen der angegebenen Datei

#include <WinAPIFiles.au3>
_WinAPI_GetFileTitle ( $sFilePath )

Parameter

$sFilePath Der Name und der Speicherort einer Datei.

Rückgabewert

Erfolg: Der Name der Datei.
Fehler: Leerer String.

Bemerkungen

_WinAPI_GetFileTitle() ssollte nur mit legalen Dateinamen aufgerufen werden; die Verwendung eines illegalen Dateinamens hat ein undefiniertes Ergebnis.

_WinAPI_GetFileTitle() gibt den String zurück, die das System verwenden würde, um dem Benutzer den Dateinamen anzuzeigen.
Der Anzeigename enthält nur dann eine Erweiterung, wenn der Benutzer dies für die Anzeige von Dateinamen bevorzugt.
Dies bedeutet, dass der zurückgegebene String die Datei möglicherweise nicht genau identifiziert, wenn er in Aufrufen von Dateisystemfunktionen verwendet wird.
