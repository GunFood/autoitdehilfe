Funktionreferenz

_IE_Introduction



Beschreibung anzeigen in

Zeigt grundlegende Informationen über die IE.au3 in einem neuen Browserfenster an

#include <IE.au3>
_IE_Introduction ( [$sModule = "basic"] )

Parameter

$sModule [optional] Gibt an, welches Modul gestartet werden soll
    "basic" = (Standard) grundlegende Einführung

Rückgabewert

Erfolg: eine Objektvariable, die auf ein InternetExplorer.Application-Objekt verweist
Fehler: Setzt das @error Flag auf ungleich null
@Error: 5 ($_IEStatus_InvalidValue) = Ungültiger Wert
@Extended: Beinhaltet Nummer des ungültigen Parameters

Bemerkungen

Die AutoIt-Hilfe erlaubt es im Moment nicht, dass UDFs die eigenen Funktionen gruppiert darstellen, sondern es werden immer nur einzelne Funktionen angezeigt.
Diese Funktion soll diesen Überblick verschaffen und mit der Zeit ausgebaut werden.

Verwandte Funktionen

_IE_Example

Beispiel

#include <IE.au3>

; Öffnet ein neues Browserfenster mit der Einführung in die Verwendung der IE.au3

Local $oIE = _IE_Introduction("basic")
