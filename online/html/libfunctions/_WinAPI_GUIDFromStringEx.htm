Konvertiert den String einer GUID in die binäre Form

#include <WinAPIConv.au3>
_WinAPI_GUIDFromStringEx ( $sGUID, $tGUID )

Parameter

$sGUID GUID in Form eines Strings
$tGUID Eine $tagGUID Struktur oder ein Pointer zu dieser, in der die GUID abgelegt wird

Rückgabewert

Erfolg: True
Fehler: False

Verwandte Funktionen

$tagGUID, _WinAPI_StringFromGUID
