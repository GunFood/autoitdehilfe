_WinAPI_CoInitialize



Beschreibung anzeigen in

Initialisiert die COM-Bibliothek zur Verwendung durch den aufrufenden Prozess

#include <WinAPICom.au3>
_WinAPI_CoInitialize ( [$iFlags = 0] )

Parameter

$iFlags [optional] Dieser Parameter kann einer oder mehrere der folgenden Werte sein.
    $COINIT_APARTMENTTHREADED
    $COINIT_DISABLE_OLE1DDE
    $COINIT_MULTITHREADED (Standard)
    $COINIT_SPEED_OVER_MEMORY

Rückgabewert

Erfolg: 1.
Fehler: 0 und setzt das @error Flag auf ungleich null, das @extended Flag kann den HRESULT-Fehlercode enthalten.
